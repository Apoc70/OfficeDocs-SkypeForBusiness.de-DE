</div>

<div id="mainSection">

<div id="mainBody">

_**Letztes Änderungsstand des Themas:** 2013-02-17_

  - Sicherungstools. Ausführliche Informationen finden Sie unter [Sicherungs-und Wiederherstellungsanforderungen in lync Server 2013: Tools and Permissions](lync-server-2013-backup-and-restoration-requirements-tools-and-permissions.md).

  - Datenträger für die Sicherungen. Ausführliche Informationen finden Sie unter [Einrichten eines Sicherungsspeicherorts für lync Server 2013](lync-server-2013-setting-up-a-backup-location.md).
