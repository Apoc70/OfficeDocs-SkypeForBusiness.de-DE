# <a name="new-virtualization-features-in-lync-server-2013"></a>Neue Features für die Virtualisierung in lync Server 2013

_**Letztes Änderungsstand des Themas:** 2013-11-07_

Lync Server 2013 unterstützt die Virtualisierung auf Windows Server 2012, Windows Server 2012 R2 und Windows Server 2008 R2. Unterstützung für Windows Server 2012 und Windows Server 2012 R2 bietet Unterstützung für die einzelnen root I/O Virtualization (SR-IOV)-Funktionen. Mit SR-IOV wird die virtuelle Funktion eines physischen Netzwerkadapters direkt einem virtuellen Computer zugewiesen. Dies erhöht den Netzwerkdurchsatz und verringert die Netzwerkwartezeit und reduziert gleichzeitig den CPU-Host Aufwand, der für die Verarbeitung des Netzwerkdatenverkehrs erforderlich ist. Um die Vorteile von SR-IOV nutzen zu können, müssen Sie einen Hostserver verwenden, der über ein BIOS verfügt, das SR-IOV unterstützt, sowie Netzwerkadapter verwenden, die SR-IOV unterstützen.
