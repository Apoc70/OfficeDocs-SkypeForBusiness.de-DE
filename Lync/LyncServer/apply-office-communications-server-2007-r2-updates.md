</div>

<div id="mainBody">

<span> </span>

_**Letztes Änderungsdatum des Themas:** 2012-10-19_

Bevor Sie zu lync Server 2013 migrieren, müssen Updates auf Ihre Office Communications Server 2007 R2-Umgebung angewendet werden. Aktuelle Informationen zu Office Communications Server 2007 R2 finden Sie unter **Updates für** **Communications Server 2007 R2** unter [http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802).

Wenn Sie Updates für Office Communications Server 2007 R2 installieren möchten, empfehlen wir, dass Sie die im Microsoft Knowledge Base-Artikel 968802, "Updates für Communications Server 2007 R2", beschriebene Installationsprozedur für das [http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802) **kumulative Server Update** für die Methode befolgen.
