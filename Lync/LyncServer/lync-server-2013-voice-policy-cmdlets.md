# <a name="voice-policy-cmdlets-in-lync-server-2013"></a>VoIP-Richtlinien-Cmdlets in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsdatum des Themas:** 2013-02-18_

Das Verwalten von Enterprise-VoIP umfasst die Konfiguration von VoIP-Richtlinien und Wählplänen sowie das Zuordnen von VoIP-Richtlinien zu VoIP-Routen. Cmdlets zum Verwalten von VoIP-Richtlinien können verwendet werden, um Features wie das gleichzeitige Klingeln festzulegen (die Möglichkeit, jedes Mal, wenn jemand Ihr Office-Telefon anruft, ein zweites Telefon klingeln zu lassen), die Anrufweiterleitung und die Wähl Anforderung.

<div>
