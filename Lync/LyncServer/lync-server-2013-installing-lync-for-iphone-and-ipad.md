# <a name="installing-lync-for-iphone-and-ipad-in-lync-server-2013"></a>Installieren von lync für iPhone und iPad in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2014-03-10_

Lync 2013 für iPhone und lync 2013 für iPad sind Benutzer installierbare Anwendungen, die im Apple App Store verfügbar sind.

<div>

## <a name="installing-lync-for-iphone-and-lync-for-ipad"></a>Installieren von lync für iPhone und lync für iPad

Sie können Ihre Benutzer anweisen, lync 2013 für iPhone und lync 2013 für iPad zu installieren, indem Sie Sie über Ihre Geräte an den App Store weiterleiten. Der App Store für jedes Gerät ist auch online verfügbar.

  - Lync für iPhone steht im App Store unter \< h<span></span>ttp://www.Apple.com/iPhone/from-the-App-Store/zur Verfügung >

  - Lync für iPad steht im App Store unter \< HT<span></span>TP://www.Apple.com/iPad/from-the-App-Store/zur Verfügung >

<div>


> [!IMPORTANT]  
> iPhone-Benutzer, die die lync 2013-APP nicht installiert haben und die versuchen, an einer lync-Besprechung aus einer Besprechungseinladung teilzunehmen, werden an eine Startseite des Joins umgeleitet. Diese Seite enthält einen Link zum Installieren der lync 2013-app. Anstatt den Benutzer jedoch zum App Store zu leiten, wird mit diesem Link eine leere Safari-Browser Seite geöffnet. Der Benutzer kann eines der folgenden Schritte Unternehmen, um dieses Problem zu umgehen: 
> <UL>
> <LI>
> <P>Verwenden Sie die <STRONG>Home</STRONG> -Schaltfläche, um die Safari-Seite an den Hintergrund zu senden, und öffnen Sie dann Safari erneut. Wenn die Benachrichtigung "Diese Seite im App Store öffnen" angezeigt wird, tippen Sie auf <STRONG>Öffnen</STRONG> , um zu lync 2013 Download im App Store geleitet zu werden.</P>
> <LI>
> <P>Öffnen Sie den App-Store manuell, suchen Sie nach "lync 2013", und laden Sie die APP herunter.</P></LI></UL>

## <a name="verifying-mobile-client-installation"></a>Überprüfen der Installation des mobilen Clients

Nachdem Sie den Client konfiguriert und sich erfolgreich angemeldet haben, können Sie anhand der folgenden Tests überprüfen, ob die lync-Installation auf Ihrem mobilen Gerät ordnungsgemäß funktioniert.

**Suchen nach einem Kontakt im Unternehmensverzeichnis**

1.  Tippen Sie in der Kontaktliste oben auf die Suchleiste, und beginnen Sie mit der Eingabe des Namens eines Kontakts, der nur in der globalen Adressliste (GAL) vorhanden ist.

2.  Stellen Sie sicher, dass der Kontaktname in den Suchergebnissen angezeigt wird.

**Testen von Chatnachrichten und Anwesenheitsinformationen**

1.  Tippen Sie in der Kontaktliste auf einen Kontakt.

2.  Tippen Sie in der Visitenkarte auf das Symbol **IM**.

3.  Stellen Sie sicher, dass ein Chatfenster angezeigt wird und Sie ein Chatnachrichten eingeben und senden können.

**Testen der Dial-Out-Konferenz**

1.  Planen Sie in Outlook eine Lync-Besprechung.

2.  Öffnen Sie auf dem mobilen Gerät die Besprechungseinladung.

3.  Klicken Sie in der Besprechung auf den Link, um an der Besprechung teilzunehmen.

4.  Nehmen Sie den Anruf vom Konferenzdienst entgegen, und stellen Sie sicher, dass Sie mit dem Besprechungsaudio verbunden sind.

**Testen von Pushbenachrichtigungen**

1.  Melden Sie sich auf dem mobilen Gerät von Benutzer A bei Lync mit dem Konto des Benutzers A an.

2.  Öffnen Sie eine andere Anwendung auf dem mobilen Gerät.

3.  Melden Sie sich in einem anderen Client mit dem Konto von Benutzer B bei Lync an.

4.  Senden Sie eine Sofortnachricht von Benutzer B an Benutzer A.

5.  Stellen Sie sicher, dass die Sofortnachricht auf dem mobilen Gerät von Benutzer A erscheint.
