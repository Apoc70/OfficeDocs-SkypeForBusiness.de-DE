</div>

Sie müssen die erforderliche Software auf jedem Edgeserver bereitstellen, den Sie vor der Installation von lync Server 2013 bereitstellen. Dies umfasst die Installation des Betriebssystems auf einem Server, der die Systemanforderungen erfüllt. Details zu den Systemanforderungen, einschließlich der unterstützten Betriebssysteme, finden Sie unter [Systemanforderungen für externe Benutzerzugriffs Komponenten für lync Server 2013](lync-server-2013-system-requirements-for-external-user-access-components.md).
