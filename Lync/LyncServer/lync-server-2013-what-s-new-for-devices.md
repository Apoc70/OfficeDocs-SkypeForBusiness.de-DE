_**Letztes Änderungsstand des Themas:** 2012-06-22_

Lync Server 2013 umfasst lync-Phone Edition, Software, die auf qualifizierten Geräten ausgeführt wird und traditionelle und erweiterte Telefoniefunktionen, integrierte Sicherheit, Verwaltbarkeit und vieles mehr bietet. Lync Phone Edition funktioniert auf die gleiche Weise wie bei lync Server 2013 mit lync Server 2010. Ausführliche Informationen zu den neuesten Funktionen im Zusammenhang mit Geräten finden Sie unter [What es New for Devices](https://go.microsoft.com/fwlink/p/?linkid=256490) in the lync Server 2010 TechNet Library.
