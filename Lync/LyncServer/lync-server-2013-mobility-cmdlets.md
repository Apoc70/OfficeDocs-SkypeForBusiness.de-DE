# <a name="mobility-cmdlets-in-lync-server-2013"></a>Mobilitäts-Cmdlets in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-06-20_

Mobilitäts-Cmdlets wurden eingeführt, um das im kumulativen Update für lync Server 2010 hinzugefügte Mobilitätsfeature zu verwalten: November 2011. Mit diesen Cmdlets können Einstellungen für Mobilitätsfeatures verwaltet werden, beispielsweise die Konfiguration und die Benutzerrichtlinien für den Mobilitätsdienst.

<div>
