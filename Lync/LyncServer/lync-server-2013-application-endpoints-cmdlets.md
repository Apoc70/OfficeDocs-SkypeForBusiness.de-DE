# <a name="application-endpoints-cmdlets-in-lync-server-2013"></a>Cmdlets für Anwendungsendpunkte in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsdatum des Themas:** 2012-06-20_

Anwendungsendpunkte sind Kontaktobjekte in den Active Directory-Domänendiensten. Sie können Cmdlets in der lync Server-Verwaltungsshell verwenden, um diese Kontakte für Microsoft lync Server 2013 zu verwalten.

<div>

## <a name="application-endpoints-cmdlets"></a>Cmdlets für Anwendungsendpunkte

Verwenden Sie die folgenden Cmdlets zum Verwalten von Anwendungsendpunkten.

**Anwendungsendpunkte**

  - <span></span>  
    [Get-CsApplicationEndpoint](https://technet.microsoft.com/en-us/library/Gg398655(v=OCS.15))

  - <span></span>  
    [Move-CsApplicationEndpoint](https://technet.microsoft.com/en-us/library/Gg398188(v=OCS.15))
