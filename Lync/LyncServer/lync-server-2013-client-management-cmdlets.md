## <a name="client-management-cmdlets"></a>Cmdlets für die Client Verwaltung

Die meisten Verwaltungsaufgaben, die für die Clientverwaltung gelten, können über die lync Server 2013-Systemsteuerung ausgeführt werden. Dieselben Aufgaben können mit Cmdlets aus der lync Server-Verwaltungsshell oder in einem Skript ausgeführt werden. Mithilfe eines Skripts können Sie bestimmte Aufgaben automatisieren.
