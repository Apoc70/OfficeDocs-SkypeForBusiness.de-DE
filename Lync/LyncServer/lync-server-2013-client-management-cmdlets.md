# <a name="client-management-cmdlets-in-lync-server-2013"></a>Cmdlets für die Client Verwaltung in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-09-27_

Die Clientverwaltung besteht in erster Linie darin, zu ermitteln, welche Clientanwendungen (wie Microsoft lync 2013) sich bei lync Server 2013 anmelden können, und die Funktionen zu ermitteln, die diesen Clientanwendungen zur Verfügung stehen, nachdem Sie sich angemeldet haben.

## <a name="client-management-cmdlets"></a>Cmdlets für die Clientverwaltung

Die meisten Verwaltungsaufgaben, die für die Clientverwaltung gelten, können in der lync Server 2013-Systemsteuerung ausgeführt werden. Dieselben Aufgaben können mit Cmdlets aus dem lync Server-Verwaltungsshell oder in einem Skript ausgeführt werden. Durch die Verwendung eines Skripts können bestimmte Aufgaben automatisiert werden. In der folgenden Liste werden Cmdlets aufgeführt, die im Rahmen der Clientverwaltung eingesetzt werden:
