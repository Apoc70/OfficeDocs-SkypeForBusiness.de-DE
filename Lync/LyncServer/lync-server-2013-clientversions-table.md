# <a name="clientversions-table-in-lync-server-2013"></a>ClientVersions-Tabelle in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsdatum des Themas:** 2012-09-28_
