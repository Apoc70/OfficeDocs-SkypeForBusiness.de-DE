# <a name="using-scan-results-to-analyze-and-resolve-issues-reported-by-best-practices-analyzer-in-lync-server-2013"></a>Verwenden von Überprüfungsergebnissen zum Analysieren und Beheben von Problemen, die von Best Practices Analyzer in lync Server 2013 gemeldet wurden

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-06-14_

Wenn Sie lync Server 2013, Best Practices Analyzer in ihrer lync Server 2013 Umgebung ausführen, verwendet das Tool die Überprüfungsergebnisse, um Berichte über Probleme mit Ihrer Bereitstellung zu generieren. Mit den vom Best Practices Analyzer generierten Berichten können Sie bestimmte Probleme erkennen und beheben.

<div>


> [!NOTE]  
> Lync Server 2013 überprüft und meldet Best Practices Analyzer nur Probleme mit lync Server 2013-Komponenten. Wenn Ihre Bereitstellung Microsoft lync Server 2010-oder Office Communications Server 2007 R2 Komponenten enthält, verwenden Sie die vorherige Version von Best Practices Analyzer, um diese Komponenten zu analysieren. Ausführliche Informationen finden Sie unter <A href="lync-server-2013-requirements-for-running-best-practices-analyzer.md">Requirements for Running Best Practices Analyzer in lync Server 2013</A>.
