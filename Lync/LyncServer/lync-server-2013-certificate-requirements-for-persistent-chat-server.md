# <a name="certificate-requirements-for-persistent-chat-server-in-lync-server-2013"></a>Zertifikatanforderungen für den Server für beständigen Chat in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-10-03_

Um den Server für beständigen Chat zu installieren, müssen Sie über ein Zertifikat verfügen, das von derselben Zertifizierungsstelle wie das von lync Server 2013 internen Servern für jeden Server ausgestellt wurde, auf dem der beständige Chat Webdienste für den Datei Upload/-Download ausgeführt wird. Stellen Sie sicher, dass Sie über die erforderlichen Zertifikate verfügen, bevor Sie mit der Installation des beständigen Chats beginnen, insbesondere dann, wenn Sie eine externe Zertifizierungsstelle verwenden.
