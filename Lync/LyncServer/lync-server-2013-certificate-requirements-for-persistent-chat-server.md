# <a name="certificate-requirements-for-persistent-chat-server-in-lync-server-2013"></a>Zertifikatanforderungen für den Server für beständigen Chat in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsdatum des Themas:** 2012-10-03_

Zum Installieren des beständigen Chat Servers müssen Sie über ein Zertifikat verfügen, das von der gleichen Zertifizierungsstelle wie das von lync Server 2013-internen Servern für jeden Server ausgestellt ist, auf dem die beständigen Chat-Webdienste für Datei Upload/-Download ausgeführt werden. Stellen Sie sicher, dass Sie über die erforderlichen Zertifikate verfügen, bevor Sie mit der Installation des beständigen Chats beginnen, insbesondere dann, wenn Sie eine externe Zertifizierungsstelle verwenden.
