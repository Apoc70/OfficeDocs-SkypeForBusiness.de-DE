# <a name="manage-sip-federated-providers-for-your-organization-in-lync-server-2013"></a>Verwalten von SIP-Verbund Anbietern für Ihre Organisation in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-10-19_

Dies ist eine vorläufige Dokumentation und kann jederzeit geändert werden. Leere Themen wurden als Platzhalter hinzugefügt.

Um die Unterstützung für Benutzer von SIP-Verbund Anbietern zu konfigurieren, müssen Sie folgende Schritte ausführen:

  - Konfigurieren einer oder mehrerer Richtlinien für den externen Benutzer Zugriff zur Unterstützung der Kommunikation mit SIP-Verbund Anbieter Kontakten

  - Angeben, welche gehosteten Anbieter unterstützt werden sollen

  - Angeben, welche öffentlichen Chat Anbieter Sie unterstützen möchten

Verwenden Sie zum Ausführen dieser Aufgaben die Verfahren in diesem Abschnitt.
