# <a name="hosted-exchange-unified-messaging-integration-in-lync-server-2013"></a>Hosted Exchange Unified Messaging-Integration in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-09-20_

Zusätzlich zu der Unterstützung, die frühere lync Server 2013-Versionen für die Integration in *lokale* Bereitstellungen von Exchange Unified Messaging (um) bereitgestellt haben, bietet lync Server 2013 die Unterstützung für die Integration in *gehostete* Exchange um. Hosted Exchange um ermöglicht lync Server 2013 das Bereitstellen von Sprachnachrichten für Ihre Benutzer, wenn Sie einige oder alle diese an einen gehosteten Exchange-Dienstanbieter wie Microsoft Exchange Online übertragen.

Lync Server 2013 Enterprise-VoIP verwendet die Exchange um-Infrastruktur, um Mailboxansage, Anrufbenachrichtigung, Sprachzugriff (einschließlich Voicemail) und automatische Telefonzentralendienste bereitzustellen. Ausführliche Informationen finden Sie unter [Features of Integrated Unified Messaging and lync Server 2013](lync-server-2013-features-of-integrated-unified-messaging.md).
