# <a name="configure-iis-for-lync-server-2013"></a>Konfigurieren von IIS für lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2011-12-16_

Beim Konfigurieren Internet Information Services (IIS) für lync Server 2013 müssen die richtigen Komponenten installiert werden, um die von lync Server 2013 benötigten Webdienste zu unterstützen. Ausführliche Informationen zum Installieren von IIS finden Sie unter [IIS-Konfiguration in lync Server 2013](lync-server-2013-iis-configuration.md). Wenn Sie eine Richtlinie zum Ausführen des Sicherheitskonfigurations-Assistenten auf Servern vor der Inbetriebnahme oder als typischer Bestandteil ihrer Wartung haben, finden Sie unter [reactivate Server after Security Configuration Wizard Ports in IIS](lync-server-2013-re-activate-server-after-security-configuration-wizard-closes-ports-in-iis.md) , um Informationen zu einem Nebeneffekt der Ausführung des Assistenten zu erhalten, der Ports in einer lync Server 2013 IIS-Konfiguration schließt.
