# <a name="planning-for-announcements-in-lync-server-2013"></a>Planen von Ankündigungen in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-09-13_

Im lync Server Ankündigungsanwendung können Sie die Verarbeitung eingehender Anrufe konfigurieren, wenn die gewählte Nummer für Ihre Organisation gültig ist, aber keinem Benutzer oder Telefon zugewiesen ist. Sie können diese Anrufe an ein vorab festgelegtes Ziel (Telefonnummer, SIP-URI oder Voicemail) oder eine Audioansage oder beides weitergeben. Die Ankündigungsanwendung hilft Ihnen, die Situationen zu vermeiden, in denen ein Anrufer sich verwählt und einen besetzt-Ton hört oder der SIP-Client eine Fehlermeldung erhält. Dieser Abschnitt enthält Planungsinformationen, die speziell für die Ankündigungsanwendung sind.
