_**Letztes Änderungsstand des Themas:** 2012-10-01_

Die lync Server 2013, die Server Dienste für beständigen Chat verwenden eine *gestreckte Pool* Konfiguration für die Notfallwiederherstellung. Ein gestreckter Pool ist ein Pool mit Computern, die zwischen zwei physischen Rechenzentren verteilt sind, aber sich innerhalb eines einzelnen logischen lync Server-Standorts befinden.
