# <a name="scanning-for-viruses-and-checking-virus-definitions-in-lync-server-2013"></a>Überprüfen auf Viren und Virendefinitionen in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2014-05-01_

Es wird dringend empfohlen, ein Antivirenprogramm für eine Sofortnachrichten Ebene zu installieren. Im ist eine bekannte Quelle für die schnelle Verbreitung von Viren und bösartiger Software in einer Organisation. Microsoft Forefront® Sicherheit für lync Server bietet Multi-Engine-Scans mit Viren, bösartiger Software, Datei-und Keyword-Filterschutz und nahtloser Integration in Office Communications Server.

Neben Forefront Security für lync Server empfehlen wir auch, eine Antivirus-Lösung auf Dateiebene zu installieren, um das Dateisystem des Servers zu schützen.

Das Aktualisieren von Scanner-Engines und Virendefinitionen ist sehr wichtig. Durch das Konfigurieren und Überwachen der Integrität der Updates wird sichergestellt, dass die aktuellsten Überprüfungsinformationen verwendet werden, um sowohl Office Communications Server als auch das Dateisystem zu schützen.

<div>


> [!IMPORTANT]  
> Wenn Sie eine Antivirensoftware eines Drittanbieters auf Dateiebene auf einem Server verwenden, auf dem lync Server 2013 und Forefront Security für lync Server ausgeführt wird, stellen Sie sicher, dass die Ordner, in denen Forefront Security für lync Server und die lync Server installiert sind, nicht überprüft werden, um zu verhindern Korruption. Die vollständige Liste der Ausschlüsse finden Sie <A class=uri href="http://support.microsoft.com/kb/943620">http://support.microsoft.com/kb/943620</A>unter.
