# <a name="planning-for-archiving-in-lync-server-2013"></a>Planen der Archivierung in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-09-28_

Unternehmen und andere Organisationen müssen eine zunehmende Anzahl von Rechtsvorschriften beachten, die die Speicherung bestimmter Kommunikationsarten erfordern. Wenn Ihre Organisation solche Anforderungen erfüllt, können Sie die Archivierung in lync Server 2013 verwenden, um Chatnachrichten (Instant Messaging) und Konferenzen (Besprechungen) zu archivieren, die über lync Server gesendet wurden, um die Einhaltung einiger Compliance-Anforderungen zu unterstützen.
