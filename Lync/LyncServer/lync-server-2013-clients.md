# <a name="clients-for-lync-server-2013"></a>Clients für Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2014-01-23_

Lync Server 2013 unterstützt verschiedene Arten von Client Software, die Sie für die Benutzer Ihrer Organisation bereitstellen können, einschließlich Computer installierter Client Software, webbasierter Clients und mobiler Geräte. In diesem Thema werden die Clients beschrieben, die Sie verwenden können. Einen detaillierten Vergleich der von lync Server 2013 Clients bereitgestellten Features finden Sie unter [Client comparison tables for lync Server 2013](lync-server-2013-desktop-client-comparison-tables.md).

<div>

## <a name="lync-2013"></a>Lync 2013

Lync 2013 ist der Client mit vollem Funktionsumfang für lync Server. Die lync 2013-Benutzeroberfläche wurde vollständig neu gestaltet und enthält neu integrierte Features wie beständiger Chat (lync 2010 hatte einen separaten Client für Chat Funktionen), Unterhaltungen im Registerkartenformat, Video Vorschau und Video mit mehreren Teil Videos. Eine Zusammenfassung der Änderungen finden Sie unter [What es New for Clients in lync Server 2013](lync-server-2013-what-s-new-for-clients.md).

Lync 2013 Client-Setup ist Teil des Office-Setupprogramms auf dem Installationsmedium.

</div>

<div>

## <a name="online-meeting-add-in-for-lync-2013"></a>Online-Besprechungs-Add-in für lync 2013

Das Online-Besprechungs-Add-in für lync 2013 unterstützt die Besprechungsverwaltung in Microsoft Outlook Messaging-und Zusammenarbeitsclient. Das Online Besprechungs-Add-in für lync 2013 Software wird automatisch mit lync 2013 installiert.

</div>

<div>

## <a name="lync-web-scheduler"></a>Lync-Webplaner

Lync-Webplaner ist ein webbasiertes Tool für die Planung und Verwaltung von Besprechungen für Benutzer, die keinen Zugriff auf Microsoft Outlook haben oder die sich auf einem Betriebssystem befinden, das nicht auf Windows basiert. Mit dem lync-Webplaner können Benutzer neue Besprechungen erstellen, vorhandene Besprechungen ändern und Einladungen mit Ihrem bevorzugten e-Mail-Programm senden.

</div>

<div>

## <a name="lync-web-app"></a>Lync Web App

Lync Web App ist der webbasierte Konferenz Client für lync Server 2013-Besprechungen. In dieser Version bietet das Hinzufügen von Computer-Audio-und-Video zu lync Web App eine umfassende Besprechungs Erfahrung für alle Benutzer, die keinen lync-Client lokal installiert haben. Besprechungsteilnehmer können auf alle Funktionen für die Zusammenarbeit und Freigabe sowie auf alle Besprechungssteuerelemente des Referenten zugreifen.

Wenn lync 2013 nicht auf dem Computer eines Benutzers installiert ist und der Benutzer in einer Besprechungsanfrage auf einen Besprechungslink klickt, wird lync Web App geöffnet. Sie können auch die Seite für den besprechungsbeitritt so konfigurieren, dass Benutzer mit früheren Versionen von Clients an Besprechungen teilnehmen können. Weitere Informationen finden Sie unter [Konfigurieren der Seite für den besprechungsbeitritt in lync Server 2013](lync-server-2013-configuring-the-meeting-join-page.md) in der Bereitstellungsdokumentation.

Aufgrund der Verbesserungen an lync Web App ist eine aktualisierte Version von Microsoft lync 2010 Attendee für lync Server 2013 nicht verfügbar. Lync Web App ist der bevorzugte Client für Teilnehmer außerhalb Ihrer Organisation. Bei lync Web App ist keine lokale Clientinstallation erforderlich, obwohl für Audio-, Video-und Freigabefunktionen bei der ersten Verwendung eine Installation eines Plug-Ins erforderlich ist.

</div>

<div>

## <a name="lync-2013-basic"></a>Lync 2013 Basic

Lync 2013 Basic ist ein herunterladbarer Client für Kunden mit einer lizenzierten lokalen lync Server 2013-Bereitstellung und Kunden, die einen Microsoft 365-oder Office 365-Plan abonnieren, der nicht den vollständigen lync 2013-Client enthält. Der lync Basic-Client umfasst erweiterte Anwesenheitsinformationen, Kontakte, Chatnachrichten, lync-Besprechungen und grundlegende VoIP-Funktionen. Zu den in lync Basic nicht unterstützten Features gehören mehrteiliges Video, OneNote-Integration, Unterstützung für die virtuelle Desktopinfrastruktur (VDI), Skill-Suche, Aufzeichnung, Enterprise-VoIP-Funktionen und erweiterte Anrufverarbeitung (beispielsweise Anrufweiterleitung und Teamanruf). Ausführliche Informationen finden Sie unter [Client Vergleichstabellen für lync Server 2013](lync-server-2013-desktop-client-comparison-tables.md).

</div>

<div>

## <a name="lync-windows-store-app"></a>Lync Windows Store-App

Die lync Windows Store-App ist eine Touch-optimierte lync-APP, die speziell für Windows 8.1, Windows 8 und Windows RT entwickelt wurde. Benutzer können die APP über den Windows Store herunterladen, indem Sie nach "lync" suchen. Weitere Informationen finden Sie unter [Client Vergleichstabellen für lync Server 2013](lync-server-2013-desktop-client-comparison-tables.md), [lync Windows Store-App-Anforderungen für lync Server 2013](lync-server-2013-lync-windows-store-app-requirements.md)und [Bereitstellen der lync Windows Store-App in lync Server 2013](lync-server-2013-deploying-lync-windows-store-app.md).

</div>

<div>

## <a name="lync-2013-for-mobile-devices"></a>Lync 2013 für mobile Geräte

Lync 2013 Mobile Apps nun zusätzlich zu Kontakten, Anwesenheit und Chat Features VoIP-und Video over IP-Funktionen hinzu. Mobile Benutzer können über Chatnachrichten, Sprachanrufe oder Videoanrufe mit anderen Benutzern kommunizieren, indem Sie entweder Wi-Fi oder Ihre mobilfunkdatenverbindung verwenden. Mit einem einzigen Mausklick auf den Besprechungslink in einem Kalenderelement können mobile Benutzer an sprach-und Videobesprechungen teilnehmen. Weitere Informationen zu lync 2013 Mobile Apps finden Sie unter [Planning for Mobile Clients in lync Server 2013](lync-server-2013-planning-for-mobile-clients.md).

</div>

<div>

## <a name="supported-clients-from-previous-releases"></a>Unterstützte Clients aus vorherigen Versionen

Lync Server 2013 unterstützt die folgenden Clients aus früheren Server Versionen. Sie können Benutzern bestimmte frühere Clients zur Verfügung stellen, wenn Sie an Besprechungen teilnehmen. Ausführliche Informationen finden Sie unter [Konfigurieren der Seite für den besprechungsbeitritt in lync Server 2013](lync-server-2013-configuring-the-meeting-join-page.md) in der Bereitstellungsdokumentation.

  - **Lync 2010**     Lync 2010 bietet eine umfassende Desktopumgebung, einschließlich Sofortnachrichten, erweiterte Anwesenheitsinformationen, sprach-, Video-, Freigabe-und Telefoniefunktionen. Es ist jedoch keines der neuen Features, die in lync Server 2013 eingeführt werden, verfügbar, bis der Client des Benutzers auf lync 2013 aktualisiert wurde.

  - **Lync 2010 Mobile**     Lync Server 2013 unterstützt alle Microsoft lync 2010 Mobile Mobile Apps. Microsoft lync 2010 Mobile bietet Sofortnachrichten, erweiterte Anwesenheitsinformationen und Telefonie für Benutzer in Ihrer Organisation, die von einem Smartphone oder einem Telefon mit einer Professional Edition von Windows Mobile eine Verbindung herstellen. Sie können Ihre Benutzer anweisen, Microsoft lync 2010 Mobile zu installieren, indem Sie Sie an den App-Marktplatz für Ihr Mobiltelefon weiterleiten. Ausführliche Informationen finden Sie unter "Planning for Mobile Clients" in der lync Server 2010-Dokumentation unter [https://go.microsoft.com/fwlink/p/?LinkID=235955](https://go.microsoft.com/fwlink/p/?linkid=235955) .

  - **Lync Phone Edition**     Lync Phone Edition-Software für intelligente IP-Telefone (beispielsweise mit USB-angeschlossenen Telefonen) wurde nicht für lync Server 2013 aktualisiert. Lync Phone Edition wird weiterhin für das platzieren und empfangen von anrufen, erweiterte Anwesenheitsfunktionen und Client-Audiofunktionen für Konferenzen unterstützt.

  - **Lync 2010 Attendant**     Das Microsoft lync 2010 Attendant integrierte Anruf Verwaltungsprogramm ermöglicht es einer Empfangsdame, mehrere Unterhaltungen gleichzeitig durch schnelle Anrufbehandlung, Sofortnachrichten und das Onscreen-Routing zu verwalten.

</div>
