</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsdatum des Themas:** 2012-10-19_

Bevor Sie zu lync Server 2013 migrieren, müssen Updates auf Ihre lync Server 2010-Umgebung angewendet werden. Aktuelle Informationen zu lync Server 2010 finden Sie unter **Updates Resource Center für lync** unter <http://go.microsoft.com/fwlink/p/?linkid=232630>.
