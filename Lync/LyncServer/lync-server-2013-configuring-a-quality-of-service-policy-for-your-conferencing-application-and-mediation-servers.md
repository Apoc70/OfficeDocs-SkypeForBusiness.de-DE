<span data-ttu-id="cce4f-p101">Das Konfigurieren von Portbereichen vereinfacht die Verwendung der Dienstqualität (Quality of Service), indem sichergestellt wird, dass der gesamte Datenverkehr eines angegebenen Typs (z. B. der gesamte Audio-Datenverkehr) über dieselben Ports übermittelt wird. Dadurch ist es für das System einfacher, ein bestimmtes Paket zu identifizieren und zu markieren: Wenn der Port 49152 für den Audio-Datenverkehr reserviert ist, kann jedes beliebige Paket, das über den Port 49152 übermittelt wird, mit einem DSCP-Code markiert werden. Dieser Code gibt an, dass es sich dabei um ein Audiopaket handelt. Anschließend können diese Router das Paket als ein Audiopaket identifizieren und ihm eine höhere Priorität als unmarkierte Pakete verleihen (wie z. B. Pakete, die Dateien von einem Server zu einem anderen Server kopieren).</span><span class="sxs-lookup"><span data-stu-id="cce4f-p101">Configuring port ranges facilitates the use of Quality of Service by ensuring that all traffic of a specified type (for example, all audio traffic) travels through the same set of ports.
