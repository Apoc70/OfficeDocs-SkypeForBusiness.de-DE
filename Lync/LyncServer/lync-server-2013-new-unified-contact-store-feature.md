# <a name="new-unified-contact-store-feature-in-lync-server-2013"></a>Neuer Unified Contact Store-Feature in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-09-13_

Lync Server 2013 stellt einen einheitlichen Kontaktspeicher vor. Mit dem einheitlichen Kontaktspeicher können Benutzer alle Ihre Kontaktinformationen in Microsoft Exchange Server 2013 speichern. Nachdem die lync-Kontakte von Benutzern zu Exchange 2013 migriert wurden, können die Benutzer über lync 2013, Outlook oder Outlook Web App auf Ihre Kontakte zugreifen und diese verwalten, und Ihre Favoriten bleiben synchronisiert. Wenn ein Benutzer beispielsweise einen Kontakt zu den Favoriten in Outlook hinzufügt, wird der Kontakt in der Gruppe Favoriten in lync 2013 angezeigt. Benutzer müssen nicht bei lync angemeldet sein, um Ihre Kontakte in Outlook oder Outlook Web App verwalten zu können.

Der einheitliche Kontaktspeicher ist standardmäßig aktiviert. Sie können Benutzer für den einheitlichen Kontaktspeicher Global, nach Standort, vom Mandanten oder von Einzelpersonen oder Personengruppen aktivieren oder deaktivieren.
