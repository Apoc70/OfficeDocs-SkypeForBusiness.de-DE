# <a name="planning-for-group-call-pickup-in-lync-server-2013"></a>Planen der gruppenanrufannahme in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2013-02-01_

Kumulatives Update für lync Server 2013: Februar 2013 bietet eine Einführung in die gruppenanrufannahme als neue Enterprise-VoIP-Funktion. Bei der gruppenanrufannahme können Benutzer Anrufe abholen, die für einen anderen Benutzer Klingeln, indem Sie eine Rufnummer für die Anrufannahme Gruppe wählen. Dieser Abschnitt enthält Planungsinformationen, die für die gruppenanrufannahme spezifisch sind.
