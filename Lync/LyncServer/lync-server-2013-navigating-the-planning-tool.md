# <a name="navigating-the-planning-tool-in-lync-server-2013"></a>Navigieren im Planungs Tool in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2013-02-21_

Sie Navigieren im Microsoft lync Server 2013 Planungs Tool mit einer Kombination aus einer Symbolleiste, seitenspezifischen Schaltflächen und Links sowie kontextspezifischen Bereichen. Die kontextspezifischen Bereiche bieten Entwurfsinformationen für Planung und Kapazität, die für die Auswahloptionen auf einer bestimmten Seite relevant sind.

Beim Starten des Planungstools sieht ein Designer zunächst das **Willkommen beim Planungstool für Microsoft lync Server 2013** Seite.

![Startseite des Planungstools](images/Gg558601.ff5b72e6-bcf0-49e9-8784-3636fe8187c5(OCS.15).jpg "Startseite des Planungstools")

Auf der Willkommensseite wählt der Designer **Erste Schritte**, **Entwurfs Websites**oder **Anzeige**aus. Ausführliche Informationen finden Sie unter [Erstellen des anfänglichen Topologie Designs für lync Server 2013](lync-server-2013-create-the-initial-topology-design.md).

Am oberen Rand des Planungstools befindet sich eine Symbolleiste, die einfachen Zugriff auf häufig verwendete Funktionen ermöglicht. Die Symbolleiste wird hier als Referenz angezeigt, und jede Funktion wird in verwandten Themen behandelt.

![Planungs Tool-Symbolleiste](images/Gg558601.a008ddd1-b73d-4406-9d4b-df68bed9906e(OCS.15).jpg "Planungs Tool-Symbolleiste")

Das Planungstool verfügt über einen Abschnitt externe Links auf der linken Seite des Tools. Von hier aus erhält der für den Entwurf verantwortliche Benutzer schnellen Zugriff auf Informationen zu Planung und Bereitstellung sowie auf weitere technische Ressourcen, z. B. Schulungsinformationen, technische Blogs, Foren und andere herunterladbare Ressourcen. Im Abschnitt externe Links finden Sie auch einen Feedback Link zum Team von lync Server 2013 Planungstools.

![Planungs Tool externe Links (Dialogfeld)](images/Gg558601.76959057-8eb2-4158-b1b3-585cca80be7e(OCS.15).jpg "Planungs Tool externe Links (Dialogfeld)")

Ein kontextsensitiver Aktionsbereich wird auf vielen Seiten im Planungs Tool angezeigt. Der Bereich Aktionen bietet dem Designer einfachen Zugriff auf die Hauptabschnitte der Topologie. Die im Bereich "Aktionen" verfügbaren Links ändern sich basierend auf der Detailebene in Ihrer Topologie. Der Bereich Aktionen steht zur Verfügung, nachdem Sie die Fragen zum Interview abgeschlossen und Ihre Topologie angezeigt haben. Im Bereich "Aktionen" ist der Abschnitt "Übersicht" enthalten, in dem die Nummern angezeigt werden, die der Designer als Teil des Gesprächs Prozesses eingegeben hat. Die Übersicht steht in Beziehung zu den angezeigten Informationen.

Zusätzlich werden in der Übersicht unterhalb des Bereichs "Aktionen" Hardwareinformationen angezeigt. Die Hardwarekonfiguration zeigt eine Liste der Hardwareanforderungen, die für die aktuelle Topologie empfohlen werden.

![Bereich "Planungs Tool-Aktionen"](images/Gg558601.9679d8fd-4de8-4a5a-bfcf-699da9aa7283(OCS.15).jpg "Bereich "Planungs Tool-Aktionen"")
