# <a name="analyzing-and-resolving-issues-identified-by-best-practices-analyzer-in-lync-server-2013"></a>Analysieren und Beheben von Problemen, die von Best Practices Analyzer in lync Server 2013 identifiziert wurden

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsdatum des Themas:** 2012-06-25_

Best Practices Analyzer nimmt keine Änderungen an Ihrer Umgebung vor, um Probleme zu beheben. Sie erkennt nur potenzielle Probleme und zeigt Informationen dazu an, wie Sie aufgelöst werden können. Wenn Sie Probleme identifizieren, die Sie beheben müssen, müssen Sie die entsprechende Lösung ermitteln. Die Themen in diesem Abschnitt helfen Ihnen, einige der wichtigsten potenziellen Probleme zu identifizieren und zu beheben.
