# <a name="unassigned-number-cmdlets-in-lync-server-2013"></a>Cmdlets für nicht zugewiesene Nummern in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsdatum des Themas:** 2012-03-21_

Nicht zugewiesene Nummern sind Telefonnummern, die zu einer Organisation gehören, aber keinem Benutzer oder Telefon zugewiesen wurden. Nicht zugewiesene Nummern können so konfiguriert werden, dass Sie unterschiedliche Arten von Ankündigungen wiedergeben, wenn ein Anruf an diese Nummern durchgeführt wird.

<div>
