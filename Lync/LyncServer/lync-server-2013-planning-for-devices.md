# <a name="planning-for-devices-in-lync-server-2013"></a>Planen von Geräten in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2013-02-28_

Lync Server 2013 umfasst lync-Phone Edition, Software, die auf qualifizierten Geräten ausgeführt wird und traditionelle und erweiterte Telefoniefunktionen, integrierte Sicherheit, Verwaltbarkeit und vieles mehr bietet. Lync Phone Edition funktioniert auf die gleiche Weise wie bei lync Server 2013 mit lync Server 2010. Ausführliche Informationen zur Planung von Geräten finden Sie unter [Planning for Devices](https://go.microsoft.com/fwlink/p/?linkid=285880) in the lync Server 2010 TechNet Library.
