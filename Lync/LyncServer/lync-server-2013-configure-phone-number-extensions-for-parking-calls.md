# <a name="configure-phone-number-extensions-for-parking-calls-in-lync-server-2013"></a>Konfigurieren von Telefonnummern Erweiterungen für Parken von Anrufen in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsdatum des Themas:** 2012-09-10_

Die Anruf Park Anwendung verwendet Durchwahlnummern in der Orbit-Tabelle des Anruf Parks, um Anrufe zu parken. Sie müssen die Orbit-Tabelle des Anruf Parks mit den Bereichen der Durchwahlnummern konfigurieren, die Ihre Organisation für geparkte Anrufe reserviert. Dabei muss es sich um virtuelle Durchwahlnummern handeln (also Nummern, denen kein Benutzer oder Telefon zugeordnet ist). Jeder lync-Server Pool, in dem eine Anruf Park Anwendung bereitgestellt und konfiguriert ist, kann einen oder mehrere Umlaufbahn Bereiche aufweisen. Umlaufbahn Bereiche müssen in der lync Server-Bereitstellung global eindeutig sein.

<div>


> [!IMPORTANT]  
> Sie müssen das Kontrollkästchen <STRONG>Anruf Park aktivieren</STRONG> in Ihrer VoIP-Richtlinie aktivieren, bevor Sie den Anruf Park verwenden können. Diese Option ist standardmäßig nicht aktiviert.
