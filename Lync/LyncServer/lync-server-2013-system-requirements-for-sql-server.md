# <a name="system-requirements-for-sql-server-in-lync-server-2013"></a>System Anforderungen für SQL Server in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2013-10-25_

Installieren Sie vor dem Bereitstellen von Enterprise Edition-Server Microsoft SQL Server 2008 R2 oder Microsoft SQL Server 2012 auf einem dedizierten Computer, der die Hardwareanforderungen erfüllt. Ausführliche Informationen zu den Hardwareanforderungen finden Sie unter [Server Hardware Platforms for lync Server 2013](lync-server-2013-server-hardware-platforms.md) in der Unterstützungsdokumentation. Ausführliche Informationen zu den Softwareanforderungen finden Sie unter [Database Software Support in lync Server 2013](lync-server-2013-database-software-support.md) in der Unterstützungsdokumentation. Informationen zu den Berechtigungen, die für die Bereitstellung erforderlich sind, finden Sie unter [Deployment Permissions for SQL Server in lync Server 2013](lync-server-2013-deployment-permissions-for-sql-server.md).

Bevor Sie die Front-End-Pool erstellen, müssen Sie auch die Windows-Firewall so konfigurieren, dass lync Server 2013 Zugriff auf SQL Server über bestimmte Ports möglich ist, indem Sie Ports für den Server mit SQL Server Configuration Manager definieren und Ports in der Windows-Firewall mit öffnen. Erweiterte Sicherheit.
