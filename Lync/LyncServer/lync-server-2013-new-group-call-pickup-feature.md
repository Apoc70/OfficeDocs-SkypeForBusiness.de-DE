# <a name="new-group-call-pickup-feature-in-lync-server-2013"></a>Neue Funktion für die gruppenanrufannahme in lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

_**Letztes Änderungsstand des Themas:** 2013-02-12_

Kumulative Updates für lync Server 2013: Februar 2013 stellt die gruppenanrufannahme als neue Enterprise-VoIP-Funktion vor. Mit der gruppenanrufannahme können Sie Benutzergruppen zuweisen, damit andere Benutzer eingehende Anrufe an Benutzer, die sich in der Gruppe befinden, über ihre eigenen Telefone beantworten können.

Die gruppenanrufannahme basiert auf dem Anwendung zum Parken von anrufen. Bei der gruppenanrufannahme richten Sie wie bei der Funktion zum Parken von anrufen einen Bereich virtueller Nummern ein, die als Gruppennummern verwendet werden sollen. Ein Benutzer wählt die Gruppennummer, um einen Anruf aufzunehmen, der für einen anderen Benutzer in der Gruppe klingelt.
