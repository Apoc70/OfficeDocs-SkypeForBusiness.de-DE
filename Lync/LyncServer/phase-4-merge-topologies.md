# <a name="phase-4-merge-topologies"></a>Phase 4: Zusammenführen von Topologien

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

_**Letztes Änderungsstand des Themas:** 2012-03-29_

In den folgenden Themen werden die Schritte beschrieben, die zum Zusammenführen Ihrer Microsoft Office Communications Server 2007 R2 Pools mit Microsoft lync Server 2013 Pools erforderlich sind. Verwenden Sie zunächst den Zusammenführungs-Assistenten des Topologie-Generators, um Topologieinformationen zusammenzuführen. Dieses Tool sammelt Informationen zu Ihrer Office Communications Server 2007 R2 Umgebung, einschließlich Edgeserver Informationen, und veröffentlicht diese Informationen in einer Datenbank, die für lync Server 2013 freigegeben wurde. Nachdem Sie die zusammengeführte Topologie veröffentlicht haben, wird der Topologie-Generator verwendet, um die Office Communications Server 2007 R2 Topologieinformationen sowie Informationen zur neu bereitgestellten lync Server 2013 Topologie anzuzeigen. Importieren Sie die Richtlinien und Konfigurationseinstellungen abschließend mithilfe der Lync Server Management Shell-Cmdlets.
