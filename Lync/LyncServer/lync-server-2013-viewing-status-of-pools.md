# <a name="viewing-status-of-pools-in-lync-server-2013"></a>Anzeigen des Status von Pools in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2014-05-01_

Für jeden Enterprise-Pool und den Pool jeder Standard Edition-Server können Sie Informationen zum Pool Status und zum Status anderer Komponenten anzeigen, die vom Pool über die lync Server 2013-Systemsteuerung verwendet werden. Wir empfehlen die Implementierung von System Center Operations Manager 2012 mit dem lync Server-Systemsteuerung, um Administratoren beim Anzeigen der Integrität von Pools in Ihrer Umgebung zu unterstützen.

<div>

## <a name="view-pools"></a>Anzeigen von Pools

1.  Öffnen Sie mit dem lync Server-Systemsteuerung das lync Server-Systemsteuerung.

2.  Klicken Sie auf Topologie, und überprüfen Sie den Status aller Server in der Topologie.
