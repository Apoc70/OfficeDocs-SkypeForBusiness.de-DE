# <a name="quality-of-experience-qoe-in-lync-server-2013"></a>QoE (Quality of Experience) in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-11-01_

Bei der QoE-Datenerfassung (Quality of Experience) werden numerische Daten aufgezeichnet, die die Medienqualität sowie Informationen zu Teilnehmern, Gerätenamen, Treibern, IP-Adressen und Endpunkttypen im Zusammenhang mit Anrufen und Sitzungen angeben. Wenn Sie lync Server 2013 installieren, installieren Sie auch eine vordefinierte Sammlung globaler Konfigurationseinstellungen für QoE. Konfigurieren Sie QoE-Einstellungen mithilfe der Themen in diesem Abschnitt.
