# <a name="features-of-integrated-unified-messaging-and-lync-server-2013"></a>Features integrierter Unified Messaging-und lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-10-01_

Lync Server 2013 verwendet Enterprise-VoIP die Exchange Unified Messaging (um)-Infrastruktur zur Bereitstellung von Mailboxansage, Anrufbenachrichtigung, Sprachzugriff (einschließlich Voicemail) und automatischen Telefonzentralen Diensten.

<div>

## <a name="call-answering"></a>Anrufbeantwortung

Mailboxansage ist der Empfang von Sprachnachrichten im Namen von Benutzern, deren Anrufe nicht beantwortet werden oder ausgelastet sind. Es umfasst die Wiedergabe einer persönlichen Begrüßung, das Aufzeichnen einer Nachricht und das Senden der Nachricht an die Warteschlange für die Zustellung an das Postfach des Benutzers, das auf dem Exchange-Postfachserver gespeichert ist.

Wenn ein Anrufer eine Nachricht hinterlässt, wird diese an den Posteingang des Benutzers übermittelt. Wenn ein Anrufer keine Nachricht hinterlässt, wird eine Benachrichtigung über einen entgangenen Anruf im Posteingang des Benutzers gespeichert. Benutzer können folgendermaßen auf ihren Posteingang zugreifen: über einen Microsoft Office Outlook-Client für Messaging und Zusammenarbeit, über Outlook Web Access, die Exchange ActiveSync-Technologie oder Outlook Voice Access. Betreff und Priorität von Anrufen können auf ähnliche Weise angezeigt werden wie für E-Mails.

</div>

<div>

## <a name="outlook-voice-access"></a>Outlook Voice Access

Outlook Voice Access ermöglicht Enterprise-VoIP-Benutzern über eine Telefonie-Benutzeroberfläche nicht nur den Zugriff auf Voicemail, sondern auch auf den Posteingang von Exchange, einschließlich E-Mail, Sprachnachrichten, Kalender und Kontakten. Die Teilnehmerzugriffsnummer wird von einem Exchange um Administrator zugewiesen.

</div>

<div>

## <a name="auto-attendant"></a>Automatische Telefonzentrale

Die automatische Telefonzentrale ist ein Exchange UMes Feature, mit dem eine Telefonnummer konfiguriert werden kann, die von externen Benutzern zum Erreichen von Unternehmensvertretern gewählt werden kann. Diese Funktion stellt ferner verschiedene Ansagen bereit, die einem externen Anrufer die Navigation in einem Menüsystem ermöglichen. Die Liste der verfügbaren Optionen wird vom Exchange um Administrator auf dem Exchange um Server konfiguriert.

</div>

<div>

## <a name="fax-services"></a>Faxdienste

Exchange um enthält Fax-Features, mit denen Benutzer eingehende Faxe in Ihren Exchange-Postfächern empfangen können. Ausführliche Informationen finden Sie unter "Unified Messaging" in der Exchange Server- [https://go.microsoft.com/fwlink/p/?linkId=135652](https://go.microsoft.com/fwlink/p/?linkid=135652)Dokumentation unter.

<div>


> [!NOTE]  
> Die vom Exchange um-Server bereitgestellten Faxdienst sind in lync Server-Bereitstellungen nicht verfügbar, die mit Microsoft Exchange Server 2010, Exchange 2010 mit dem neuesten Service Pack oder Exchange 2013 integriert sind.
