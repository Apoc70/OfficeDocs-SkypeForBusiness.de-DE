# <a name="enable-or-disable-group-call-pickup-for-users-in-lync-server-2013"></a>Aktivieren oder Deaktivieren der gruppenanrufannahme für Benutzer in lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2013-02-22_

Wenn der Anruf Park Orbit-Tabelle ein Gruppenanruf-abholbereich hinzugefügt wird, können Sie die gruppenanrufannahme für den Benutzer nach Bedarf aktivieren und deaktivieren. In diesem Abschnitt wird die Verwendung des SEFAUtil Resource Kit-Tools zum Aktivieren und Deaktivieren der gruppenanrufannahme für Benutzer beschrieben.

<div>


> [!NOTE]  
> Der Parameter SEFAUtil,/enablegrouppickup, aktiviert die gruppenanrufannahme und weist die Gruppennummer zu.
