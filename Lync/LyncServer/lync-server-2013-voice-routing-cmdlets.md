# <a name="voice-routing-cmdlets-in-lync-server-2013"></a>Cmdlets für das VoIP-Routing in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsdatum des Themas:** 2012-06-26_

VoIP-Routen enthalten Anweisungen, die Microsoft lync Server 2013 mitteilen, wie Anrufe von Enterprise-VoIP-Benutzern an Telefonnummern im öffentlichen Telefonnetz (PSTN) oder an eine Private Branch Exchange (PBX) weitergeleitet werden.

<div>
