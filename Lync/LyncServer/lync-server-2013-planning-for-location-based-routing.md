# <a name="planning-for-location-based-routing-in-lync-server-2013"></a>Planen von Location-Based Routing in lync Server 2013

_**Letztes Änderungsstand des Themas:** 2013-07-31_

Die Informationen in diesem Thema beziehen sich auf kumulative Updates für lync Server 2013: Februar 2013.

Location-Based Routing ermöglicht es, das Routing von anrufen zwischen VoIP-Endpunkten und PSTN-Endpunkten basierend auf dem Standort der Parteien im Anruf einzuschränken. Location-Based Routing ist Teil der lync Server 2013 Enterprise-VoIP-Infrastruktur. Location-Based Routing ist ein Anruf Verwaltungsfeature, das steuert, wie Anrufe von lync Server 2013 ku1 weitergeleitet werden. Es erzwingt Anruf Autorisierungsregeln, ob Anrufe basierend auf dem geografischen Standort des lync-Anrufers an PBX-oder PSTN-Endpunkte weitergeleitet werden können.
