# <a name="av-conferencing-requirements-in-lync-server-2013"></a>A/V-Konferenz Anforderungen in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-09-30_

Lync Server 2013 Infrastrukturanforderungen für Konferenzen entsprechen denen für die Bereitstellung von lync Server 2013. Ausführliche Informationen finden Sie unter [bestimmen der Infrastrukturanforderungen für lync Server 2013](lync-server-2013-determining-your-infrastructure-requirements.md) in der Planungsdokumentation.

Es ist wichtig, dass Sie über die erforderliche Medien Bandbreite verfügen, um den durch Konferenzen generierten Mediendatenverkehr zu verarbeiten. Ausführliche Informationen, die Sie zum Berechnen der benötigten Bandbreite verwenden können, finden Sie unter Anforderungen an die [Netzwerkbandbreite für Mediendatenverkehr in lync Server 2013](lync-server-2013-network-bandwidth-requirements-for-media-traffic.md). Ausführliche Informationen zum Begrenzen der Bandbreite finden Sie unter [erstellen oder Ändern von Bandbreitenrichtlinien Profilen in lync Server 2013](lync-server-2013-creating-or-modifying-bandwidth-policy-profiles.md). Ausführliche Informationen zu den Infrastrukturanforderungen finden Sie unter Anforderungen an die [Netzwerkinfrastruktur für lync Server 2013](lync-server-2013-network-infrastructure-requirements.md).

Um die Konferenzfunktionen verwenden zu können, müssen für lync Server 2013 bestimmte Ports geöffnet sein. Ausführliche Informationen zu Port Anforderungen finden Sie unter [Ports und Protokolle für interne Server in lync Server 2013](lync-server-2013-ports-and-protocols-for-internal-servers.md). Ausführliche Informationen zum Konfigurieren von Ports finden Sie unter [Configuring Port Ranges in lync Server 2013 für Ihre Konferenz-, Anwendungs-und Vermittlungsserver](lync-server-2013-configuring-port-ranges-for-your-conferencing-application-and-mediation-servers.md).
