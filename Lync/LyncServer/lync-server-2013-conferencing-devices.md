# <a name="conferencing-devices-in-lync-server-2013"></a>Konferenzgeräte in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2014-02-05_

Konferenzgeräte bringen lync Phone Edition Funktionen in Konferenzräume, sodass Personen im Raum hören können und je nach Gerät Personen an anderen Standorten angezeigt werden.

In den Themen in diesem Abschnitt erfahren Sie, wie Sie Ihre Konferenzgeräte einrichten und verwalten können.
