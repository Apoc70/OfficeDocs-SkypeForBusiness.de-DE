_**Letztes Änderungsstand des Themas:** 2012-06-27_

Die Cmdlets für die Interoperabilität werden verwendet, um die Server-zu-Server-Authentifizierung und-Autorisierung zwischen Microsoft lync Server 2013 und anderen Serverprodukten wie Microsoft Exchange Server 2013 zu konfigurieren. Die Server-zu-Server-Authentifizierung und die Autorisierung ermöglichen diesen Servern das nahtlose Austauschen und Freigeben von Daten.
