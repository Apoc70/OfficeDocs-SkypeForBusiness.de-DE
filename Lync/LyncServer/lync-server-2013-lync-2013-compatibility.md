## <a name="public-instant-messaging-clients-and-lync-2013"></a>Öffentliche Instant Messaging-Clients und-lync 2013

Wenn Sie den Server für die Verbindung mit öffentlichen Chat Diensten konfiguriert haben, unterstützt lync die folgenden Funktionen mit öffentlichen Chat Netzwerken. Der Anwesenheitsstatus wird nach den vom öffentlichen Sofortnachrichtenclient unterstützten Anwesenheitsstatus gefiltert. Ausführliche Informationen finden Sie unter [Planning for Public Instant Messaging Connectivity in lync Server 2013](lync-server-2013-planning-for-public-instant-messaging-connectivity.md) in der Dokumentation zur Planungsdokumentation und [Verwalten von Richtlinien für den externen Zugriff in lync Server 2013](lync-server-2013-manage-external-access-policy-for-your-organization.md) in der Betriebsdokumentation.

Darüber hinaus ermöglicht das XMPP-Integrations Feature von lync Server 2013 Benutzern das Austauschen von Chatnachrichten und Anwesenheitsinformationen mit Benutzern von öffentlichen Instant Messaging-Anbietern, die das Extensible Messaging and Presence Protocol (beispielsweise Google Talk) verwenden. Ausführliche Informationen finden Sie unter [Planning for Extensible Messaging and Presence Protocol (XMPP) Federation in lync Server 2013](lync-server-2013-planning-for-extensible-messaging-and-presence-protocol-xmpp-federation.md) in der Planungsdokumentation.
