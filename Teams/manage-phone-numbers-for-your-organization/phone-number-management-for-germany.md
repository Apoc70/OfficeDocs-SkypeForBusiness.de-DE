# <a name="phone-number-management-for-germany"></a>Telefonnummernverwaltung für Deutschland

Verwenden Sie diese Tabelle, um Informationen zum Abrufen und Verwalten von Telefonnummern in Deutschland für Microsoft Teams zu finden. 
  
Weitere Informationen finden Sie unter [Verwalten von Telefonnummern für Ihre Organisation](manage-phone-numbers-for-your-organization.md).
  
|**Was möchten Sie tun?**|**Laden Sie die richtigen Formulare herunter.**|
|:-----|:-----|
|Neue Telefonnummern für Benutzer oder Dienstleistungen abrufen, die nicht im Microsoft Teams Admin Center aufgeführt sind.  <br/> | In Deutschland können Sie über das Microsoft Teams Admin Center keine Telefonnummern für Ihre Organisation erhalten. Sie müssen eine Anforderung für neue Telefonnummern senden. Laden Sie zunächst dieses Formular herunter. <br/> [Neue Telefonnummern Anfrage für Deutschland (v. 4.0) (de-de)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-germany-(v.4.0)-(de-de).pdf?raw=true) <br/> [Neue Telefonnummern Anfrage für Deutschland (v. 4.0) (en-US)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-germany-(v.4.0)-(en-us).pdf?raw=true) </br>Füllen Sie das Formular aus und [Senden Sie es an uns](mailto:ptneu@microsoft.com).  <br/> |
|Holen Sie sich Telefonnummern, die an uns übertragen werden.  <br/> | Laden Sie eines dieser Formulare herunter. <br/> Für Benutzernummern: <br/> [Schriftliche Vollmacht für Deutschland (ortsgebundene Nummern) (v.1.0) (de-de)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-germany-(geographic-numbers)-(v.1.0)-(de-de).pdf?raw=true) <br/><br/> Für gebührenfreie Nummern: <br/> [Letter of Authorization (LoA) für Deutschland (gebührenfreie Nummern) (v. 2.0) (de-de)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-germany-(toll-free-numbers)-(v.2.0)-(de-de).pdf?raw=true) <br/><br/> Füllen Sie das Formular aus, und [senden Sie es an uns](mailto:ptneu@microsoft.com).  <br/> |
|Veröffentlichen/Aufheben der Veröffentlichung von Telefonnummern im nationalen Telefonverzeichnis.  <br/> |[Senden Sie uns](mailto:ptneu@microsoft.com) eine Liste der Telefonnummern, die veröffentlicht bzw. nicht mehr veröffentlicht werden sollen. <br/> |
|Ändern Sie die Nummer eines Benutzers in eine Dienstnummer.  <br/> |[Senden Sie uns](mailto:ptneu@microsoft.com) eine Liste der Benutzernummern, die Sie in Servicenummern umwandeln möchten. <br/> |
