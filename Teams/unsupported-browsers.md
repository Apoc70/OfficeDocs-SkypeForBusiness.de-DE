<span data-ttu-id="e4f7b-104">Einige Browser, wie Internet Explorer 11, Safari und Firefox, unterstützen die Microsoft Teams Web-App, unterstützen jedoch nicht einige der Funktionen für Anrufe und Besprechungen von Teams.</span><span class="sxs-lookup"><span data-stu-id="e4f7b-104">Some browsers, such as Internet Explorer 11, Safari, and Firefox, support the Microsoft Teams web app but don't support some of the Teams calling and meeting features.</span></span> <span data-ttu-id="e4f7b-105">Um diese Einschränkung zu umgehen, können Benutzer mit der Team-Web-App Audio über eine PSTN-Verbindung empfangen, sodass Sie die vorgestellten Inhalte (Bildschirmfreigabe) zu einer reduzierten Anzeigerate anzeigen können.</span><span class="sxs-lookup"><span data-stu-id="e4f7b-105">To work around this limitation, the Teams web app lets users receive audio through a PSTN connection and lets them view presented content (screen share) at a reduced display rate.</span></span>

<span data-ttu-id="e4f7b-106">Wenn Teams einen nicht unterstützten Browser erkennt, wird automatisch eine Meldung angezeigt, in der das Problem und die Sitzungs Einschränkungen erläutert werden.</span><span class="sxs-lookup"><span data-stu-id="e4f7b-106">When Teams detects an unsupported browser, it automatically displays a message explaining the issue and the session limitations.</span></span> <span data-ttu-id="e4f7b-107">Die Nachricht enthält weitere Anweisungen für den Zugriff auf die Audiofunktionen der Besprechung, wie etwa die Benachrichtigung des Benutzers, eine Rückrufnummer zu hinterlassen, damit Teams den Benutzer anrufen oder den Benutzer anweisen können, die in der Besprechungseinladung enthaltene Konferenznummer anzurufen.</span><span class="sxs-lookup"><span data-stu-id="e4f7b-107">The message provides further instructions for accessing the meeting audio, such as advising the user to leave a call back number so that Teams can call the user, or instructing the user to call the conference number included in the meeting invitation.</span></span> <span data-ttu-id="e4f7b-108">Die Nachricht ermutigt den Benutzer auch zum herunterladen und Verwenden des [Teams-Desktop Clients](https://teams.microsoft.com/downloads) für die gesamte Team Erfahrung.</span><span class="sxs-lookup"><span data-stu-id="e4f7b-108">The message also encourages the user to download and use the [Teams desktop client](https://teams.microsoft.com/downloads) for the full Teams experience.</span></span>

<span data-ttu-id="e4f7b-109">Wenn PSTN nicht verfügbar ist, werden die Anweisungen für den Zugriff auf die Besprechung nicht angezeigt und können nicht an der Besprechung teilnehmen.</span><span class="sxs-lookup"><span data-stu-id="e4f7b-109">If PSTN is unavailable, the user won't see the instructions for accessing the meeting and won't be able to join the meeting.</span></span>
