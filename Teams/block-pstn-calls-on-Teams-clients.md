# <a name="understand-how-users-can-block-pstn-calls-on-microsoft-teams-desktop-and-mobile-clients"></a>Verstehen, wie Benutzer PSTN-Anrufe auf Microsoft Teams-Desktop-und mobilen Clients blockieren können

Wenn Benutzer Anrufe empfangen, die Sie nicht auf Ihren Microsoft Teams-Desktop-oder mobilen (IOS-oder Android-) Clients erhalten möchten, können Sie das Feature "Blockieren von Anrufen" verwenden, um zukünftige Anrufe des Absenders zu verhindern. Benutzer können auch Ihre blockierte Anrufliste verwalten.

Mit diesem Feature können Benutzer die folgenden Anrufarten reduzieren:

- Unbekannte Anrufer (keine Rufnummernanzeige)
- Spam-Anrufe
- Robo-Anrufe
- Anrufe von Telemarketing-Nutzern
- Anrufe von anderen Personen, mit denen Sie nicht sprechen möchten.

Weitere Informationen und Schritte zum Blockieren von Anrufern finden Sie unter [Verwalten der Anrufeinstellungen in Teams](https://support.office.com/article/Manage-your-call-settings-in-Teams-456cb611-3477-496f-b31a-6ab752a7595f).
