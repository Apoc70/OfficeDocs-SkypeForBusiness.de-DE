# <a name="toll-free-dialing-restrictions-in-mexico"></a>Einschränkungen für gebührenfreie Telefonnummern in Mexiko

[Suchen Sie nach einem anderen Land oder einer anderen Region](../toll-free-dialing-limitations-and-restrictions.md)oder erfahren Sie mehr über gebührenfreie Wähleinschränkungen und-Einschränkungen.


|**Details**|**Beschreibung**|
|:-----|:-----|
|Sind gebührenfreie Anrufe verfügbar?  <br/> |Ja  <br/> |
|Wählformate  <br/> | 001 800 XXX XXXX <br/>  01 800 XXX XXXX <br/> |
|ISO-Code des Lands/der Region  <br/> |MX  <br/> |
|Abgedeckte Inseln/Gebiete  <br/> |Nicht zutreffend  <br/> |
|Nicht abgedeckte Inseln/Gebiete  <br/> |Nicht zutreffend  <br/> |
|Verfügbarkeit über drahtgebundene Leitungen/Festnetz  <br/> |Partiell. Siehe Kommentare.  <br/> |
|Verfügbarkeit über Mobilnetze  <br/> |Partiell. Siehe die Kommentare.  <br/> |
|Kommentare, Einschränkungen und Zugriffsaspekte  <br/> |Der Zugriff über internationales Roaming ist nicht garantiert.  <br/> Für das Nummernformat 001-800: Für Zugriff über Festnetz - Telmex. Für Zugriff über Mobilnetz - nur Telcel. Die Nutzungszeit wird berechnet. Keine zusätzliche Gebühr, wenn der Endkunde mit Telmex als Orts-und fern-Anbieter arbeitet.  <br/> Für das Nummernformat 01-800: Für den Zugriff über Mobilnetze wird die Nutzungszeit berechnet.  <br/> |
