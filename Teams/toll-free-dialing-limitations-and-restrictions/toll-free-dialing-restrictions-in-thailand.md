# <a name="toll-free-dialing-restrictions-in-thailand"></a>Einschränkungen für gebührenfreie Telefonnummern in Thailand

[Suchen Sie nach einem anderen Land oder einer anderen Region](../toll-free-dialing-limitations-and-restrictions.md)oder erfahren Sie mehr über gebührenfreie Wähleinschränkungen und-Einschränkungen.


|**Details**|**Beschreibung**|
|:-----|:-----|
|Sind gebührenfreie Anrufe verfügbar?  <br/> |Ja  <br/> |
|Wählformate  <br/> | 001 800 XXX XXX XXXX <br/>  1-800-XXX-XXX <br/> |
|ISO-Code des Lands/der Region  <br/> |TH  <br/> |
|Abgedeckte Inseln/Gebiete  <br/> |Nicht zutreffend  <br/> |
|Nicht abgedeckte Inseln/Gebiete  <br/> |Nicht zutreffend  <br/> |

|Verfügbarkeit über Mobilnetze  <br/> |Ja. Siehe Kommentare.  <br/> |
|Kommentare, Einschränkungen und Zugriffsaspekte  <br/> |Mobilgeräte-Zugriff: Die Übertragungszeit wird nicht berechnet, außer für Roamingdienste, für die sie als lokale Übertragungszeit berechnet wird. Für Mobilgeräte-Zugriff: Zugriff von allen Mobilgeräte-Netzwerke.  <br/> |
