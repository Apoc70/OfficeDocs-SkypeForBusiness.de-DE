# <a name="schema-attributes-and-descriptions-in-skype-for-business-server"></a>Schemaattribute und Beschreibungen in Skype für Business Server
 
Dieser Abschnitt beschreibt alle Schemaattribute von Skype für Business Server verwendet wird. Die Klassen im Zusammenhang mit Attributen finden Sie unter [Schemaattribute nach Klasse in Skype für Business Server](schema-attributes-by-class.md).
  
Attribute, die verknüpfte Paare handelt, werden als vorwärts verweisende oder rückverweisende Links angegeben. Ein Attribut, das auf ein anderes Objekt verwiesen wird, ist ein Forwardlink; Das Attribut des Objekts, das wieder auf das erste Objekt verweist ist eine Verknüpfung an. Vorwärts verweisende Links können aktualisiert werden, während rückverweisende Links schreibgeschützt sind.
  
Einige Attribute weisen eine Bitmaske. Bei diesen Attributen wird jede Einstellung durch ein Bit dargestellt, und der angezeigte Dezimalwert steht für die Bitposition. Bit Positionen beginnt mit 0 Bit. Beispiel 1 (binär) ist 0 Bit ist festgelegt und 10000 (binär) Bit 4 ist festgelegt ist. Jedes Bit stellt eine Eigenschaft dar. Es folgen einige Beispiele: 
  
- 10000 (binär) hat den Dezimalwert 16 (d. h., Bit 4 ist festgelegt).
    
- 100000000 (binär) hat den Dezimalwert 256 (d. h., Bit 8 ist festgelegt).
    
- 1100 (binär) hat den Dezimalwert 12 (d. h., die Bits 2 und 3 sind festgelegt; durch beide Bits dargestellte Eigenschaften sind aktiviert).
    
- 1111000001 (binär) hat den Dezimalwert 961 (d. h., die Bits 0, 6, 7, 8 und 9 sind festgelegt; Eigenschaften für diese Bits sind aktiviert).
