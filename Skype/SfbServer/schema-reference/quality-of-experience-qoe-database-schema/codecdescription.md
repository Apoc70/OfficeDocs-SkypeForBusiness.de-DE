# <a name="codecdescription-table"></a>CodecDescription-Tabelle
 
CodecDescription-Tabelle ordnet eindeutige Codec-IDs dem entsprechenden Codec zu. Codecs dienen zum Codieren digitale Signale für die Übertragung und Übertragung, und klicken Sie dann zum Decodieren Signale für die Wiedergabe. Diese Tabelle wurde in Microsoft Lync Server 2013 eingeführt.
  
|**Spalte**|**Datentyp**|**Schlüssel/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**Codecdescriptionkey entspricht** <br/> |smallint  <br/> |Primary  <br/> |Eindeutiger Bezeichner, die dem Codec zugewiesen.  <br/> |
|**CodecDescription** <br/> |varchar(256)-Wert  <br/> |Eindeutige  <br/> |Eindeutige Beschreibung des Codecs entspricht dem codecdescriptionkey entspricht.  <br/> |
