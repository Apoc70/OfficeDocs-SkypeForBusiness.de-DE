|:-----|:-----|:-----|:-----|
|**Region1Key** <br/> |int  <br/> |Primär, Fremd  <br/> |Verweis von der [Region Table](region.md).  <br/> |
|**Region2Key** <br/> |int  <br/> |Primär, Fremd  <br/> |Verweis von der [Region Table](region.md).  <br/> |
