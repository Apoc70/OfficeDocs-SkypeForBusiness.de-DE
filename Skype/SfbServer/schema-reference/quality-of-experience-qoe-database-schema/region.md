# <a name="region-table"></a>Region-Tabelle
 
Die Region-Tabelle ist eine Tabelle. Jeder Datensatz steht für eine Land/Region, die in der Einstellung der Netzwerk-Konfiguration.
  
|**Spalte**|**Datentyp**|**Schlüssel/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**RegionKey** <br/> |int  <br/> |Primary  <br/> |Eindeutige Zahl, die das Land/Region identifiziert.  <br/> |
|**RegionName** <br/> |nvarchar(128)  <br/> |Eindeutige  <br/> |Der Name des Lands/der Region.  <br/> |
