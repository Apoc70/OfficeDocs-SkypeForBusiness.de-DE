# <a name="networkconfigurationsettings-view"></a>NetworkConfigurationSettings-Ansicht
 
NetworkConfigurationSettings-Ansicht speichert Informationen über die Netzwerkkonfiguration. Diese Ansicht wurde in Microsoft Lync Server 2013 eingeführt.
  
|**Spalte**|**Datentyp**|**Details**|
|:-----|:-----|:-----|
|SubnetIP  <br/> |int  <br/> |Subnetz-IP-Adresse (nur IPv4-Adressen werden unterstützt).  <br/> |
|UserSiteName  <br/> |nvarchar(128)  <br/> |Site-Name des Benutzers.  <br/> |
|RegionName  <br/> |nvarchar(128)  <br/> |Name des Lands/der Region des Benutzers.  <br/> |
