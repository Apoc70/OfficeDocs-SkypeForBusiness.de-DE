# <a name="manufacturers-table-in-skype-for-business-server-2015"></a>Tabelle "Hersteller" in Skype for Business Server 2015
 
Die Tabelle "Hersteller" ist eine unterstützende Tabelle. Jeder Datensatz speichert Informationen zu einem Gerät (Telefonapparat) Hersteller.
  
|**Spalte**|**Datentyp**|**Schlüssel/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**ManufacturerId** <br/> |int  <br/> |Primary  <br/> |Eindeutige Nummer, die diesen Hersteller kennzeichnet.  <br/> |
|**Hersteller** <br/> |nvarchar(256)  <br/> | <br/> |Herstellername.  <br/> |
