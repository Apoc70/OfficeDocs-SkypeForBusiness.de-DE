# <a name="manufacturers-table-in-skype-for-business-server-2015"></a>Manufacturers-Tabelle in Skype für Business Server 2015
 
Manufacturers-Tabelle ist eine Tabelle. Jeder Datensatz speichert Informationen zu einem Gerätehersteller (Telefonapparat).
  
|**Spalte**|**Datentyp**|**Schlüssel/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**ManufacturerId** <br/> |int  <br/> |Primary  <br/> |Eindeutige Zahl, die diesen Hersteller identifiziert.  <br/> |
|**Hersteller** <br/> |nvarchar(256)  <br/> | <br/> |Name des Herstellers.  <br/> |
