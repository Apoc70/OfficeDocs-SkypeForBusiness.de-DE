# <a name="edgeservers-table-in-skype-for-business-server-2015"></a>EdgeServers-Tabelle in Skype for Business Server 2015
 
Die Tabelle EdgeServers ist eine unterstützende Tabelle. Jeder Datensatz speichert Informationen zu einem Edgeserver, der an anrufen beteiligt ist, die Datensätze in der Datenbank aufweisen.
  
|**Spalte**|**Datentyp**|**Schlüssel/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**EdgeServerId** <br/> |int  <br/> |Primary  <br/> |Eindeutige Nummer, die diesen Edgeserver kennzeichnet.  <br/> |
|**EdgeServer** <br/> |nvarchar(256)  <br/> | <br/> |Name des Edge-Servers.  <br/> |
