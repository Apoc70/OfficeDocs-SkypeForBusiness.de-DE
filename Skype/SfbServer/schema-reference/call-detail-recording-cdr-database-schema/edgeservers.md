# <a name="edgeservers-table-in-skype-for-business-server-2015"></a>EdgeServers-Tabelle in Skype für Business Server 2015
 
EdgeServers-Tabelle ist eine Tabelle. Jeder Datensatz speichert Informationen über einen Edge-Server, die beteiligt ist in Anrufe, die Datensätze in der Datenbank verfügen.
  
|**Spalte**|**Datentyp**|**Schlüssel/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**EdgeServerId** <br/> |int  <br/> |Primary  <br/> |Eindeutige Zahl, die diesen Edgeserver identifiziert.  <br/> |
|**EdgeServer** <br/> |nvarchar(256)  <br/> | <br/> |Name des Edgeservers.  <br/> |
