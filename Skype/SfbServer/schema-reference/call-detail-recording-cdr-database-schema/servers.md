# <a name="servers-table"></a>Servers-Tabelle
 
Die Tabelle Server ist eine unterstützende Tabelle, in der Informationen zu den verschiedenen Servern gespeichert werden. Jeder Datensatz in der Tabelle steht für einen Server.
  
|**Spalte**|**Datentyp**|**Schlüssel/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**ServerID** <br/> |int  <br/> |Primary  <br/> |Eindeutige Nummer, die diesen Server identifiziert.  <br/> |
|**ServerFQDN** <br/> |nvarchar(256)  <br/> | <br/> |Server-FQDN  <br/> |
