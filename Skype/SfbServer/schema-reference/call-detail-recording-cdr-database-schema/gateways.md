# <a name="gateways-table-in-skype-for-business-server-2015"></a>Tabelle "Gateways" in Skype for Business Server 2015
 
Die Tabelle Gateways ist eine unterstützende Tabelle. Jeder Datensatz speichert Informationen zu einem Gateway, das an PSTN-anrufen (Public Switched Telephone Network) beteiligt ist, die Datensätze in der Datenbank aufweisen.
  
|**Spalte**|**Datentyp**|**Schlüssel/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**Gatewayserver** <br/> |int  <br/> |Primary  <br/> |Eindeutige Nummer, die dieses Gateway kennzeichnet.  <br/> |
|**Gateway** <br/> |nvarchar(256)  <br/> | <br/> |Gateway-Name.  <br/> |
