# <a name="gateways-table-in-skype-for-business-server-2015"></a>Gateways-Tabelle in Skype für Business Server 2015
 
Die Gateways-Tabelle ist eine Tabelle. Jeder Datensatz speichert Informationen über ein Gateway, die beteiligt ist im öffentlichen Telefonnetz (Network, PSTN) anrufen, die Datensätze in der Datenbank verfügen.
  
|**Spalte**|**Datentyp**|**Schlüssel/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**GatewayId** <br/> |int  <br/> |Primary  <br/> |Eindeutige Zahl, die dieses Gateway identifiziert.  <br/> |
|**Gateway** <br/> |nvarchar(256)  <br/> | <br/> |Gatewayname  <br/> |
