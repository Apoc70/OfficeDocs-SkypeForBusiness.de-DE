# <a name="mcus-view"></a>MCUs-Ansicht
 
Mcus-Ansicht speichert Informationen zu den MCUs, die konferenzsitzungen teilgenommen haben. Diese Ansicht wurde in Microsoft Lync Server 2013 eingeführt.
  
|**Spalte**|**Datentyp**|**Details**|
|:-----|:-----|:-----|
|**McuId** <br/> |int  <br/> |Eindeutige Zahl, die die MCU identifiziert.  <br/> |
|**McuUri** <br/> |nvarchar(450)  <br/> |Der URI der MCU.  <br/> |
|**McuUriType** <br/> |nvarchar(256)  <br/> |URI der MCU-Typ. Finden Sie weitere Informationen der [UriTypes-Tabelle](uritypes.md) . <br/> |
