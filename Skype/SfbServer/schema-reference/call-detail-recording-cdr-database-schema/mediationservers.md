# <a name="mediationservers-table"></a>MediationServers-Tabelle
 
Die MediationServers-Tabelle ist eine Tabelle. Jeder Datensatz speichert Informationen über einen Vermittlungsserver, die beteiligt ist in Anrufe, die Datensätze in der Datenbank verfügen.
  
|**Spalte**|**Datentyp**|**Schlüssel/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**MediationServerId** <br/> |int  <br/> |Primary  <br/> |Eindeutige Zahl, die diesen Vermittlungsserver identifiziert.  <br/> |
|**MediationServer** <br/> |nvarchar(256)  <br/> | <br/> |Name des Vermittlungsservers.  <br/> |
