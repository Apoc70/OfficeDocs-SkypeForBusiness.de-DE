# <a name="conferenceuris-view"></a>ConferenceUris-Ansicht
 
In der ConfernceUris-Ansicht werden Informationen zu den URIs gespeichert, die an Konferenzsitzungen teilgenommen haben. Diese Ansicht wurde in Microsoft lync Server 2013 eingeführt.
  
|**Spalte**|**Datentyp**|**Details**|
|:-----|:-----|:-----|
|ConferenceUriId  <br/> |int  <br/> |Eindeutige Zahl, die den Konferenz-URI kennzeichnet.  <br/> |
|ConferenceUri  <br/> |nvarchar (450)  <br/> |URI der Konferenz.  <br/> |
|ConferenceUriType  <br/> |nvarchar(256)  <br/> |Typ des Konferenz-URI. Weitere Informationen finden Sie in der [UriTypes-Tabelle](uritypes.md) . <br/> |
