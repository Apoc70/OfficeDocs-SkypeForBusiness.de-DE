# <a name="pools-table"></a>Pools-Tabelle
 
Die Pools-Tabelle ist eine unterstützende Tabelle, die Informationen zu den verschiedenen Pool gespeichert. Jeder Datensatz in der Tabelle steht für einen Pool.
  
|**Spalte**|**Datentyp**|**Schlüssel/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**PoolId** <br/> |int  <br/> |Primary  <br/> |Eindeutige Zahl, die diesen Pool identifiziert.  <br/> |
|**"Poolfqdn"** <br/> |nvarchar(256)  <br/> | <br/> |FQDN des Pools.  <br/> |
