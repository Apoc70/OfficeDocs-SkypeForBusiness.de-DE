# <a name="uritypes-table"></a>UriTypes-Tabelle
 
UriTypes-Tabelle sind die unterschiedlichen URI (Uniform Resource Identifier) Typen in Skype für Business Server 2015 überwacht.
  
|**Spalte**|**Datentyp**|**Schlüssel/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**UriTypeId** <br/> |tinyint  <br/> |Primary  <br/> |Eindeutiger Bezeichner, einem URI-Typ zugewiesen sind.  <br/> |
|**UriType** <br/> |nvarchar(256)  <br/> || Eine Beschreibung der verschiedenen URI. Zulässige Werte: <br/>  0 – Telefon-Uri <br/>  1 – Benutzer-Uri <br/> |
