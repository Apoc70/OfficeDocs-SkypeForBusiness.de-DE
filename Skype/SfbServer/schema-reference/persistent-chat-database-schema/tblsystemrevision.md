# <a name="tblsystemrevision"></a>"tblsystemrevision"
 
"tblsystemrevision" enthält die Revisionsnummer, die mit der Adminlock-Tabelle verwendet wird, um die Konsistenz übergreifend über mehrere administratorclients zu erzielen.
  
**Spalten**

|**Spalte**|**Typ**|**Beschreibung**|
|:-----|:-----|:-----|
|sysRevision  <br/> |Bigint, nicht null  <br/> |Revisionsnummer.  <br/> |
