# <a name="requirements-for-your-skype-for-business-2015-environment"></a>Voraussetzungen für Ihre Skype for Business 2015-Umgebung
 
**Zusammenfassung:** Beschreibung der Schritte für Anforderungen des Systems, des Netzwerk und der Infrastruktur für Skype for Business Server 2015. Wenn Sie vor dem Installieren und Bereitstellen von Skype for Business Server 2015 eine andere Komponente installieren oder konfigurieren müssen, finden Sie diese hier.
  
Dies ist die Startseite für Artikel zu den Anforderungen in Bezug auf eine Skype for Business Server 2015-Bereitstellung. Dazu gehören sowohl Server Anforderungen als auch allgemeinere Vorbereitungen, einschließlich ihrer Domänen-und Netzwerkstruktur und Zertifikate.
  
## <a name="whats-in-this-topic"></a>Inhalt dieses Themas

In den beiden Artikeln im Abschnitt Anforderungen für Ihre Umgebung wird erläutert, was Sie benötigen, und/oder konfigurieren, bevor Sie mit der erfolgreichen Bereitstellung oder dem Upgrade auf Skype for Business Server 2015 in der Server Umgebung Ihres Unternehmens fortfahren können.
  
### <a name="server-requirements"></a>Serveranforderungen

[Server Anforderungen für Skype for Business Server 2015](server-requirements.md) sind die Dinge, die Sie benötigen, um sich auf den einzelnen Servern vorzubereiten, die Sie in Ihrer Bereitstellung verwenden möchten.
  
### <a name="environmental-requirements"></a>Anforderungen an die Umgebung

Die [Umweltanforderungen für Skype for Business Server 2015](environmental-requirements.md) sind ein breiteres Spektrum an Aufgaben, die vor der Bereitstellung von Skype for Business Server 2015 in Ihrer Infrastruktur bereitgestellt werden müssen.
