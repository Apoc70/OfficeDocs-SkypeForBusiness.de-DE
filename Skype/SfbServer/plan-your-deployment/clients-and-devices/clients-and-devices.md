# <a name="plan-for-clients-and-devices"></a>Planen für Clients und Geräte
 
**Zusammenfassung:** Überprüfen der unterstützten Clients und apps für Skype für Unternehmen.
  
Heutzutage sind Arbeitskräfte ständig unterwegs. Mitarbeiter müssen kommunizieren und zusammenarbeiten können – unabhängig davon, ob sie am Unternehmensstandort, in regionalen Niederlassungen, von zu Hause aus oder unterwegs arbeiten. Skype für Business Server 2015 unterstützt diese Anforderungen über eine Auflistung von Clientschnittstellen, die Sie für die Benutzer Ihrer Organisation bereitstellen können. Sorgfältige Planung wird sichergestellt, dass die Mitarbeiter erhalten, was sie benötigen und Skype für Unternehmen zur Verfügung ist dorthin, wo sie waren. 
  
## <a name="available-clients"></a>Verfügbare Clients

Skype für Business Server 2015 unterstützt verschiedene Arten von Clients, einschließlich Computern installierte Clientsoftware, webbasierte Clients und Clients für mobile Geräte. In diesem Abschnitt wird jeder eingeführt. Wenn Sie bereits eine Kombination von Lync-Clients verwendet haben, beachten Sie, dass es nicht unterstützter [ältere Clients](clients-and-devices.md#Legacy) , die nicht mit Skype für Unternehmen kompatibel sind. Updates stattfinden regelmäßig so überprüfen regelmäßig die neuesten Clientinformationen in diesem Thema.

Skype für Business 2016 ist der empfohlene mit vollem Funktionsumfang Client für Skype für Business Server 2015. ### <a name="online-meeting-add-in-for-skype-for-business"></a>Onlinebesprechungs-Add-in für Skype für Unternehmen

Das Onlinebesprechungs-Add-in für Skype für Unternehmen unterstützt besprechungsverwaltung aus Microsoft Outlook-Client für messaging und Zusammenarbeit. Das Onlinebesprechungs-Add-in für Skype für Unternehmenssoftware wird automatisch mit Skype für Unternehmen installiert.
  
### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Skype für Business Web App und Skype Besprechungen App

Skype für Web-Geschäfts-App ist die webbasierte Browser-Plug-Ins Konferenzclient für Skype für Business Server 2015 Besprechungen. Besprechungsteilnehmer können auf alle Funktionen für die Zusammenarbeit und Freigabe sowie auf alle Besprechungssteuerelemente für Referenten zugreifen. Dieser Client kann auch Gäste von außerhalb Ihrer Organisation, die keinen regelmäßigen Client für die Teilnahme an Besprechungen, die von Ihrer Skype für Unternehmensbenutzer erstellt haben. 
  
Wenn Skype für Unternehmen auf dem Computer eines Benutzers nicht installiert ist, und der Benutzer klickt auf einen Link zur Besprechung in einer Besprechungsanfrage, Skype für Web-Geschäfts-App installiert werden, und öffnen. Sie können auch die Teilnahme an einer Besprechung-Seite, um Benutzern an Besprechungen teilnehmen, die mit früheren Versionen von Clients konfigurieren. Skype für Web-Geschäfts-App ist der Client Wahl für Teilnehmer außerhalb Ihrer Organisation. Mit Skype für Web-Geschäfts-App ist keine lokalen Client-Installation erforderlich, obwohl audio, video und Freigabe Features ein plug-in bei der ersten Verwendung installiert. Finden Sie unter [Planen von Besprechungen-Clients (Web App und Besprechungen App)](meetings-clients.md) für die Anforderungen für die Verwendung dieser Clients.
  
Die Skype Besprechungen App ist eine direkte Windows 10-app, die nur einen Browser während der Installation verwendet und als systemeigenen Windows-10-app ausgeführt wird. Es erfordert bestimmte Serverkonfiguration Verteilung vom Front-End-Server zu aktivieren. Wie Skype für Web-Geschäfts-App ermöglicht dieser Client auch Gäste von außerhalb Ihrer Organisation, die keinen regelmäßigen Client für die Teilnahme an Besprechungen, die von Ihrer Skype für Unternehmensbenutzer erstellt haben. 
  
### <a name="skype-for-business-web-scheduler"></a>Skype für Business-Webplaner

Skype für Business Webplaner ist ein webbasierter meeting-Tool zum Planen und verwalten für Benutzer, die keinen Zugriff auf Microsoft Outlook, oder auf einem Betriebssystem nicht auf Windows basierenden stehen. Benutzer können mit Skype für Business Webplaner neue Besprechungen erstellen, ändern vorhandene Besprechungen und Senden von Einladungen mithilfe ihrer bevorzugten e-Mail-Programm.
  
[Skype für Business Webplaner](https://support.office.com/en-us/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US) -Dokumentation enthält weitere Details.
  
### <a name="vdi-plugins"></a>VDI-Plug-Ins

Eine VDI-Umgebung (Virtual Desktop Infrastructure) wird in einigen Organisationen verwendet, in denen Sicherheit und Complianceprobleme besonders sensibel sind. Verwendung von Skype für Unternehmen mit vollständigen Audio- und Videodaten bei einer Verbindung kann es muss mit dem starker Auslastung von Audio und Video Verarbeitung auf dem Client auf einem virtuellen Desktop verwaltet. Zusätzlicher VDI-Plug-in-Software verfügbar ist, die die Verarbeitung auf dem lokalen Computer des Endbenutzers verlagert und reduziert die Last auf dem virtuellen Desktop. Einzelheiten zur Verwendung dieser-Plug-Ins finden Sie unter [Skype für Unternehmen in einer VDI-Umgebung planen](vdi-environments.md) .
  
### <a name="skype-room-systems-v2"></a>Skype Room Systems V2

Skype Raum Systemen v2 ist Microsofts neueste Lösung für Webkonferenzen verwendet eine vertraute Skype für Business-Schnittstelle und wird auf einfache Weise bereitgestellt und verwaltet, nutzen vorhandene Geräte wie LCD-Bereiche zur Erleichterung der Installation. Skype-Chatroom-Systemen v2 verwendet eine spezielle UWP app auf einem Surface Pro 4 oder Surface Pro in einem Konsolenmodus ausgeführt (nach der Bereitstellung der app UWP ist die einzige app, die auf dem Gerät ausgeführt wird) und ein eigenen Konto Gerät auf Ihre Skype für Business Implementierung erfordert. Software wird über den Windows Store und Windows Update aktualisiert. Informationen zur Verwendung dieser Raum Konsolen in Ihrer Bereitstellung finden Sie unter [Planen von Skype Raum Systemen v2](skype-room-systems-v2-0.md) .
  
### <a name="skype-for-business-on-surface-hub"></a>Skype für Unternehmen auf Fläche Hub

Microsoft Surface Hub ist ein all-in-One Produktivität Gerät, das für die direkte Verwendung ist für Brainstorming, Zusammenarbeit und Präsentationen. Es wurde eine eigene Iteration der Skype für Business-Client im [Administratorhandbuch zu Microsoft Surface Hub](https://docs.microsoft.com/en-us/surface-hub/)dokumentiert.

## <a name="ways-to-deploy-the-client-to-your-users"></a>Methoden für die Bereitstellung des Clients für Ihre Benutzer
<a name="BK_User_Deploy"> </a>

Clientinstallationsprogramme sind sowohl als MSI-Datei als auch als Klick-und-Los-Version verfügbar. Die Skype für Client-Sustainment Unternehmensstrategie beeinträchtigen fertigzustellen, damit Sie die folgenden Punkte vertraut sein sollten: 
  
- Im Allgemeinen Skype für Unternehmen nicht neue Funktionen für Clients aus älteren hinzu
    
- Skype für Business plan im Allgemeinen nicht auf neue Features für Business MSI-Datei nach der ersten Version in der Skype ausgeliefert. MSI-Verbesserungen zwischen Versionen werden in erster Linie Qualität/Sicherheit Natur. 
    
- Die neuesten und interessantesten Skype Business Client wünschen wird in der Skype für Business 2016 Klick-und-Los-Installer gefunden werden. 
    
Führen Sie eine benutzerdefinierte Bereitstellung des Clients wie in [Skype für Business Server 2015 Anpassen von Windows-Clientinstallation](../../deploy/deploy-clients/customize-windows-client-installation.md)beschrieben. Installationsmethoden werden in [Bereitstellen von Clients für Skype für Business Server 2015](../../deploy/deploy-clients/deploy-clients.md) ausführlicher beschrieben.
  
### <a name="click-to-run"></a>Klick-und-Los

Klick-und-los ist ein Microsoft streaming und Virtualisierungstechnologie, die Sie zum Installieren und Aktualisieren von Office-Produkten, einschließlich Skype für Unternehmen verwenden können. Diese Streaming- und Virtualisierungsfunktionen basieren auf Technologien in Microsoft Application Virtualization (App-V). Klick-und-Los bietet die folgenden Vorteile: 
  
- Gestreamte Installation der Office-Suite, die den Zeitbedarf für die Installation verringert
    
- Slipstreamupdates und -patches
    
- Geringerer Platzbedarf auf dem Datenträger
    
- Benutzerbasierte Lizenzierung: fünf Installationen pro Benutzer
    
- Anpassbar über XML-Editor für die Installation von unabhängigen Programmen
    
Sie können das [Office-Bereitstellungstools für 2016](https://www.microsoft.com/en-us/download/details.aspx?id=49117) für diese Art von Installer verwenden möchten.
  
Sowohl die grundlegende und vollständige Clientversionen (mit Auswahl der 32- und 64-Bit-Versionen) sind mit einem Klick-und-Los-Installer verfügbar, die Benutzer können unter [Herunterladen Skype für Unternehmen für alle Ihre Geräte](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3)herunterladen. 
  
### <a name="msi"></a>MSI

MSI ist eine herkömmlichere Installationsmethode. Sie können Sie manuell installieren von Updates und Patches, Verwendung Volumenlizenzierung und Aktivierung und kann über das [Office-Anpassungstool](http://www.microsoft.com/en-us/download/details.aspx?id=49030)angepasst werden. Sie können Clients verteilen, indem Sie Gruppenrichtlinien anwenden, System Center Configuration Manager verwenden oder ein Tool eines Drittanbieters nutzen.
