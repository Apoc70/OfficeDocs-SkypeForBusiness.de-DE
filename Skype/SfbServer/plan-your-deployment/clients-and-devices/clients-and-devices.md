# <a name="plan-for-clients-and-devices"></a>Planen für Clients und Geräte

**Zusammenfassung:** Überprüfen Sie die unterstützten Clients und Apps für Skype for Business.

Heutzutage sind Arbeitskräfte ständig unterwegs. Mitarbeiter müssen kommunizieren und zusammenarbeiten können – unabhängig davon, ob sie am Unternehmensstandort, in regionalen Niederlassungen, von zu Hause aus oder unterwegs arbeiten. Skype für Business Server unterstützt diese Anforderungen über eine Auflistung von Clientschnittstellen, die Sie für die Benutzer Ihrer Organisation bereitstellen können. Sorgfältige Planung wird sichergestellt, dass die Mitarbeiter erhalten, was sie benötigen und Skype für Unternehmen zur Verfügung ist dorthin, wo sie waren.

## <a name="available-clients"></a>Verfügbare Clients

Skype für Business Server unterstützt verschiedene Arten von Clients, einschließlich Computern installierte Clientsoftware, webbasierte Clients und Clients für mobile Geräte. In diesem Abschnitt werden die wichtigsten Clients eingeführt, die eine ausführliche Liste aller unterstützten Clients finden Sie unter [Featurevergleich für Skype für Business Server 2015 Desktopclient](desktop-feature-comparison.md) oder [Desktopclient Featurevergleich für Skype für Business Server 2019](../../../SfBServer2019/plan/feature-comparison.md). Wenn Sie bereits eine Kombination von Lync-Clients verwendet haben, beachten Sie, dass es nicht unterstützter- [Legacy-Clients](clients-and-devices.md#Legacy) , die mit Skype für Business Server 2019 nicht kompatibel sind. Updates stattfinden regelmäßig so überprüfen regelmäßig die neuesten Clientinformationen in diesem Thema.

### <a name="vdi-plugins"></a>VDI-Plug-Ins

Eine VDI-Umgebung (Virtual Desktop Infrastructure) wird in einigen Organisationen verwendet, in denen Sicherheit und Complianceprobleme besonders sensibel sind. Verwendung von Skype für Unternehmen mit vollständigen Audio- und Videodaten bei einer Verbindung kann es muss mit dem starker Auslastung von Audio und Video Verarbeitung auf dem Client auf einem virtuellen Desktop verwaltet. Zusätzlicher VDI-Plug-in-Software verfügbar ist, die die Verarbeitung auf dem lokalen Computer des Endbenutzers verlagert und reduziert die Last auf dem virtuellen Desktop. Einzelheiten zur Verwendung dieser-Plug-Ins finden Sie unter [Skype für Unternehmen in einer VDI-Umgebung planen](vdi-environments.md) .

### <a name="skype-room-systems-v2"></a>Skype Room Systems V2

Skype Raum Systemen v2 ist Microsofts neueste Lösung für Webkonferenzen verwendet eine vertraute Skype für Business-Schnittstelle und wird auf einfache Weise bereitgestellt und verwaltet, nutzen vorhandene Geräte wie LCD-Bereiche zur Erleichterung der Installation. Skype-Chatroom-Systemen v2 verwendet eine spezielle UWP app auf einem Surface Pro 4 oder Surface Pro in einem Konsolenmodus ausgeführt (nach der Bereitstellung der app UWP ist die einzige app, die auf dem Gerät ausgeführt wird) und ein eigenen Konto Gerät auf Ihre Skype für Business Implementierung erfordert. Software wird über den Windows Store und Windows Update aktualisiert. Finden Sie unter https://aka.ms/SRSDocs detaillierte Informationen zur Verwendung dieser Raum Konsolen in Ihrer Bereitstellung.

## <a name="choosing-your-organizations-preferred-client"></a>Auswählen des bevorzugten Clients für Ihre Organisation

Wenn Ihre Organisation die entsprechenden Lizenzen erworben hat, wählen Sie den vollständigen Client, anderenfalls wählen Sie den Standardclient.

Ihre Benutzer können von [Skype für Unternehmen für alle Ihre Geräte herunterladen](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3)des Clients für sich selbst installieren. Der Client ist auch installiert, wenn Benutzer Office 365 auf Windows installieren. Wenn einige Ihrer Benutzer Macs haben, müssen diese Benutzer einen anderen Satz von Features, wie in den vorherigen Abschnitten beschrieben.

Einige Features sind mit Skype für Business Server 2015 stehen nicht in Skype Business Online oder Skype für Business Server 2019, finden Sie unter [Online oder Hybrid Benutzerkonto Einschränkungen für 2015](desktop-feature-comparison.md#Online-Hybrid) oder [Online oder Hybrid Benutzer Konto Einschränkungen für 2019 ](desktop-feature-comparison.md#Online-Hybrid)Einzelheiten.  Überprüfen Sie vor dem Bereitstellen oder auf Skype für Unternehmen aktualisieren, welche Clients bereits in Ihrer Organisation verwendet werden. Verwenden Sie den [Desktopclient Featurevergleich für Skype für Unternehmen](desktop-feature-comparison.md) , die Feature Support Auswirkungen auf die diese Clients zu verstehen. Dann können Sie den Benutzern die Änderungen vermitteln sowie das Tempo des Rollouts bestimmen und die Vorteile des Upgrades auf den neuesten Client in vollem Umfang verstehen.

## <a name="ways-to-deploy-the-client-to-your-users"></a>Methoden für die Bereitstellung des Clients für Ihre Benutzer
<a name="BK_User_Deploy"> </a>

Clientinstallationsprogramme sind sowohl als MSI-Datei als auch als Klick-und-Los-Version verfügbar. Die Skype für Client-Sustainment Unternehmensstrategie beeinträchtigen fertigzustellen, damit Sie die folgenden Punkte vertraut sein sollten:

- Im Allgemeinen werden zu bereits veröffentlichten Skype for Business-Clients KEINE neuen Funktionen hinzugefügt.

- Skype für Business plan im Allgemeinen nicht auf neue Features für Business MSI-Datei nach der ersten Version in der Skype ausgeliefert. MSI-Verbesserungen zwischen Versionen werden in erster Linie Qualität/Sicherheit Natur.

- Die neuesten und besten Skype for Business-Clientfunktionen sind im Klick-und-Los-Installationsprogramm für Skype for Business 2019 enthalten.

Sie können eine benutzerdefinierte Bereitstellung des Clients wie unter [Anpassen von Windows-Clientinstallation Skype für Business Server](../../deploy/deploy-clients/customize-windows-client-installation.md)durchführen. Installationsmethoden werden in [Bereitstellen von Clients für Skype für Business Server](../../deploy/deploy-clients/deploy-clients.md) ausführlicher beschrieben.

### <a name="click-to-run"></a>Klick-und-Los

Klick-und-los ist ein Microsoft streaming und Virtualisierungstechnologie, die Sie zum Installieren und Aktualisieren von Office-Produkten, einschließlich Skype für Unternehmen verwenden können. Diese Streaming- und Virtualisierungsfunktionen basieren auf Technologien in Microsoft Application Virtualization (App-V). Klick-und-Los bietet die folgenden Vorteile:

- Gestreamte Installation der Office-Suite, die den Zeitbedarf für die Installation verringert

- Slipstreamupdates und -patches

- Geringerer Platzbedarf auf dem Datenträger

- Benutzerbasierte Lizenzierung: fünf Installationen pro Benutzer

- Anpassbar über XML-Editor für die Installation von unabhängigen Programmen

Für diesen Installationsprogrammtyp können Sie das [Office-Bereitstellungstool](https://www.microsoft.com/en-us/download/details.aspx?id=49117) verwenden.

Sowohl die Standardclientversion als auch die Clientversion mit vollem Funktionsumfang (wahlweise als 32-Bit- oder 64-Bit-Version) ist über ein Klick-und-Los-Installationsprogramm verfügbar, das die Benutzer unter [Skype for Business für alle Geräte herunterladen](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3) herunterladen können.

### <a name="msi"></a>MSI

MSI-Datei ist eine traditionellere Installationsmethode, der Skype für Business 2015 und 2016 Clients verwendet. Sie können Sie manuell installieren von Updates und Patches, Verwendung Volumenlizenzierung und Aktivierung und kann über das [Office-Anpassungstool](https://www.microsoft.com/en-us/download/details.aspx?id=49030)angepasst werden. Sie können Clients verteilen, indem Sie Gruppenrichtlinien anwenden, System Center Configuration Manager verwenden oder ein Tool eines Drittanbieters nutzen.
