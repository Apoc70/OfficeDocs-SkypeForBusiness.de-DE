# <a name="plan-for-clients-and-devices"></a>Planen für Clients und Geräte

**Zusammenfassung:** Überprüfen Sie die unterstützten Clients und Apps für Skype for Business.

Heutzutage sind Arbeitskräfte ständig unterwegs. Mitarbeiter müssen kommunizieren und zusammenarbeiten können – unabhängig davon, ob sie am Unternehmensstandort, in regionalen Niederlassungen, von zu Hause aus oder unterwegs arbeiten. Skype for Business Server ist dank einer Reihe von Clientbenutzeroberflächen, die Sie den Benutzern Ihres Unternehmens bereitstellen können, ganz auf diese Anforderungen zugeschnitten. Eine sorgfältige Planung gewährleistet, dass die Mitarbeiter alle benötigten Anwendungen und Funktionen erhalten und dass ihnen Skype for Business überall zur Verfügung steht.

## <a name="available-clients"></a>Verfügbare Clients

Skype for Business Server unterstützt mehrere Clienttypen, u. a. auf Computern installierte Clientsoftware, webbasierte Clients und Clients für mobile Geräte. Die wichtigsten Clients werden in diesem Abschnitt vorgestellt. Eine detaillierte Auflistung aller unterstützten Clients finden Sie unter [Vergleich der Funktionen der Skype for Business Server 2015-Desktopclients](desktop-feature-comparison.md) oder [Vergleich der Funktionen der Skype for Business Server 2019-Desktopclients](../../../SfBServer2019/plan/feature-comparison.md). Wenn Sie vorher eine Kombination aus Lync-Clients verwendet haben, beachten Sie, dass es nicht unterstützte [Legacyclients](clients-and-devices.md#Legacy) gibt, die nicht mit Skype for Business Server 2019 kompatibel sind. Es gibt regelmäßige Updates. Überprüfen Sie daher dieses Thema von Zeit zu Zeit auf die neuesten Clientinformationen.

### <a name="skype-for-business-2019"></a>Skype for Business (2019)

Skype für Unternehmen (2019) ist der empfohlene mit vollem Funktionsumfang Client für Skype für Business Server 2015 und 2019. Eine Beschreibung der neuen Funktionen finden Sie unter [Führen Sie die neuesten Updates in Skype für Unternehmen](https://support.office.com/en-us/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) . Die von den Clients unterstützten Funktionen werden im Detail unter [Vergleich der Funktionen der Skype for Business-Desktopclients](desktop-feature-comparison.md) beschrieben, und die Benutzerdokumentation finden Sie in der [Hilfe zu Skype for Business](https://support.office.com/en-US/Skype-for-business). Dieser Client ist enthalten, wenn Benutzer Office 365 installieren.

Außerdem ist ein kostenloser Standardclient verfügbar, der weniger Funktionen unterstützt. Beide Versionen können unter [Skype for Business für alle Geräte herunterladen](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3) heruntergeladen werden. Die Unterschiede werden im Abschnitt zu den [Unterschieden zwischen dem Client mit vollem Funktionsumfang und dem Standardclient](desktop-feature-comparison.md#Full-Basic) beschrieben.

### <a name="skype-for-business-2016"></a>Skype for Business 2016

Eine Beschreibung der neuen Funktionen finden Sie unter [Neuigkeiten in Skype for Business 2016](https://support.office.com/en-us/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781). Dieser Client ist enthalten, wenn Benutzer Office 365 installieren.

### <a name="skype-for-business-on-mac"></a>Skype for Business auf dem Mac

Der [Skype für Unternehmen auf Mac](https://www.microsoft.com/en-us/download/details.aspx?id=54108) -Client ist zum Download zur Verfügung. Finden Sie unter [Skype für Unternehmen auf Mac-Clientanforderungen](mac-requirements.md) erforderlichen Komponenten zu überprüfen.

### <a name="skype-for-business-for-mobile-devices"></a>Skype für Unternehmen für mobile Geräte

Clients stehen für Windows Phone, iPhone/iPad und Android. Benutzer können sie unter [Herunterladen Skype für Unternehmen für alle Ihre Geräte](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3)abgerufen werden. Unterstützung für diese Clients wird in [mobilen Client Featurevergleich für Skype für Unternehmen](mobile-feature-comparison.md)beschrieben.

> [!NOTE]
> Unterstützung für mobile Clients von Vorversionen MCX (Mobility Service) ist nicht mehr in Skype für Business Server 2019 verfügbar. Alle aktuellen Skype für mobile Clients Business Unified Communications Web API (UCWA) zur Unterstützung von Sofortnachrichten (IM), Anwesenheit und Kontakte bereits verwenden. Benutzer mit Clients von Vorversionen von MCX müssen an einen aktuellen Client aktualisieren.

### <a name="online-meeting-add-in-for-skype-for-business"></a>Onlinebesprechungs-Add-in für Skype für Unternehmen

Das Onlinebesprechungs-Add-in für Skype für Unternehmen unterstützt die besprechungsverwaltung aus Microsoft Outlook-Client für messaging und Zusammenarbeit auf Windows. Das Onlinebesprechungs-Add-in für Skype für Unternehmenssoftware wird automatisch mit Skype für Unternehmen installiert.

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Skype für Business Web App und Skype Besprechungen App

Wenn Skype für Unternehmen auf dem Computer eines Benutzers nicht installiert ist, und der Benutzer klickt auf einen Link zur Besprechung in einer Besprechungsanfrage auf einem Windows-Computer, der Skype Besprechungen App oder Skype für Web-Geschäfts-App installiert werden und zu öffnen.  Skype-Besprechungen App ist der Client Wahl für Teilnehmer außerhalb Ihrer Organisation. (Beachten Sie, dass auf einem Mac Skype für Unternehmen auf Mac installiert werden, und öffnen). Finden Sie unter [Planen von Besprechungen-Clients (Web App und Besprechungen App)](meetings-clients.md) für die Anforderungen, um diese Clients verwenden.


### <a name="skype-for-business-web-scheduler"></a>Skype für Business-Webplaner

[Skype für Business Webplaner](https://sched.lync.com) ist ein webbasierter meeting Planung und Verwaltung Tool für Benutzer von Skype für Business Online, die keinen Zugriff auf Microsoft Outlook, oder auf einem Betriebssystem nicht auf Windows basierenden stehen. Benutzer können mit Skype für Business Webplaner neue Besprechungen erstellen, ändern vorhandene Besprechungen und Senden von Einladungen mithilfe ihrer bevorzugten e-Mail-Programm. Skype für Business Webplaner- [Dokumentation](https://support.office.com/en-us/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US) enthält weitere Details.

### <a name="vdi-plugins"></a>VDI-Plug-Ins

Eine VDI-Umgebung (Virtual Desktop Infrastructure) wird in einigen Organisationen verwendet, in denen Sicherheit und Complianceprobleme besonders sensibel sind. Verwendung von Skype für Unternehmen mit vollständigen Audio- und Videodaten bei einer Verbindung kann es muss mit dem starker Auslastung von Audio und Video Verarbeitung auf dem Client auf einem virtuellen Desktop verwaltet. Zusätzlicher VDI-Plug-in-Software verfügbar ist, die die Verarbeitung auf dem lokalen Computer des Endbenutzers verlagert und reduziert die Last auf dem virtuellen Desktop. Details zu diesen Plug-Ins finden Sie unter [Plan for Skype for Business in VDI environments](vdi-environments.md).

### <a name="microsoft-teams-rooms"></a>Microsoft Teams-Räume

Microsoft-Teams Chatrooms ist Microsofts neueste Lösung für Webkonferenzen verwendet eine vertraute Oberfläche und wird auf einfache Weise bereitgestellt und verwaltet, nutzen vorhandene Geräte wie LCD-Bereiche für einfache Installation. Microsoft-Teams Chatrooms verwendet eine spezielle UWP app auf einem Surface Pro 4 oder Surface Pro in einem Konsolenmodus ausgeführt (nach der Bereitstellung der app UWP ist die einzige app, die auf dem Gerät ausgeführt wird) und ein eigenen Konto Gerät auf Ihre Implementierung erfordert. Software wird über den Windows Store und Windows Update aktualisiert. Finden Sie unter https://aka.ms/MTRDocs detaillierte Informationen zur Verwendung dieser Raum Konsolen in Ihrer Bereitstellung. 

### <a name="skype-for-business-on-surface-hub"></a>Skype für Unternehmen auf Fläche Hub

Microsoft Surface Hub ist ein all-in-One Produktivität Gerät, das für die direkte Verwendung ist für Brainstorming, Zusammenarbeit und Präsentationen. Es wurde eine eigene Iteration der Skype für Business-Client im [Administratorhandbuch zu Microsoft Surface Hub](https://docs.microsoft.com/surface-hub/)dokumentiert.

## <a name="choosing-your-organizations-preferred-client"></a>Auswählen des bevorzugten Clients für Ihre Organisation
<a name="BK_client_choose"> </a>

Wenn Ihre Organisation die entsprechenden Lizenzen erworben hat, wählen Sie den vollständigen Client, anderenfalls wählen Sie den Standardclient. 

Ihre Benutzer können von [Skype für Unternehmen für alle Ihre Geräte herunterladen](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3)des Clients für sich selbst installieren. Der Client ist auch installiert, wenn Benutzer Office 365 auf Windows installieren. Wenn einige Ihrer Benutzer Macs haben, müssen diese Benutzer einen anderen Satz von Features, wie in den vorherigen Abschnitten beschrieben.

Einige Features sind mit Skype für Business Server 2015 stehen nicht in Skype Business Online oder Skype für Business Server 2019, finden Sie unter [Online oder Hybrid Benutzerkonto Einschränkungen für 2015](desktop-feature-comparison.md#Online-Hybrid) oder [Online oder Hybrid Benutzer Konto Einschränkungen für 2019 ](desktop-feature-comparison.md#Online-Hybrid)Einzelheiten. Skype für Business Online-Administratoren möglicherweise [Skype für Business Online Service Description](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) Informationen zu den verschiedenen Plänen darauf verweisen möchten.

 Überprüfen Sie vor dem Bereitstellen oder auf Skype für Unternehmen aktualisieren, welche Clients bereits in Ihrer Organisation verwendet werden. Verwenden Sie den [Desktopclient Featurevergleich für Skype für Unternehmen](desktop-feature-comparison.md) , die Feature Support Auswirkungen auf die diese Clients zu verstehen. Dann können Sie den Benutzern die Änderungen vermitteln sowie das Tempo des Rollouts bestimmen und die Vorteile des Upgrades auf den neuesten Client in vollem Umfang verstehen.

## <a name="ways-to-deploy-the-client-to-your-users"></a>Methoden für die Bereitstellung des Clients für Ihre Benutzer
<a name="BK_User_Deploy"> </a>

Clientinstallationsprogramme sind sowohl als MSI-Datei als auch als Klick-und-Los-Version verfügbar. Die Skype für Client-Sustainment Unternehmensstrategie beeinträchtigen fertigzustellen, damit Sie die folgenden Punkte vertraut sein sollten:

- Im Allgemeinen Skype für Unternehmen nicht neue Funktionen für Clients aus älteren hinzu

- Skype für Business plan im Allgemeinen nicht auf neue Features für Business MSI-Datei nach der ersten Version in der Skype ausgeliefert. MSI-Verbesserungen zwischen Versionen werden in erster Linie Qualität/Sicherheit Natur.

- Die neuesten und interessantesten Skype Business Client wünschen wird in der Skype für Business 2019 Klick-und-Los-Installer gefunden werden.

Sie können eine benutzerdefinierte Bereitstellung des Clients wie unter [Anpassen von Windows-Clientinstallation Skype für Business Server](../../deploy/deploy-clients/customize-windows-client-installation.md)durchführen. Installationsmethoden werden in [Bereitstellen von Clients für Skype für Business Server](../../deploy/deploy-clients/deploy-clients.md) ausführlicher beschrieben.

### <a name="click-to-run"></a>Klick-und-Los

Klick-und-los ist ein Microsoft streaming und Virtualisierungstechnologie, die Sie zum Installieren und Aktualisieren von Office-Produkten, einschließlich Skype für Unternehmen verwenden können. Diese Streaming- und Virtualisierungsfunktionen basieren auf Technologien in Microsoft Application Virtualization (App-V). Klick-und-Los bietet die folgenden Vorteile:

- Gestreamte Installation der Office-Suite, die den Zeitbedarf für die Installation verringert

- Slipstreamupdates und -patches

- Geringerer Platzbedarf auf dem Datenträger

- Benutzerbasierte Lizenzierung: fünf Installationen pro Benutzer

- Anpassbar über XML-Editor für die Installation von unabhängigen Programmen

Sie können das [Office-Bereitstellungstools](https://www.microsoft.com/en-us/download/details.aspx?id=49117) für diese Art von Installer verwenden möchten.

Sowohl die grundlegende und vollständige Clientversionen (mit Auswahl der 32- und 64-Bit-Versionen) sind mit einem Klick-und-Los-Installer verfügbar, die Benutzer können unter [Herunterladen Skype für Unternehmen für alle Ihre Geräte](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3)herunterladen.

### <a name="msi"></a>MSI

MSI-Datei ist eine traditionellere Installationsmethode, der Skype für Business 2015 und 2016 Clients verwendet. Sie können Sie manuell installieren von Updates und Patches, Verwendung Volumenlizenzierung und Aktivierung und kann über das [Office-Anpassungstool](https://www.microsoft.com/en-us/download/details.aspx?id=49030)angepasst werden. Sie können Clients verteilen, indem Sie Gruppenrichtlinien anwenden, System Center Configuration Manager verwenden oder ein Tool eines Drittanbieters nutzen.
