# <a name="define-your-requirements-for-emergency-calls-in-skype-for-business-server"></a>Definieren Sie Ihre Anforderungen für Notrufe in Skype for Business Server.
 
Fasst die erforderlichen Schritte zum Aktivieren von E9-1-1 in Skype for Business Server Enterprise-VoIP zusammen, je nachdem, ob Sie über einen SIP Trunk E9-1-1-Dienstanbieter oder ein Elin-Gateway verfügen.
  
Bevor Sie mit der Bereitstellung von Skype for Business Server E9-1-1 beginnen, sollten Sie zunächst in der Lage sein, die in den folgenden Abschnitten aufgeführten Fragen zu beantworten. Welche Planungsmaßnahmen erforderlich sind, ist abhängig von der E9-1-1-Lösung, die Sie bereitstellen möchten: einen SIP-Trunk-E9-1-1-Dienstanbieter oder ein ELIN-Gateway (Emergency Location Identification Number). In der folgenden Tabelle sind die Abschnitte in dieser Planungsarbeitsmappe aufgeführt, die Sie für jede dieser Lösungen überprüfen müssen.
