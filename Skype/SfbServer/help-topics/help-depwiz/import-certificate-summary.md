# <a name="import-certificate-summary"></a>Importieren von Zertifikaten (Zusammenfassung)
 
Auf der Seite **Zertifikatzusammenfassung importieren** wird eine Zusammenfassung des Datei namens, des Pfads und des Vorhandenseins eines privaten Schlüssels angezeigt. Nachdem Sie die Zusammenfassung überprüft haben, klicken Sie auf **weiter** , um fortzufahren.
