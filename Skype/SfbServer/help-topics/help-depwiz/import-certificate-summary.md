# <a name="import-certificate-summary"></a>Importieren von Zertifikaten (Zusammenfassung)
 
Auf der Seite **Zertifikat importieren – Zusammenfassung** wird eine Zusammenfassung der Dateiname, Pfad und Anwesenheit eines privaten Schlüssels angezeigt. Nachdem Sie die Zusammenfassung überprüft haben, klicken Sie auf **Weiter** .
