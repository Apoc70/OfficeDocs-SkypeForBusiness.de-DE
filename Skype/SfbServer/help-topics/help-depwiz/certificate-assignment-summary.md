# <a name="certificate-assignment-summary"></a>Zertifikatzuweisung (Zusammenfassung)
 
Bevor der zertifikatzuweisung abgeschlossen ist, wird mit einer **Zertifikatszuweisung – Zusammenfassung** angezeigt, in der die Details des Zertifikats aufgelistet, die zugewiesen werden. Lesen Sie die Informationen, um sicherzustellen, dass den Antragstellernamen und den alternativen Antragstellernamen richtig sind. Wenn Sie die Zusammenfassung gelesen haben, klicken Sie auf **Weiter**.
