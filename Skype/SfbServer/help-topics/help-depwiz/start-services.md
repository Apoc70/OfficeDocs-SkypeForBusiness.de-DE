# <a name="start-services"></a>Starten von Diensten
 
Um die Dienste auf dem bereitgestellten Server-Role zu starten, klicken Sie auf **Ausführen** , in **Schritt 4: Dienste starten**. Sie müssen als Mitglied der Gruppe der lokalen Administratoren oder Entsprechung angemeldet sein. 
  
> [!NOTE]
> Wenn **Dienste starten** bereits ausgeführt wurde, können Sie den Befehl erneut ausführen, indem Sie auf **Erneut ausführen**.
