# <a name="certificate-request-certificate-signing-request-generated"></a>Zertifikatanforderung (Signieranforderung generierte Zertifikat)
 
Nach Abschluss eine offlineanforderung für ein Zertifikat, wird die Seite **Zertifikatanforderungsdatei** angezeigt. Zum Anzeigen der zertifikatanforderung und Vorbereiten an Ihre offline Zertifizierungsstelle (CA) zu senden oder öffentlichen Zertifizierungsstelle, klicken auf die Schaltfläche **Ansicht** . Die Anforderung wird (standardmäßig) im Windows-Editor angezeigt. Sie können kopieren und fügen Sie die codierte zertifikatanforderung aus dieser Anzeige oder beenden Sie die Verarbeitung der zertifikatanforderung durch Befolgen der Anweisungen aus Ihrer Wahl, offline oder öffentliche Zertifizierungsstelle. Klicken Sie auf **Fertig stellen** , um den Vorgang fortzusetzen.
