# <a name="persistent-chat-planning-tool"></a>Persistent Chat (Planning Tool)
 
Der Server für beständigen Chat ermöglicht Benutzern die Teilnahme an mehrteiligen, themenbasierten Konversationen, die im Laufe der Zeit fortbestehen.
  
Weitere Informationen zum Vorbereiten Ihrer Umgebung für beständigen Chat Server finden Sie unter [Planen des beständigen Chat Servers in Skype for Business Server 2015](../../plan-your-deployment/persistent-chat-server/persistent-chat-server.md).
