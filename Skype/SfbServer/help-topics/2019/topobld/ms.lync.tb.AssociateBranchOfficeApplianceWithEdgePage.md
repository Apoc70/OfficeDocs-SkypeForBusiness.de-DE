# <a name="associate-branch-office-appliance-with-edge"></a>Zuordnen der Filialanwendung zur Edgekomponente
 
Wenn Sie Edgeserver bereitgestellt haben und Mediendienste für die externe Verwendung bereitstellen müssen, wählen Sie den Edgeserver oder Pool aus, der der Survivable Branch Appliance oder dem Überlebenden Verzweigungs Server zugeordnet wird, indem Sie ihn in der Dropdownliste auswählen. Klicken Sie auf **Weiter**, um fortzufahren.
  
> [!NOTE]
> Wenn Sie keinen Edgeserver oder Edgeserver-Pool in Ihrer Topologie definiert haben, können Sie diesen Schritt überspringen. Wenn Sie einen neuen Edgeserver oder Edgeserver-Pool definieren, oder wenn Sie die Eigenschaftenseiten der Survivable Branch-Appliance oder Survivable-Einheit bearbeiten, können Sie die Survivable Branch-Appliance oder den Überlebenden Branch-Server einem Edge-Server oder Edge-Server-Pool zuordnen. Verzweigungs Server nach dem Definieren eines Edgeserver-oder Edgeserver-Pools in der Topologie.
