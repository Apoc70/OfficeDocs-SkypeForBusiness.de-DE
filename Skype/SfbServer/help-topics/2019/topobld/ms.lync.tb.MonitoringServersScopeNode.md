# <a name="monitoring-servers-scope-node"></a>Monitoring Server – Bereichsknoten
 
Der Knoten **Überwachungsserver** listet die Überwachungsserver in Ihrer Topologie auf. Folgende Möglichkeiten stehen zur Verfügung:
  
- **Neuer Überwachungs Server**. Startet den Assistenten zum **Definieren eines neuen Überwachungsservers** , mit dem Sie der Topologie einen neuen Überwachungsserver hinzufügen.
    
Zusätzlich zu den Monitoring Server-Optionen können Sie die folgenden allgemeinen Aufgaben ausführen:
  
- **Topologie**. Sie können diese Option verwenden, um eine neue Topologie zu definieren, eine vorhandene Topologie zu öffnen, die aktuelle Topologie herunterzuladen, eine Kopie der Topologie zu speichern, die Topologie zu veröffentlichen, eine Datenbank zu installieren, Komponenten aus früheren Versionen von Office Communications Server zu verbinden oder die Bereitstellungs.
    
- **Ansicht**: Mithilfe dieser Option können Sie die Ansicht des Topologie-Generators anpassen, indem Sie die Elemente auswählen, die in den Ansichten der Konsolenstruktur ein- bzw. ausgeblendet werden sollen.
    
- **Hilfe**: Über diese Option greifen Sie auf die Hilfe für den aktiven Knoten zu.
