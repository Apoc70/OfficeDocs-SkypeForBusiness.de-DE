# <a name="add-file-store"></a>Hinzufügen des Dateispeichers
 
Um einen neuen Dateispeicher zu definieren, geben Sie Folgendes ein:
  
- Der vollqualifizierte Domänenname (FQDN) des Servers, der den neuen Dateispeicher hosten wird. Den FQDN des Servers muss in das Domain Name System (DNS) vorhanden sein, bevor Sie die Topologie veröffentlichen.
    
- Der Name der Dateifreigabe auf dem Hostserver. Die Dateifreigabe muss vorhanden sein, bevor Sie die Topologie veröffentlichen.
