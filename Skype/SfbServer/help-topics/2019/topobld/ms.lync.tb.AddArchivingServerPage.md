# <a name="add-archiving-server"></a>Hinzufügen des Archivierungsservers
 
Zum Bereitstellen eines Archivierungsservers innerhalb des Topologieentwurfs müssen Sie den vollqualifizierten Domänennamen des Servers angeben, auf dem Sie die Archivierung bereitstellen möchten. Vor dem Veröffentlichen einer Topologie, die Archivierung enthält, und Installieren von Skype für Business Server, Sie ausgefüllt haben sollten alle erforderlichen Komponenten für die Bereitstellung von Archivierung.
