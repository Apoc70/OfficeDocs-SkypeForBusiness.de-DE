# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Hinzufügen von Front-End-Sql-Speicher-Seite für Lync Server 2010
 
Sie definieren Sie die Eigenschaften der SQL Server-Speicher für den neuen Front-End-Pool, den Sie konfigurieren.
  
> [!IMPORTANT]
> Wenn Sie einen Standard Edition-Server konfigurieren, wird dieses Dialogfeld angezeigt werden, aber keine Einstellungen konfiguriert werden. Klicken Sie auf **Weiter**, um fortzufahren.
  
Die Einstellungen, die Sie konfigurieren können:
  
- **Verwenden Sie zuvor definierten SQL Server-Speicher**: Wählen Sie einen vorhandenen SQL Server-Speicher aus der Liste für den Front-End-Pool-Speicher.
    
- Wählen Sie zum Erstellen eines neuen SQL Server-Speichers **definieren eine neue SQL Server speichern** .
    
    Geben Sie den vollqualifizierten Domänennamen des SQL-Servers in **SQL Server-FQDN**
    
    Die **Standardinstanz** wird zunächst ausgewählt. Wenn Sie eine Instanz zu definieren lieber, wählen Sie **benannte Instanz** aus, und geben Sie den Namen der Instanz, die Sie erstellen müssen.
    
 **Zurück**: wechselt zurück zum vorherigen Dialogfeld.
  
 **Weiter**: übernimmt die Änderungen und wechselt zum nächsten Schritt Dialogfeld
  
 **Abbrechen**: verwirft Änderungen und die Konfiguration wird abgebrochen
  
 **Hilfe**: Zeigt Hilfeinformationen wie dieses Thema
