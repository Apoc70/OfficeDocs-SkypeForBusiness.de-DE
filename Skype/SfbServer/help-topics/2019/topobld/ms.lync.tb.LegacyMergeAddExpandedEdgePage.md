# <a name="legacy-merge-add-expanded-edge"></a>Legacyzusammenführung – Hinzufügen einer erweiterten Edgekomponente
 
Skype für Business Server unterstützt das Zusammenführen von einer Vorversion Office Communications Server 2007 R2 nicht erweiterten Edge-Server-Konfiguration. Der erweiterte Edgeserver muss zunächst in einen konsolidierten Edgeserver umgewandelt werden, ehe die Zusammenführung der Edgeserverinformationen erfolgen kann.
