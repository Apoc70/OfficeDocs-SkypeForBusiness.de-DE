# <a name="add-additional-sip-domains"></a>Hinzufügen zusätzlicher SIP-Domänen
 
Geben Sie den Domänennamen jeder zusätzlichen Domäne an, die Sie in Ihrer Bereitstellung unterstützen möchten. Wenn Sie beispielsweise contoso.net als primäre SIP-Domäne (Session Initiation Protocol) angegeben haben und auch fabrikam.net für den SIP-Datenverkehr in Ihrer Bereitstellung unterstützen möchten, geben Sie fabrikam.net als zusätzliche unterstützte Domäne an.
