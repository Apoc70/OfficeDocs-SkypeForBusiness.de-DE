# <a name="legacy-merge-edge-type"></a>Legacyzusammenführung – Edgetyp
 
Klicken Sie für einen **einzelnen Edgeserver** oder **konsolidierten Edgeserver mit Lastenausgleich** auf **weiter** , um Konfigurationsinformationen anzugeben. Bei einem **erweiterten Edge**muss die Konfiguration jedes Servers in eine konsolidierte Edge-Konfiguration geändert werden, bevor Sie zusammengeführt werden kann.
