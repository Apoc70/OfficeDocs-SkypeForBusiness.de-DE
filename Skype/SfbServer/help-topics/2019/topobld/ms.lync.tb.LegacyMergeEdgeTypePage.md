# <a name="legacy-merge-edge-type"></a>Legacyzusammenführung – Edge-Typ
 
Klicken Sie für einen **Einzelnen Edgeserver** oder **Lastenausgleich konsolidierte Edge** -Server auf **Weiter** , um Konfigurationsinformationen angeben. Für eine **Erweiterte Edge**muss Konfiguration des Servers in einer konsolidierten Konfiguration für Zugriffsedge geändert werden, bevor sie mit Lync Server 2013 zusammengeführt werden kann.
