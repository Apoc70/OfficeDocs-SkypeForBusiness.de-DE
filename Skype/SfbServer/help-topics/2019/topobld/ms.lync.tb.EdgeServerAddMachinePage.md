# <a name="edge-server-add-machine"></a>Hinzufügen eines Edgeservercomputers

Geben Sie den vollqualifizierten Domänennamen (Fully Qualified Domain Name, FQDN) jedes Computers an, den Sie als Edgeserver in diesem Pool hinzufügen möchten. Nachdem Sie der Liste einen Computer hinzugefügt haben, können Sie den FQDN des Computers aktualisieren oder ihn aus dem Pool entfernen, bevor Sie die Topologie veröffentlichen. Nachdem Sie die Topologie veröffentlicht haben, müssen Sie den Server im Topologie-Generator durch Ändern des FQDN löschen und dann einen neuen Server zum Pool mit dem neuen FQDN hinzufügen. Details zum Entwerfen der Topologie für einen Edge-Pool finden Sie unter [Definieren Ihrer Edge-Topologie](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) in der Edge-Bereitstellungsdokumentation.
