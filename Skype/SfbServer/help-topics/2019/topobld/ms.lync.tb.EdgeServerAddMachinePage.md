Nachdem Sie der Liste einen Computer hinzugefügt haben, können Sie den FQDN des Computers aktualisieren oder ihn aus dem Pool entfernen, bevor Sie die Topologie veröffentlichen. Nachdem Sie die Topologie veröffentlicht haben, müssen Sie den Server im Topologie-Generator durch Ändern des FQDN löschen und dann einen neuen Server zum Pool mit dem neuen FQDN hinzufügen. Details zum Entwerfen der Topologie für einen Edge-Pool finden Sie unter Definieren Ihrer Edge-Topologie in der Edge-Bereitstellungsdokumentation.
