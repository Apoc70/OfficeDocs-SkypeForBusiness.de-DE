# <a name="add-mediation-server-nexthop"></a>Hinzufügen von Vermittlungsservern – NextHop
 
Wenn Sie den nächsten Hop für einen von Ihnen definierten Vermittlungsserver definieren möchten, wählen Sie in der Dropdownliste **Nächster Hop-Pool** den Enterprise Edition-Server oder den Standard Edition-Server aus, der als nächster Hop für den Vermittlungsserver fungieren soll. Klicken Sie auf **Weiter**, um fortzufahren.
