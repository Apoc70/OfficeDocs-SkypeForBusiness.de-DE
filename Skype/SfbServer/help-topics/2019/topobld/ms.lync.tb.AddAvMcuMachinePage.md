# <a name="add-av-mcu-machine"></a>Hinzufügen des A / V-MCU-Computers
 
Geben Sie den vollqualifizierten Domänennamen (FQDN) jeder Computer, den Sie als ein A hinzufügen möchten / V-Konferenzserver in diesen A / V-konferenzpool. Nach dem Hinzufügen eines Computers zur Liste können Sie den vollqualifizierten Domänennamen des Computers zu einem beliebigen Zeitpunkt vor der Veröffentlichung der Topologie aktualisieren oder aus dem Pool entfernen. Nach der Veröffentlichung der Topologie erfordert das Ändern des vollqualifizierten Domänennamens Löschen des Servers im Topologie-Generator, und klicken Sie dann einen neuen Server mit den neuen FQDN dem Pool hinzufügen.
