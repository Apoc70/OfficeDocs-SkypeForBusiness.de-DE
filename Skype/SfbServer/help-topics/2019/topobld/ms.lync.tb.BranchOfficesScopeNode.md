# <a name="branch-offices-scope-node"></a>Zweigstellen – Bereichsknoten
 
Im Knoten **Zweigstellen** werden die Zweigstellen aufgelistet, die diesem zentralen Standort zugeordnet sind. Folgende Möglichkeiten stehen zur Verfügung:
  
- **Neue Zweigstelle**. Startet den Assistenten zum Definieren einer neuen Zweigstelle****, in dem Sie eine neue Zweigstelle zur Topologie hinzufügen.
    
Zusätzlich zu den Optionen für Zweigstellen sind die folgenden allgemeinen Optionen verfügbar:
  
- **Topologie**: Über diese Option können Sie eine neue Topologie definieren, eine vorhandene Topologie öffnen, die aktuelle Topologie herunterladen, eine Kopie der Topologie speichern, die Topologie veröffentlichen, eine Datenbank installieren, Komponenten vorheriger Versionen zusammenführen oder die Bereitstellung entfernen.
    
- **Ansicht**: Mit dieser Option können Sie die Ansicht des Topologie-Generators anpassen, indem Sie die Elemente auswählen, die in den Ansichten der Konsolenstruktur ein- bzw. ausgeblendet werden sollen.
    
- **Hilfe**: Mit dieser Option greifen Sie auf die Hilfe für den aktiven Knoten zu.
