# <a name="add-default-sip-domain"></a>Hinzufügen der Standard-SIP-Domäne
 
Geben Sie die primäre SIP-Domäne (Session Initiation Protocol) an, die von der Bereitstellung unterstützt werden soll. Die primäre SIP-Domäne kann mit der Domäne der Active Directory-Domänendienste identisch oder anders sein. Die Active Directory-Domäne könnte beispielsweise fabrikam.net sein, und Sie können Ihre primäre SIP-Domäne als contoso.net angeben.
