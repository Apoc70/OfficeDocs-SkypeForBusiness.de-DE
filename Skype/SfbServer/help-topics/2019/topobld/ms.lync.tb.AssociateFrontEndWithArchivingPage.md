# <a name="associate-front-end-with-archiving"></a>Zuordnen des Front-Ends zur Archivierung
 
Jedem Front-End-Pool und einer Survivable Branch-Appliance kann nur ein Archivierungs Server zugeordnet sein. Wenn die Archivierung für die Website aktiviert ist, sammelt und erfasst der Archivierungs Server Sofortnachrichten Inhalte und Webkonferenzinhalte aus Unterhaltungen, an denen sich die Benutzer befinden, die sich in den zugehörigen Pools und überlebensfähigen Branch-Appliances befinden. 
  
Alle Pools auf einer Website und die Pools mehrerer zentraler Standorte können denselben Archivierungsserver verwenden, wenn die Auslastung des Archivierungsservers nicht überschritten wird.
