# <a name="file-store-settings-expander"></a>Dateispeicher-Einstellungen – Erweiterung
 
So bearbeiten Sie die Eigenschaften der definierten Dateispeicher
  
Für den Namen der Datei freizugeben, können Sie den Namen des neuen Freigabe definieren. Sie müssen dann die Freigabe auf dem Server erstellen, mit dem Namen in der Eigenschaft Dateiserver voll qualifizierte Domäne Name (FQDN).
  
> [!NOTE]
> Sie können den FQDN des Servers, die in der Dateiserver-FQDN-Eigenschaft angegeben wird, nicht ändern. Wenn die Datei Store Dateifreigabe auf einem anderen Server platziert werden müssen, erstellen Sie einen neuen Eintrag der Datei speichern.
