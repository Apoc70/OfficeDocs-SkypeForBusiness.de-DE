# <a name="define-the-root-trunk-for-a-new-ip-or-pstn-gateway"></a>Definieren des Stammtrunks für eine neue IP- oder PSTN-Gateway
 
Sie definieren den stammtrunk für die IP-Adresse oder das Telefonfestnetz (PSTN), indem Sie Folgendes konfigurieren:
  
- **Trunkname**: Definieren des vollqualifizierten Domänennamens dem Trunk zugeordnet
    
- **Überwachungsport für IP/PSTN-Gateway**: definiert den Port, der diesem Trunk überwacht wird
    
- **SIP-Transportprotokoll**: Wählen Sie aus der Liste entweder **TCP** oder **TLS**, basierend auf trunkanforderungen
    
- **Zugeordneter Vermittlungsserver**: Wählen Sie aus der Liste der verfügbaren Mediation Server in Ihrer Bereitstellung
    
- **Zugeordneter Vermittlungsserver Port**: definiert den Port, die den ausgewählten Mediation Server abgehört wird
