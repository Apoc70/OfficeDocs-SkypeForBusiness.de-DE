# <a name="select-from-active-directory"></a>Auswählen aus Active Directory
 
Verwenden Sie das Dialogfeld **aus Active Directory auswählen** , um die Benutzer zu finden, die Sie für Skype for Business Server aktivieren möchten. Geben Sie im Suchfeld die SIP-Adresse oder den Anzeigenamen des Benutzers ein, nach dem gesucht werden soll. Wenn Sie alle verfügbaren Benutzer abrufen möchten, geben Sie im Suchfeld keine Suchzeichenfolge ein.
