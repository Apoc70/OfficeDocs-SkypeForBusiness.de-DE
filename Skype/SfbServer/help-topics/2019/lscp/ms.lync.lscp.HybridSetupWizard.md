# <a name="hybrid-manageability-wizard"></a>Hybridverwaltungs-Assistent

Der Assistent Hybrid Verwaltbarkeit können Sie einige der die zum Einrichten einer hybridbereitstellung mit Skype für Business Online erforderlichen Konfigurationsaufgaben ausgeführt werden. Weitere Informationen zum Konfigurieren der Bereitstellung für hybride finden Sie unter [Konfigurieren der Umgebung mit dem Assistenten für die hybride Verwaltbarkeit](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx).
