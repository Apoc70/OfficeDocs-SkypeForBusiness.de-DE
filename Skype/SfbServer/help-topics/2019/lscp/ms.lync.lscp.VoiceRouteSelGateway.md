# <a name="select-trunk"></a>Auswählen eines Trunks
 
Verwenden Sie das Dialogfeld **trunk auswählen** , um einen vorhandenen trunk zu finden, den Sie verwenden können.
  
> [!NOTE]
> Wenn Sie alle verfügbaren Trunks abrufen möchten, geben Sie im Suchfeld nichts ein, und klicken Sie dann auf das Symbol suchen.
