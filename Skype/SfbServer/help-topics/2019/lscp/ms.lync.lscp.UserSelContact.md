# <a name="select-contacts-from-active-directory"></a>Auswählen von Kontakten aus Active Directory
 
Verwenden Sie das Dialogfeld **aus Active Directory auswählen** , um Kontakte suchen, die Sie für Skype für Unternehmen aktivieren möchten. Geben Sie im Suchfeld die SIP-Adresse oder den Anzeigenamen des Kontakts ein, nach dem gesucht werden soll. Wenn Sie alle verfügbaren Kontakte abrufen möchten, geben Sie im Suchfeld keine Suchzeichenfolge ein.
