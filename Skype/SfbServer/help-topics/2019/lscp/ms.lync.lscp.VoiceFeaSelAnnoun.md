# <a name="select-an-announcement-service"></a>Auswählen eines Ankündigungsdiensts
 
Im Dialogfeld **Ankündigungsdienst auswählen** können Sie einen Ankündigungsdienst auswählen, der eingehende Anrufe an einen Bereich nicht zugewiesener Nummern abwickeln soll.
  
Geben Sie im Suchfeld den Namen des ankündigungsdiensts, den Sie verwenden möchten, ganz oder teilweise ein. Wenn Sie alle verfügbaren Ankündigungs Dienste zurückgeben möchten, lassen Sie das Suchfeld leer, und klicken Sie auf das Symbol suchen.
  
Wählen Sie in der Liste der von der Suche zurückgegebenen Ankündigungs Dienste den Dienst aus, den Sie für die Behandlung eingehender Anrufe in den Bereich der nicht zugewiesenen Nummern verwenden möchten.
