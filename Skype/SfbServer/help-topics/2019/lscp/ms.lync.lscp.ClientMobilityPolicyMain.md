# <a name="clients-mobility-policy"></a>Clients: Mobilitätsrichtlinie

Die Seite **Clients** **Mobilitätsrichtlinie** werden in Ihrer Bereitstellung verfügbaren mobilitätsrichtlinien angezeigt.

## <a name="tasks-you-can-perform"></a>Mögliche Aufgaben

Klicken Sie auf der Seite **Mobilitätsrichtlinie** können Sie die folgenden Aufgaben ausführen:

- Zeigen vorhandener mobilitätsrichtlinien an

- Erstellen einer neuen mobilitätsrichtlinie

- Bearbeiten einer mobilitätsrichtlinie

## <a name="ui-reference"></a>Referenz zur Benutzeroberfläche

In den folgenden Listen werden die Menüs, Befehle, Felder und Eigenschaften der Seite beschrieben.



- **Suchfeld** Geben Sie eine Zeichenfolge in das Textfeld, um die angezeigten Richtlinien zu filtern.

- **Neue** Klicken Sie auf **neu**, und wählen Sie dann **entweder **Standort-** oder eine neue mobilitätsrichtlinie zu erstellen** .

- **Bearbeiten** Wählen Sie eine Richtlinie aus, und klicken Sie dann auf **Bearbeiten** , um die Eigenschaften der ausgewählten Richtlinie zu bearbeiten.

- **Aktualisieren** Aktualisiert die Liste der Richtlinien angezeigt. Wenn Sie auf **Aktualisieren**klicken, wird die Filter-Suchzeichenfolge deaktiviert ist, und alle vorhandene Richtlinien werden angezeigt.

Ausführliche Informationen zum Konfigurieren von mobilitätsrichtlinien finden Sie unter [Configuring Mobility Policy](https://technet.microsoft.com/library/595536e0-9bb3-49a3-8d13-1a77351ebc62.aspx).
