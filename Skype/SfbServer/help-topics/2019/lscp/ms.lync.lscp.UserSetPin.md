# <a name="set-pin"></a>Festlegen der PIN
 
Um die PIN des Benutzers generiert werden soll, wählen **Sie gültige PIN automatisch generieren**. 
  
Wählen Sie **Bestimmte PIN von Hand eingeben** und geben Sie anschließend eine PIN in das Textfeld ein, die die in Ihren PIN-Richtlinieneinstellungen angegebenen PIN-Anforderungen erfüllt. 
  
Ausführliche Informationen zu den verschiedenen Verfahren, die Sie mithilfe der Skype für Business Server-Systemsteuerung ausführen können, finden Sie unter [Verwalten von Skype für Business Server 2015](../../../manage/manage.md).
