# <a name="prefix-edit"></a>Bearbeiten von Präfixen
 
Rufnummernpräfix, die diese Route behandelt werden sollen, einschließlich Anfangsziffern, hinzufügen und Definieren der Präfixe der + bei Bedarf. Beispielsweise können Sie alle Rufnummern beginnend mit +123 übereinstimmen.
