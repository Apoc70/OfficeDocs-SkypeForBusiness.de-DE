# <a name="ip-version-planning-tool"></a>IP Version (Planning Tool)
 
Sie haben drei Optionen, wenn Sie in Skype for Business Server IP-Adressen konfigurieren. Sie können Skype for Business Server so konfigurieren, dass nur IP Version 4 (IPv4), nur IP Version 6 (IPv6) oder eine Kombination aus beidem (als dualer Stack bezeichnet) unterstützt wird.
  
Weitere Informationen zum Auswählen Ihrer IP-Infrastruktur finden Sie unter [Planen von IPv6 in Skype for Business](../../../plan-your-deployment/network-requirements/ipv6.md).
