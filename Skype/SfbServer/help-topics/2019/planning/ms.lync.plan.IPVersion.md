# <a name="ip-version-planning-tool"></a>IP-Version (Planungstool)
 
Ihnen stehen drei Optionen beim Konfigurieren von IP-Adressen in Skype für Business Server 2015. Sie können Skype für Business Server 2015 unterstützen nur IP Version 4 (IPv4), nur IP Version 6 (IPv6), konfigurieren oder eine Kombination aus beidem (als eine mit doppeltem Stack bezeichnet).
  
Weitere Informationen zur Auswahl der IP-Infrastruktur finden Sie unter [Planen für IPv6 in Skype für Unternehmen](../../../plan-your-deployment/network-requirements/ipv6.md).
