# <a name="voice-infrastructure-planning-tool"></a>VoIP-Infrastruktur (Planungstool)
 
Skype für Business Server bietet zuverlässige, skalierbare PSTN-Anbindung mithilfe von SIP-Trunks, direkte SIP-Verbindungen mit einem PSTN-Gateway oder direkte SIP-Verbindungen mit einer Nebenstellenanlage. Weitere Informationen hierzu finden Sie unter [Planen der PSTN-Konnektivität in Skype für Business Server](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md).
