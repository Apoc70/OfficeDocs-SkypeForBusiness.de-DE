# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
Das Mobilitätsfeature in Skype for Business Server unterstützt die Funktionalität von Skype for Business auf mobilen Geräten.
  
Die Mobilitäts Funktionalität ist in Skype for Business Server wie in lync Server 2013 identisch. Weitere Informationen zum Vorbereiten Ihrer Umgebung für Mobilität finden Sie unter [Planen der Mobilität in lync Server 2013](https://technet.microsoft.com/en-us/library/hh689981%28v=ocs.15%29.aspx).
