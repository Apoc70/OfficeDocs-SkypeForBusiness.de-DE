# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
Das Mobilitätsfeature in Skype für Business Server unterstützt Skype für Business-Funktionen auf mobilen Geräten.
  
Mobilitätsfunktionen entspricht in Skype für Business Server wie in Lync Server 2013. Ausführliche Informationen zum Vorbereiten der Umgebung für die Mobilität finden Sie unter [Planning for Mobility im Lync Server 2013](https://technet.microsoft.com/en-us/library/hh689981%28v=ocs.15%29.aspx).
