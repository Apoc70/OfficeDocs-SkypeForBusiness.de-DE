# <a name="certificate-request-additional-subject-alternate-names"></a><span data-ttu-id="9fc07-106">Zertifikatsanforderung (zusätzliche alternative Antragstellernamen)</span><span class="sxs-lookup"><span data-stu-id="9fc07-106">Certificate Request (Additional Subject Alternate Names)</span></span>
 
<span data-ttu-id="9fc07-p102">Die Option **Weitere alternative Antragstellernamen konfigurieren** bietet eine Möglichkeit, dem Zertifikat vor dessen Fertigstellung und Anforderung weitere alternative Antragstellernamen hinzuzufügen. Geben Sie zum Hinzufügen eines alternativen Antragstellernamens dessen vollqualifizierten Domänennamen ein, und klicken Sie dann auf **Hinzufügen**. Wenn Sie einen alternativen Antragstellernamen hinzugefügt haben, den Sie löschen möchten, markieren Sie dessen Eintrag und klicken Sie auf **Entfernen**. Klicken Sie zum Entfernen aller Einträge mit alternativen Antragstellernamen auf **Alle löschen**.</span><span class="sxs-lookup"><span data-stu-id="9fc07-p102">The **Configure Additional Subject Alternate Names** option provides the opportunity to add any additional subject alternate names (SANs) to the certificate before finalizing and requesting the certificate. To add the SAN, type the fully qualified domain name (FQDN) of the SAN and then click **Add**. If you have added a SAN that you must delete, highlight the SAN entry and click **Remove**. To remove all SAN entries that you have added, click **Clear All**.</span></span>
  
> [!NOTE]
> <span data-ttu-id="9fc07-p103">Die Schaltflächen **Entfernen** und **Alle löschen** funktionieren nur für alternative Antragstellernamen, die zu dieser Seite hinzugefügt wurden. Die Schaltflächen haben keinen Einfluss auf Einträge mit alternativen Antragstellernamen, die sich derzeit auf dem Zertifikat befinden.</span><span class="sxs-lookup"><span data-stu-id="9fc07-p103">The **Remove** or **Clear All** buttons work only on SAN entries that are added on this page. The buttons will not affect SAN entries that are currently on the certificate.</span></span>
