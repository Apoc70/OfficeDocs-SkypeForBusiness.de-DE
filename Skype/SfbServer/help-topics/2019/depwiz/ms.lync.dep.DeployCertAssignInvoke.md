# <a name="certificate-assignment-invoke"></a>Zertifikatzuweisung (Aufruf)
 
Die Seite **Befehle ausführen** zeigt die Aktionen, die durch die Aufgaben zum Zuweisen des Zertifikats an den Server. Wenn die Aufgaben abgeschlossen haben, können Sie das Protokoll anzeigen, das das Zertifikat Zuweisung zur Archivierung details. Klicken Sie zum Anzeigen der Protokolldatei auf **Protokoll anzeigen**.
