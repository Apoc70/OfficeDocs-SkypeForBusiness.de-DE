# <a name="remove-certificate-invoke"></a>Entfernen von Zertifikaten (Aufruf)
 
Auf der Seite " **Befehle ausführen** " finden Sie Informationen zum Entfernen des Zertifikats. Überprüfen Sie die Zusammenfassungsinformationen aus der Zertifikat Entfernung. Stellen Sie sicher, dass die Zertifikat Protokolldatei entfernen in der Dropdownliste angezeigt wird, und klicken Sie dann auf **Protokoll anzeigen** , um die Details der Zertifikat Entfernung zu überprüfen. Wenn Sie die Überprüfung der Zusammenfassung und der Protokolldatei abgeschlossen haben, klicken Sie auf **Fertig stellen**.
