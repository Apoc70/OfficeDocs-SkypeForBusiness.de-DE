# <a name="sql-server-reporting-services-groups"></a>SQLServer Reporting Services (Gruppen)

Geben Sie in der **Read-Only-Gruppe angeben**der Gruppe, die Sie schreibgeschützten Zugriff auf Überwachungsberichte gewähren möchten. Verwenden Sie das Format &lt;Domäne&gt;\\&lt;Zugriffsgruppe&gt;. Beispiel: Contoso\RTCUniversalReadOnlyAdmins.
  
Zuweisen einer Gruppe schreibgeschützten Zugriff ist optional. Wenn Sie nicht jetzt schreibgeschützten Zugriff gewähren möchten, können Sie es später direkt in SQL Server Reporting Services konfigurieren.
