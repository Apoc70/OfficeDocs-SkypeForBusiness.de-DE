# <a name="prepare-current-forrest-intro"></a>Vorbereiten der aktuellen Gesamtstruktur (Einführung)
 
Klicken Sie auf **Weiter**, um die Aufgabe **Gesamtstruktur vorbereiten** zu starten. Durch Ausführen der Aufgabe " **Gesamtstruktur vorbereiten** " werden globale Einstellungen erstellt und universelle Gruppen für die Verwendung durch Skype for Business Server erstellt.
