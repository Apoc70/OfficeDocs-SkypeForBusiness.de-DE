# <a name="sql-server-reporting-services-prerequisites-not-satisfied"></a>SQL Server Reporting Services (Voraussetzungen nicht erfüllt)

Diese Seite wird angezeigt, wenn in Ihrer Infrastruktur keine Monitoring Server bereitgestellt sind. Dies bedeutet, dass die Mindestanforderungen für die Bereitstellung von Monitoring Server-Berichten noch nicht erfüllt sind.

Um dieses Problem zu beheben, stellen Sie sicher, dass Sie einen Monitoring Server Mitglied der Domäne, dass sie im Topologie-Generator definiert ist und die Topologie veröffentlicht wurde. SQL Server Reporting Services muss auch auf dem SQL Server verfügbar, und als ein Feature in der Monitoring Server-Datenbank auf dem SQL Server installiert.

Weitere Informationen hierzu finden Sie unter [Monitoring Reports in Skype für Business Server installieren](../../../deploy/deploy-monitoring/install-monitoring-reports.md) und [Bereitstellen von Monitoring](https://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx).
