# <a name="certificate-request-online-offline"></a>Zertifikatsanforderung (online/offline)
 
Wählen Sie **Anforderung jetzt vorbereiten, aber später (offline-zertifikatanforderung) senden**, um eine verzögerte Anforderung oder einer offlineanforderung zu erstellen. Wählen Sie **Anforderung unmittelbar an eine Onlinezertifizierungsstelle senden**, um eine sofortige Anforderung an eine Onlinezertifizierungsstelle (in der Regel ein internes System in Ihrer Organisation public Key-Infrastruktur) zu erstellen.
  
Klicken Sie nach Ihrer Wahl auf **Weiter**, um fortzufahren. Wenn Sie abbrechen möchten, ohne Änderungen vorzunehmen, klicken Sie auf **Abbrechen**. Klicken Sie auf **Zurück**, um zum vorherigen Schritt zurückzukehren.
