# <a name="start-services-invoke"></a>Starten von Diensten (Aufruf)
 
Bereich "Zusammenfassung", klicken Sie im Bereich **Befehle ausführen** zeigt den Status der Aufgaben ausgegeben, um die Dienste für die Skype für Business Server Role Server starten, das Sie bereitstellen. Die Zusammenfassung im Aufgabenbereich stellt keine Angabe zum Dienststart in Echtzeit dar. Einige der Skype für Business Server-Dienste dauert längeren Zeit mit der ersten Start zu beginnen. In den Aufgaben wird der Startbefehl aufgerufen, doch es wird nicht bestimmt, ob der Dienst erfolgreich gestartet wurde. Wenn Sie den Start und den Status von Diensten überwachen müssen, verwenden Sie die Windows-Option „Dienste“ in der Microsoft Management Console (MMC).
  
Unter dem Aufgabenbereich befindet sich eine Dropdownliste, in der die Protokolldatei zu **Dienste starten** enthalten ist. Klicken Sie zum Anzeigen der Protokolldatei auf **Protokoll anzeigen**. Klicken Sie auf **Fertig stellen**, um die Aufgabe abzuschließen.
