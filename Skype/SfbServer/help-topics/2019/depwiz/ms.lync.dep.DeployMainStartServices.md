# <a name="start-services"></a>Starten von Diensten
 
Klicken Sie zum Starten der Dienste für die bereitgestellte Serverrolle in **Schritt 4: Starten von Diensten**auf **Ausführen** . Sie müssen als Mitglied der lokalen Gruppe Administratoren oder als gleichwertig angemeldet sein. 
  
> [!NOTE]
> Wenn **Dienste starten** ausgeführt wurde, können Sie den Befehl erneut ausführen, indem Sie **erneut auf Ausführen**klicken.
