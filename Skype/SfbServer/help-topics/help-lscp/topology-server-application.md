# <a name="topology-server-application"></a>Topologie: Serveranwendung
 
Die **Topologie**: **Serveranwendung** Seite zeigt die serveranwendungen in Ihrer Bereitstellung.
  
## <a name="tasks-you-can-perform"></a>Mögliche Aufgaben

Sie können die folgenden Aufgaben ausführen, auf die **Topologie**: **Serveranwendung** Seite:
  
- Anzeigen der serveranwendungen in Ihrer Bereitstellung
    
- Aktivieren oder Deaktivieren einer Serveranwendung
    
- Markieren oder Markierung aufheben einer Anwendung als kritisch
