Skype for Business Server umfasst eine umfangreiche Reihe integrierter Administratorrollen und ermöglicht Ihnen außerdem, neue Rollen zu erstellen und eine benutzerdefinierte Liste mit Cmdlets für jede neue Rolle festzulegen. Sie können auch Skripten von Cmdlets zu den zulässigen Aufgaben von vordefinierten und benutzerdefinierten RBAC-Rollen hinzufügen. Details zu den vordefinierten Administratorrollen, die in Skype for Business Server zur Verfügung stehen, finden Sie unter Rollenbasierte Zugriffssteuerung.
