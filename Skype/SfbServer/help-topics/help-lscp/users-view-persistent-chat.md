# <a name="users-view-persistent-chat"></a>Benutzer: Anzeigen des beständigen Chats

Mitglieder der **CsPersistentChatAdministrator** -Rolle (Persistent Chat-Administratoren) können Chatroom Managerrechte und-Berechtigungen anderen Benutzern und sich selbst erteilen.

A Persistent Chat Administrator können folgende Aktionen ausführen:

- Erstellen von roomkategorien und Chatrooms.

- Festlegen der Mitgliedschaft aller Kategorien und Chatrooms.

- Verwalten Sie aller Einstellungen für alle Kategorien und Chatrooms.

- Aktivieren Sie oder deaktivieren Sie der Richtlinie für Persistent Chat Server.

- Festlegen Sie und verwalten Sie der Konfigurationseinstellungen in einem Pool Persistent Chat Server.

Weitere Informationen hierzu finden Sie unter [Adding a Persistent Chat Administrator](https://technet.microsoft.com/library/c107eb20-4e58-4463-b4f9-63fb5b1d9534.aspx) in der Bereitstellungsdokumentation.

## <a name="tasks-that-you-can-perform"></a>Mögliche Aufgaben

Sie können die folgenden Aufgaben ausführen, auf dem **Benutzer: beständigen Chat anzeigen** Seite:

- [Aktivieren eines Benutzers zum Verwalten von Kategorien, Chatrooms, Benutzerrechten und Berechtigungen](https://technet.microsoft.com/library/6c551be3-bc74-4d0e-9008-ddfabd86e940.aspx)

- [Grundlegendes zur Persistent Chat-Mitgliedschaft](https://technet.microsoft.com/library/900392d6-6e9f-4dae-93d6-39d7474409ef.aspx)

Ausführliche Informationen zu den verschiedenen Verfahren, die Sie mithilfe der Skype für Business Server-Systemsteuerung ausführen können, finden Sie unter [Verwalten von Skype für Business Server 2015](../../manage/manage.md).
