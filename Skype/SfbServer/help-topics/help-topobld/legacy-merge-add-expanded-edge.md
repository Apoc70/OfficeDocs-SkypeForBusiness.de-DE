# <a name="legacy-merge-add-expanded-edge"></a>Legacyzusammenführung – Hinzufügen einer erweiterten Edgekomponente
 
Skype for Business Server bietet keine Unterstützung für das Zusammenführen eines Legacy Office Communications Server 2007 R2 erweiterte Edgeserver Konfiguration. Der erweiterte Edgeserver muss zunächst in einen konsolidierten Edgeserver umgewandelt werden, ehe die Zusammenführung der Edgeserverinformationen erfolgen kann.
