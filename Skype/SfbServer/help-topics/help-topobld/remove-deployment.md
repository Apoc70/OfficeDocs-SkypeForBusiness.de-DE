# <a name="remove-deployment"></a>Entfernen der Bereitstellung
 
Klicken Sie auf **Fertig stellen** , um als ersten Schritt bei der Außerbetriebnahme der gesamten Bereitstellung eine leere Topologie zu erstellen.
  
> [!CAUTION]
> Dieser Schritt kann nicht rückgängig gemacht werden.
