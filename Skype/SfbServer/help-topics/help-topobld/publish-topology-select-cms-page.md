# <a name="publish-topology-select-cms-page"></a>Veröffentlichen der Topologie – Seite "Zentralen Verwaltungsserver auswählen"

Sie veröffentlichen die Topologie, die Sie mithilfe des Topologie-Generators konfiguriert haben. Sie werden aufgefordert, aus einer Liste auszuwählen, die Front-End-Server oder Front-End-Pool übernimmt die Rolle des zentralen Verwaltungsspeichers. Nur eine Front-End-Server oder Front-End-Pool kann diese Rolle zu einem bestimmten Zeitpunkt innehaben.

Weitere Informationen zum zentralen Verwaltungsspeicher finden Sie unter [Migrieren des lync Server 2010 zentralen Verwaltungsservers in lync Server 2013](https://technet.microsoft.com/library/30cc98f2-1916-4dbe-99d0-8df5368ed3ec.aspx).
