# <a name="add-edge-machine-external-ip-2010"></a>Hinzufügen der externen IP-Adresse zum Edgecomputer – 2010

Sie können die Eigenschaften auf dieser Seite so konfigurieren, dass Sie den Edgeserver auf die externen Speicherorte des Zugriffs-Edgedienst, des Webkonferenz-Edge-Diensts und des A/V-Edgedienst verweisen.

- Geben Sie unter **externe IPv4-Adressen**die externen IPv4-Adressen des Edge-Servers in **Access Edge Service**, **Web Conferencing Edge Service**und **A/V Edge Service ein**.

- Klicken Sie auf **OK** , um Ihre Änderungen im Topologie-Dokument zu speichern und zu übernehmen.

- Klicken Sie auf **Abbrechen** , um die Änderungen zu verwerfen und den Assistenten zum **Hinzufügen von Servern zum Edge-Pool zu** schließen.

- Klicken Sie auf **Hilfe** , um dieses Hilfethema zu lesen.

Details zum Hinzufügen von Edgeserver zu einer vorhandenen Topologie finden Sie unter [Definieren Ihrer Edge-Topologie](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx). Informationen zum Migrieren von Edge-Servern finden Sie unter [Phase 5: Hinzufügen von lync Server 2013 Preview Edge Server zu Pilot Pool](https://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx).
