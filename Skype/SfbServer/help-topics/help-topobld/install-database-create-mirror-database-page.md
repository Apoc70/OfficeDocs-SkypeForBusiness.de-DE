# <a name="install-database-create-mirror-database-page"></a>Installieren der Datenbank – Seite "Spiegeldatenbanken erstellen"
 
Diese Dokumentation ist vorläufig und kann geändert werden. Leere Themen sind als Platzhalter enthalten.
  
Wenn Sie nicht die SQL Server-Spiegeldatenbanken während der Veröffentlichungsvorgang installiert haben, installieren Sie sie mit der **Datenbank installieren** aus einer rechten Maustaste auf den Knoten im Topologie-Generator. Sie wählen den spiegelspeicher zu installieren. Klicken Sie auf **Einstellungen** , um alle erweiterten Einstellungen zu definieren.
