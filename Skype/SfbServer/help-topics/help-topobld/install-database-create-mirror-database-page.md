# <a name="install-database-create-mirror-database-page"></a>Installieren der Datenbank – Seite "Spiegeldatenbanken erstellen"
 
Dies ist eine vorläufige Dokumentation und kann jederzeit geändert werden. Leere Themen wurden als Platzhalter hinzugefügt.
  
Wenn Sie während des Veröffentlichungsprozesses die SQL Server Spiegeldatenbanken nicht installiert haben, installieren Sie Sie mit der **Installationsdatenbank** über einen rechten Mausklick auf einen beliebigen Knoten im Topologie-Generator. Klicken Sie dazu mit der rechten Maustaste im fea-cs-topo-tool auf einen beliebigen Knoten, und wählen Sie den zu installierenden Spiegelspeicher aus. Klicken Sie auf **Einstellungen**, um erweiterte Einstellungen zu definieren.
