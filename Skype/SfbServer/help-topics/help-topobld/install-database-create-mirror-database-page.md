# <a name="install-database-create-mirror-database-page"></a>Installieren der Datenbank – Seite "Spiegeldatenbanken erstellen"
 
Diese Dokumentation ist vorläufig und kann geändert werden. Leere Themen sind als Platzhalter enthalten.
  
Wenn Sie die SQL Server-Spiegeldatenbanken während des Veröffentlichungsprozesses nicht installieren möchten, installieren Sie Sie mit der rechten Maustaste auf einen beliebigen Knoten im Topologie-Generator mit der Option " **Datenbank installieren** ". Sie wählen den zu installierenden Spiegel Speicher aus. Klicken Sie auf **Einstellungen** , um erweiterte Einstellungen zu definieren.
