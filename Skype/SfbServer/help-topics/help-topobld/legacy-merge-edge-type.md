# <a name="legacy-merge-edge-type"></a>Legacyzusammenführung – Edgetyp
 
Klicken Sie für einen **einzelnen Edgeserver** oder einen **konsolidierten Edgeserver mit Lastenausgleich** auf **weiter** , um Konfigurationsinformationen anzugeben. Für einen **erweiterten Edge**-Server muss die Konfiguration jedes Servers in eine konsolidierte Edge-Konfiguration geändert werden, bevor Sie mit lync Server 2013 zusammengeführt werden kann.
