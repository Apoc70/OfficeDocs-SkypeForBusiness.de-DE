# <a name="add-edge-server-support"></a>Hinzufügen der Edgeserverunterstützung

Sie können Unterstützung für den Zugriff durch externe Benutzer hinzufügen, wenn Sie Ihre anfängliche Topologie oder zu einem späteren Zeitpunkt bereitstellen. Ausführliche Informationen zum Hinzufügen von Edgeservern zu einer vorhandenen Topologie finden Sie in der Edgeserver-Bereitstellungsdokumentation unter [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx).
