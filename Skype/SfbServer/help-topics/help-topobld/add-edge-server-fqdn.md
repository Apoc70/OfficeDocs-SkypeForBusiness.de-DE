# <a name="add-edge-server-fqdn"></a>Hinzufügen des Edgeserver-FQDN
 
Sie müssen einen vollqualifizierten Domänennamen für den Zugriffs-Edgedienst eingeben. Wenn Sie auf der Seite **Features auswählen** die Option **eine einzelne FQDN &amp; -IP-Adresse verwenden** nicht ausgewählt haben, müssen Sie auch einen FQDN für den Webkonferenz-Edgedienst und für die A/V-Edgedienst angeben.
  
Wenn Sie die Option **nur einen einzelnen FQDN &amp; -IP-Adresse verwenden** ausgewählt haben, müssen Sie für jeden der Edge-Dienste eine andere Portnummer angeben (empfohlene Porteinstellungen: 444 für Zugriffs-Edgedienst, 8057 für Webkonferenz-Edgedienst und 443 für A/V-Edgedienst). Wenn Sie diese Option nicht ausgewählt haben, können Sie für alle drei Dienste dieselbe Portnummer (z. B. 443) verwenden.
