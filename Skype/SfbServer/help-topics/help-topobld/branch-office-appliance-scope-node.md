# <a name="branch-office-appliance-scope-node"></a>Branch Office Appliance – Bereichsknoten
 
Dieser Knoten enthält den vollqualifizierten Domänennamen (Fully Qualified Domain Name, FQDN), zugeordnete Pools und Server sowie Ausfallsicherheitsinformationen zu diesem Survivable Branch Appliance. Folgende Möglichkeiten stehen zur Verfügung:
  
- **Eigenschaften bearbeiten**. Öffnet die Seite **Eigenschaften bearbeiten** für diese Survivable Branch Appliance, in der Sie den FQDN (wenn die Topologie noch nicht veröffentlicht ist), die Zuordnungen und Einstellungen für die Ausfallsicherheit ändern können.
    
- **Löschen**: Dient zum Löschen des Servers aus der Topologie.
    
Zusätzlich zu diesen Optionen sind die folgenden allgemeinen Optionen verfügbar:
  
- **Topologie**. Sie können diese Option verwenden, um eine neue Topologie zu definieren, eine vorhandene Topologie zu öffnen, die aktuelle Topologie herunterzuladen, eine Kopie der Topologie zu speichern, die Topologie zu veröffentlichen, eine Datenbank zu installieren oder die Bereitstellung zu entfernen.
    
- **Ansicht**: Mithilfe dieser Option können Sie die Ansicht des Topologie-Generators anpassen, indem Sie die Elemente auswählen, die in den Ansichten der Konsolenstruktur ein- bzw. ausgeblendet werden sollen.
    
- **Hilfe**: Über diese Option greifen Sie auf die Hilfe für den aktiven Knoten zu.
