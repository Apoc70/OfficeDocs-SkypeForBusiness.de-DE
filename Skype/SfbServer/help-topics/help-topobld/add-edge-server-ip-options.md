# <a name="add-edge-server-ip-options"></a>Hinzufügen der IP-Adresse des Edgeservers – Optionen
 
Microsoft Lync Server 2013 können Sie IPv4 und IPv6-Adressen für jede Schnittstelle für den Edge-Server und Edge-Pool zu konfigurieren. Zu diesem Zweck führen Sie die folgenden Aufgaben:
  
- **IPv4 für interne Schnittstelle aktivieren**: Aktivieren Sie das Kontrollkästchen, wenn Sie eine IPv4-Adresse auf den Edge-Server oder Pool interne Schnittstelle des Edgeservers anwenden möchten.
    
- **IPv6 für interne Schnittstelle aktivieren**: Aktivieren Sie das Kontrollkästchen, wenn Sie eine IPv6-Adresse auf den Edge-Server oder Pool interne Schnittstelle des Edgeservers anwenden möchten.
    
- **IPv4 für externe Schnittstelle aktivieren**: Aktivieren Sie das Kontrollkästchen, wenn Sie eine IPv4-Adresse auf den Edge-Server oder Pool externe Schnittstelle des Edgeservers anwenden möchten.
    
- **IPv6 für externe Schnittstelle aktivieren**: Aktivieren Sie das Kontrollkästchen, wenn Sie eine IPv6-Adresse auf den Edge-Server oder Pool externe Schnittstelle des Edgeservers anwenden möchten.
    
Sie können auch den Edgeserver oder edgepool Verwendung eine Adresse Übersetzung-Netzwerkadresse für die externen IP-Adressen konfigurieren. Zu diesem Zweck des Kontrollkästchens **wird die externe IP-Adresse dieses Edge-Pools von NAT übersetzt**.
  
NAT-Unterstützung. Netzwerkadressübersetzung (NAT) wird nicht unterstützt, bei Verwendung der Hardwaregeräte zum Lastenausgleich, sodass die NAT-Option nicht auswählen, wenn Sie einen Edge-Server-Pool mit Hardwaregerät zum Lastenausgleich bereitstellen.
