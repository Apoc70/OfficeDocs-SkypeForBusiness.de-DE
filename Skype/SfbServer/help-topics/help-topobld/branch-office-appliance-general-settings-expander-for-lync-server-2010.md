# <a name="branch-office-appliance-general-settings-expander-for-lync-server-2010"></a>Allgemeine Einstellungen für Filialanwendungen in Lync Server 2010 – Erweiterung
 
So bearbeiten Sie Eigenschaften für Survivable Branch Appliance oder einen Survivable Branch Server, klicken Sie unter **Allgemein**, konfigurieren Sie:
  
- **FQDN**: Geben Sie den vollqualifizierten Domänennamen des Survivable Branch Appliance oder einen Survivable Branch Server.
    
    **Verwenden Sie alle konfigurierte IP-Adressen** verwendet die IP-Adressen auf dem Survivable Branch Appliance oder einen Survivable Branch Server für alle Zwecke konfiguriert sind.
    
    **Dienstnutzung auf die ausgewählten IP-Adressen beschränken**: Konfigurieren Sie die unterschiedlichen Adressen, mit denen der Server und die IP-Adresse für PSTN definiert werden.
    
    **Primäre IP-Adresse**: Die IP-Adresse ist für alle Zwecke definiert und konfiguriert, mit Ausnahme von Funktionen mit PSTN-Zuordnung.
    
    **PSTN-IP-Adresse**: Dies ist die IP-Adresse, die den PSTN-Funktionen (Public Switched Telephone Network) zugeordnet ist.
    
- Sie konfigurieren **Zuordnungen** , um sicherzustellen, dass andere Server-Rollen konfiguriert und der Survivable Branch Appliance oder einen Survivable Branch Server zugeordnet sind.
    
    **Zuordnen der Archivierungsserver** Wählen Sie aus der Liste der Archivierungsserver, die Sie der Survivable Branch Appliance oder einen Survivable Branch Server zuordnen möchten.
    
    Klicken Sie auf **neu** , wenn Sie den Archivierungsserver nicht erstellt haben, die Sie dieser Survivable Branch Appliance oder einen Survivable Branch Server zuordnen möchten.
    
    **Monitoring Server zuordnen** Wählen Sie aus der Liste der Monitoring Server, die Sie der Survivable Branch Appliance oder einen Survivable Branch Server zuordnen möchten.
    
    Klicken Sie auf **neu** , wenn Sie nicht den Monitoring Server erstellt haben, die Sie dieser Survivable Branch Appliance oder einen Survivable Branch Server zuordnen möchten.
    
    **Edgepool zuordnen (für Medienkomponenten)** Wählen Sie aus der Liste den Edgeserver oder edgepool, mit dem Sie die Survivable Branch Appliance oder einen Survivable Branch Server zuordnen möchten.
    
  **OK**: Mit dieser Option werden die Änderungen am Dialogfeld akzeptiert und übernommen.
  
  **Abbrechen**: Mit dieser Option werden die Änderungen verworfen und das Dialogfeld wird geschlossen.
  
  **Hilfe**: Mit dieser Option zeigen Sie diese Hilfeseite an.
