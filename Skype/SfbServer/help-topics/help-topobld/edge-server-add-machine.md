Nachdem Sie die Topologie veröffentlicht haben, müssen Sie den Server im Topologie-Generator durch Ändern des FQDN löschen und dann einen neuen Server zum Pool mit dem neuen FQDN hinzufügen. Details zum Entwerfen der Topologie für einen Edge-Pool finden Sie unter Definieren Ihrer Edge-Topologie in der Edge-Bereitstellungsdokumentation.
