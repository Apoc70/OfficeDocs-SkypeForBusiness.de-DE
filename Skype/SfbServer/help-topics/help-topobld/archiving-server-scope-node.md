# <a name="archiving-server-scope-node"></a>Archivierungsserver – Bereichsknoten
 
Im Topologie-Generator können Sie Aufgaben im Zusammenhang mit dem Server ausführen, auf dem die Archivierung ausgeführt wird, indem Sie in der Konsolenstruktur mit der rechten Maustaste auf den Pool klicken und **Eigenschaften bearbeiten**auswählen. Auf der Seite **Eigenschaften bearbeiten** können Sie den vollqualifizierten Domänennamen (Fully Qualified Domain Name, FQDN) des Servers, auf dem die Archivierung durchführen wird, die Archivierungs SQL Server Datenbank und den Archivierungsdatei Speicher ändern.
