# <a name="add-default-sip-domain"></a>Hinzufügen der Standard-SIP-Domäne
 
Geben Sie die primäre SIP-Domäne (Session Initiation Protocol) an, die Ihre Bereitstellung unterstützen soll. Die primäre SIP-Domäne kann mit der Active Directory-Domänendienste Domäne identisch oder unterschiedlich sein. Ihre Active Directory-Domäne kann beispielsweise "fabrikam.net" lauten, und als primäre SIP-Domäne kann "contoso.net" angegeben sein.
