# <a name="legacy-merge-summary-page"></a>Legacyzusammenführung – Zusammenfassungsseite

Die Migration von Microsoft Office Communications Server 2007 R2 zu lync Server 2013 verwendet Features des Topologie-Generators, um die vorhandenen Topologien zusammenzuführen. Der Bildschirm "Zusammenfassung" gibt alle Aktionen und Auswahlen an, die Sie während der Konfiguration der Fusion von Office Communications Server 2007 R2 in die Konfiguration des lync Server 2013-Topologie-Generators vorgenommen haben.

Überprüfen Sie die Zusammenfassung, und ermitteln Sie, ob Ihre Auswahl richtig definiert wurde. Fahren Sie mit der Fusion fort, wenn Sie mit der Auswahl zufrieden sind, die sich in der Zusammenfassung widerspiegelt.
