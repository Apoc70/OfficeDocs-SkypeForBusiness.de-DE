# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Hinzufügen des Front-End-SQL-Speichers für Lync Server 2010 – Seite
 
Sie definieren die Eigenschaften der SQL Server-Speicher für den neuen Front-End-Pool, den Sie konfigurieren.
  
> [!IMPORTANT]
> Wenn Sie einen Standard Edition-Server konfigurieren, wird dieses Dialogfeld angezeigt, Sie können jedoch keine Einstellungen konfigurieren. Klicken Sie auf **Weiter**, um fortzufahren.
  
Einstellungen, die Sie konfigurieren können:
  
- **Verwenden eines zuvor definierten SQL Server-Speichers**: Wählen Sie in der Liste für den Front-End-Poolspeicher einen vorhandenen SQL Server Store aus.
    
- Wählen Sie **einen neuen SQL Server-Speicher definieren** aus, um einen neuen SQL Server-Speicher zu erstellen.
    
    Sie geben den vollqualifizierten Domänennamen des SQL Server in **SQL Server-FQDN** ein.
    
    Die **Standardinstanz** ist zunächst ausgewählt. Wenn Sie eine Instanz definieren möchten, wählen Sie **benannte Instanz** aus, und geben Sie den Namen der Instanz ein, die Sie erstellen müssen.
    
  **Zurück**: wechselt zum vorherigen Dialogfeld zurück
  
  **Nächstes**: übergibt die Änderungen und wechselt zum nächsten Schritt-Dialogfeld.
  
  **Abbrechen**: verwirft Änderungen und bricht die Konfiguration ab
  
  **Hilfe**: Zeigt Hilfeinformationen wie dieses Thema an.
