# <a name="add-archiving-server"></a>Hinzufügen des Archivierungsservers
 
Zum Bereitstellen eines Archivierungsservers innerhalb des Topologieentwurfs müssen Sie den vollqualifizierten Domänennamen des Servers angeben, auf dem Sie die Archivierung bereitstellen möchten. Vor dem Veröffentlichen einer Topologie, die das Archivieren und Installieren von Skype for Business Server umfasst, sollten Sie alle Voraussetzungen für die Bereitstellung der Archivierung erfüllt haben.
