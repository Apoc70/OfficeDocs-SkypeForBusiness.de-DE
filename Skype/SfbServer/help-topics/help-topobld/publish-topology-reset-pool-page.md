# <a name="publish-topology-reset-pool-page"></a>Veröffentlichen der Topologie – Seite "Pool zurücksetzen"
 
Dies ist eine vorläufige Dokumentation und kann jederzeit geändert werden. Leere Themen wurden als Platzhalter hinzugefügt.
  
Wenn Sie einem vorhandenen Pool neue Computer hinzufügen möchten, müssen Sie den Pool zurücksetzen, damit der Windows-Fabric die Verbindungen zwischen allen Mitgliedern des Pools neu definieren kann. Wählen Sie dazu den Pool aus, dem Sie Server hinzugefügt haben, und klicken Sie dann auf **weiter**.
