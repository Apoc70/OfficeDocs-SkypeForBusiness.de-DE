# <a name="publish-topology-reset-pool-page"></a>Veröffentlichen der Topologie - Seite "Pool zurücksetzen"
 
Diese Dokumentation ist vorläufig und kann geändert werden. Leere Themen sind als Platzhalter enthalten.
  
Zum Hinzufügen neuer Computer zu einem vorhandenen Pool müssen Sie den Pool zurücksetzen, damit Windows Fabric die Verbindungen zwischen allen Mitgliedern des Pools neu definieren kann. Wählen Sie dazu den Pool aus, dem Sie Server hinzugefügt haben, und klicken Sie auf **Weiter**.
