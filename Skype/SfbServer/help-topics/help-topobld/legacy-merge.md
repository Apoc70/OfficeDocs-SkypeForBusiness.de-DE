# <a name="legacy-merge"></a>Legacyzusammenführung

Der **externe FQDN** für Webkonferenzen ermöglicht externen Benutzern, an lokalen Besprechungen teilzunehmen. Geben Sie den vollqualifizierten Domänennamen (Fully Qualified Domain Name, FQDN) der externen Webkonferenz Schnittstelle des Legacy Edge-Servers ein.

Der externe **Portwert externer Webkonferenzen** von **443** ist das standardmäßige TCP-Protokoll (Transmission Control Protocol), das für Konferenz Clients konfiguriert ist. Wenn der Standardwert nicht verwendet wurde, aktualisieren Sie den externen **Port-Wert externer Webkonferenzen** .

Aktivieren Sie das Kontrollkästchen **dieser Edge-Pool wird für Verbund-und öffentliche Chat Verbindungen verwendet** , wenn Sie diesen Edgeserver für Föderation verwenden möchten. Wenn Sie mehrere Edgeserver bereitgestellt haben, wird nur einer dieser Server für den Verbund aktiviert. Wenn Sie dieses Kontrollkästchen nicht aktivieren und später entscheiden, dass Sie den Verbund aktivieren möchten, müssen Sie den Zusammenführungs-Assistenten für Topologie-Builder erneut ausführen und Ihre Topologie veröffentlichen. Ausführliche Informationen finden Sie unter [Phase 4: Zusammenführen](https://technet.microsoft.com/library/81eb5bb2-1fd7-4611-a2aa-eb2393c8abc9.aspx)von Topologien.
