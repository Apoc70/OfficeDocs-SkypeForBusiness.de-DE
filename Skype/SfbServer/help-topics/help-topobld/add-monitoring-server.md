# <a name="add-monitoring-server"></a>Hinzufügen des Monitoring Servers
 
Wenn Sie um einen Monitoring Server in Ihrer Topologieentwurf zu integrieren, müssen Sie den vollqualifizierten Domänennamen (FQDN) des Servers angeben, auf denen Sie den Monitoring Server bereitstellen möchten. Vor dem Veröffentlichen einer Topologie mit dem Monitoring Server und Skype für Business Server auf dem Server installieren, sollten Sie alle erforderlichen Komponenten für die Bereitstellung von monitoring abschließen. Ausführliche Informationen zu diesen Voraussetzungen finden Sie unter [Deploying Monitoring](http://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx) in der Bereitstellungsdokumentation.
