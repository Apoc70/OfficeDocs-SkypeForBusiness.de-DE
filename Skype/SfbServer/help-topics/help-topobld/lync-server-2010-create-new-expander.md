# <a name="lync-server-2010-create-new-expander"></a>Neuerstellen unter Lync Server 2010 – Erweiterung
 
Sie können neue Serverrollen im lync Server 2010-Knoten des Topologie-Generators erstellen, indem Sie mit der rechten Maustaste auf **lync Server 2010**klicken, **Neues**auswählen und dann die zu erstellende Serverrolle auswählen.
