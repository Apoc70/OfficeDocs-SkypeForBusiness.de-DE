# <a name="define-the-root-trunk-for-a-new-ip-or-pstn-gateway"></a>Definieren des Stammtrunks für eine neue IP-Adresse oder ein neues PSTN-Gateway

Sie definieren den Stamm Stamm für das IP-oder Public Switched Telephone Network (PSTN), indem Sie Folgendes konfigurieren:

- **Trunk Name**: definieren Sie den vollqualifizierten Domänennamen, der dem Stamm zugeordnet ist.

- **Abhör-Port für IP/PSTN-Gateway**: definieren Sie den Port, auf dem dieser trunk abhört.

- **SIP-Transport Protokoll**: Wählen Sie basierend auf den trunk Anforderungen aus der Liste entweder **TCP** oder **TLS**aus.

- **Zugeordneter Vermittlungsserver**: Wählen Sie aus der Liste der verfügbaren Vermittlungsserver in Ihrer Bereitstellung aus.

- **Zugeordneter Vermittlungsserver-Port**: definieren Sie den Port, auf dem der ausgewählte Vermittlungsserver zuhört
