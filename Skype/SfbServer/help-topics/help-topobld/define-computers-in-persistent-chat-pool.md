# <a name="define-computers-in-persistent-chat-pool"></a>Definieren von Computern im Pool für beständigen Chat
 
Sie definieren die Computer, die Teil eines Server Pools für beständigen Chat sein sollen, indem Sie den vollqualifizierten Domänennamen (Fully Qualified Domain Name, FQDN) in **Computer-FQDN**eingeben. Hierbei muss es sich um den Namen des Computers handeln, der in den DNS-Hosteinträgen (A oder AAAA) des Domänennamensystems definiert ist.
  
Klicken Sie auf **Hinzufügen** , um den FQDN der Liste der Computer im Pool hinzuzufügen.
  
Klicken Sie auf **Aktualisieren** , um die Einträge in der Liste der Computer auf den Computern zu aktualisieren, die die Liste der für den Pool definierten Computer bilden.
  
Klicken Sie auf **Entfernen** , um einen definierten Computer im Pool zu löschen.
  
Klicken Sie auf **Zurück**, um zum vorherigen Dialogfeld für die Pooldefinition zurückzukehren.
  
Klicken Sie auf **weiter** , nachdem Sie die Eingabe der Computer Mitglieder für diesen Pool abgeschlossen haben, um mit der Definition des Server Pools für beständigen Chat fortzufahren.
  
Klicken Sie auf **Abbrechen**, um alle Änderungen zu verwerfen und den Assistenten **Neuen Pool für beständigen Chat definieren** zu beenden.
  
Klicken Sie auf **Hilfe**, um auf die kontextbezogene Hilfe (z. B. diese Seite) zuzugreifen.
