# <a name="persistent-chat-scope-node"></a>Beständiger Chat – Bereichsknoten
 
Um mit der Erstellung eines neuen Server Pools für beständigen Chat oder eines persistenten Chats zu beginnen, klicken Sie im Topologie-Generator mit der rechten Maustaste auf beständige **Chat Pools** und wählen dann **neuer beständiger Chat Pool** aus.
  
- **Topologie**. Sie können diese Option verwenden, um eine neue Topologie zu definieren, eine vorhandene Topologie zu öffnen, die aktuelle Topologie herunterzuladen, eine Kopie der Topologie zu speichern, die Topologie zu veröffentlichen, eine Datenbank zu installieren, Komponenten aus früheren Versionen von Office Communications Server zu verbinden oder die Bereitstellungs.
    
- **Hilfe**. Sie können diese Option verwenden, um auf die Hilfe für den aktiven Knoten zuzugreifen.
