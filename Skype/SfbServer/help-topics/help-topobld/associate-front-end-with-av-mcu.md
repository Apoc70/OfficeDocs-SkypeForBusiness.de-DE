# <a name="associate-front-end-with-av-mcu"></a>Zuordnen des Front-Ends zur A/V MCU
 
A/V-Konferenzen ermöglichen die Audio- und Videokommunikation zwischen Benutzern, sofern sie über die entsprechenden Clientgeräte wie Headsets für Audiokonferenzen und Webcams für Videokonferenzen verfügen. Wenn Ihre Bereitstellung Konferenzen unterstützt und Sie sowohl Web- als auch A/V-Konferenzen aktivieren, können Sie den A/V-Konferenzserver mit dem Front-End-Server verbinden oder einen oder mehrere eigenständige A/V-Konferenzserver bereitstellen (A/V-Konferenzpool). Wenn Sie die Option zum Bereitstelleneiner eigenständigen A/V-Konferenzserver auswählen, müssen Sie Sie im Topologie-Generator definieren.
  
Alle Pools an einem Standort und die Pools mehrerer zentraler Standorte können dieselbe A/V-Konferenzserver verwenden, wenn die Auslastung der A/V-Konferenzserver nicht überschreitet.
