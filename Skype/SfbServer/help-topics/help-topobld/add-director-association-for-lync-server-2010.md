# <a name="add-director-association-for-lync-server-2010"></a>Hinzufügen von Director-Zuordnung für Lync Server 2010
 
Definieren Sie, ob der Director oder Director-pool-wird haben einen Monitoring Server zugeordnet. Wählen Sie das Kontrollkästchen, um der Director oder Director einen Monitoring Server zuordnen Pool.
  
> [!TIP]
> Wenn Sie einen Monitoring Server-Datenbank (und optional eine Instanz) noch nicht erstellt haben, erhalten Sie die Möglichkeit, die sie später erstellen, wenn Sie dieses Kontrollkästchen, um der Director oder Director einen Monitoring Server zuordnen auswählen Pool.
