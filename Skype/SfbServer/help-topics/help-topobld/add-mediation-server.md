# <a name="add-mediation-server"></a>Hinzufügen von Vermittlungsservern
 
Um den Vermittlungsserver zu definieren, geben Sie einen vollqualifizierten Domänennamen (Fully Qualified Domain Name, FQDN) für den vermittlungsserverpool an. Wenn Sie einen Pool von Vermittlungsservern erstellen, geben Sie den Namen des Pools für die von Ihnen bereitgestellten Vermittlungsserver an. Die vollqualifizierten Domänennamen der einzelnen Server werden in einem weiteren Schritt festgelegt. Wenn Sie einen einzelnen Vermittlungsserver bereitstellen, ist der von Ihnen angegebene FQDN der FQDN des einzelnen Vermittlungsservers. Ausführliche Informationen finden Sie unter [Bereitstellen eines Vermittlungsservers im Topologie-Generator in Skype for Business Server 2015](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md).
