Die vollqualifizierten Domänennamen der einzelnen Server werden in einem weiteren Schritt festgelegt. Wenn Sie einen einzelnen Vermittlungsserver bereitstellen, ist der von Ihnen angegebene FQDN der FQDN des einzelnen Vermittlungsservers. Ausführliche Informationen finden Sie unter Bereitstellen eines Vermittlungsservers im Topologie-Generator in Skype for Business Server 2015.
