# <a name="validate-your-cloud-connector-deployment"></a>Überprüfen der Cloud Connector-Bereitstellung
 
Nachdem Sie die Schritte zum Bereitstellen und Konfigurieren von Skype für Business Cloud Connector Edition abgeschlossen haben, stellen Sie sicher, dass Sie tätigen und Entgegennehmen von anrufen können.
  
## <a name="validate-the-deployment"></a>Überprüfen der Bereitstellung

1. Melden Sie sich bei Ihrem Skype for Business-Client mit einem zuvor eingerichteten Benutzerkonto an.
    
2. Rufen Sie vom Client aus eine PSTN-Nummer an und prüfen Sie, ob der Anruf erfolgreich verläuft.
    
3. Verwenden Sie Ihr Telefon oder ein anderes Gerät, das der PSTN-Nummer zugeordnet ist, um Ihren Skype for Business-Client anzurufen, und prüfen Sie, ob der Anruf erfolgreich verläuft.
    
    > [!NOTE]
    > Sie sollten prüfen, ob Sie erfolgreich Anrufe tätigen können, nachdem zusätzliche Standorte hinzugefügt oder Standorte konvertiert wurden, um HA-Unterstützung zu aktivieren.
