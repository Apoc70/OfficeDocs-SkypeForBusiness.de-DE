# <a name="develop-call-quality-dashboard-for-skype-for-business-server-2015"></a>Entwickeln von Anrufqualität Dashboard für Skype für Business Server 2015
 
**Zusammenfassung:** Informationen Sie zum Entwickeln von Lösungen für die Qualitätsdashboard aufrufen. Das Anrufqualitäts-Dashboard ist ein Tool für Skype for Business Server 2015.
  
Dieses Thema bietet Development-Dokumentation für aufrufen Quality Dashboard (CQD) für Skype für Business Server 2015.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server-2015"></a>Entwickeln von Anrufqualität Dashboard für Skype für Business Server 2015

CQD kann erweitert und Ihrer reporting-Umgebung mit der benutzerdefinierten Entwicklung integriert werden. 
  
Zwei Gruppen von APIs Web stehen zur Verfügung, eines für den Zugriff auf Daten im Cube (über Daten-APIs) und eines für den Zugriff auf die Berichtsmetadaten (über Repository-API). 
  
- [Daten-API für Anruf Qualitätsdashboard (CQD) in Skype für Business Server 2015](data-api.md)
    
- [Repository-API für Anruf Qualitätsdashboard (CQD) in Skype für Business Server 2015](repository-api.md)
    
- [CQD-Entwicklungsbeispiele](cqd-development-samples.md)
