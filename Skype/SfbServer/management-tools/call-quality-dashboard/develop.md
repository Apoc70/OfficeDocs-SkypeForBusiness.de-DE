# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Entwickeln von Anrufqualität Dashboard für Skype für Business Server
 
**Zusammenfassung:** Informationen Sie zum Entwickeln von Lösungen für die Qualitätsdashboard aufrufen. Anruf Qualitätsdashboard ist ein Tool für Skype für Business Server.
  
Dieses Thema bietet Development-Dokumentation für aufrufen Quality Dashboard (CQD) für Skype für Business Server.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Entwickeln von Anrufqualität Dashboard für Skype für Business Server

CQD kann erweitert und Ihrer reporting-Umgebung mit der benutzerdefinierten Entwicklung integriert werden. 
  
Zwei Gruppen von APIs Web stehen zur Verfügung, eines für den Zugriff auf Daten im Cube (über Daten-APIs) und eines für den Zugriff auf die Berichtsmetadaten (über Repository-API). 
  
- [Daten-API für Anruf Qualitätsdashboard (CQD) in Skype für Business Server](data-api.md)
    
- [Repository-API für Anruf Qualitätsdashboard (CQD) in Skype für Business Server](repository-api.md)
    
- [CQD-Entwicklungsbeispiele](cqd-development-samples.md)
