# <a name="skype-for-business-server-2015-management-tools"></a>Skype for Business Server 2015-Verwaltungstools
 
**Zusammenfassung:** Lernen Sie die Service-Verwaltungstools in Skype für Business Server 2015 aus.
  
Skype für Business Server 2015 Kommunikationssoftware (ehemals Lync Server) bietet Sofortnachrichten (IM), Anwesenheitsinformationen, Konferenzen und telefonielösungen, die Anforderungen an die Zusammenarbeit auf Unternehmensebene zu unterstützen. Die Werkzeuge zur Verwaltung dieser Dienste sind sowohl flexibel als auch leistungsfähig.
