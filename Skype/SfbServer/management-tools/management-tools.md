# <a name="skype-for-business-server-2015-management-tools"></a>Skype for Business Server 2015-Verwaltungstools
 
**Zusammenfassung:** Informieren Sie sich über die Dienstverwaltungstools in Skype for Business Server 2015.
  
Die Skype for Business Server 2015-Kommunikationssoftware (vormals lync Server) bietet Instant Messaging (im), Anwesenheits-, Konferenz-und Telefonie-Lösungen, die die Anforderungen an die Zusammenarbeit auf Unternehmensebene unterstützen. Die Werkzeuge zur Verwaltung dieser Dienste sind sowohl flexibel als auch leistungsfähig.
