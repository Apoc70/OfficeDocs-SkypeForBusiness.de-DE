# <a name="skype-for-business-server-2015-management-tools"></a>Skype for Business Server 2015 Verwaltungs Tools
 
**Zusammenfassung:** Erfahren Sie mehr über die Dienstverwaltungstools in Skype for Business Server 2015.
  
Skype for Business Server 2015 Kommunikationssoftware (ehemals lync Server) bietet Instant Messaging-, Anwesenheits-, Konferenz-und Telefonie-Lösungen, die Anforderungen an die Zusammenarbeit auf Unternehmensebene unterstützen. Die Tools für die Verwaltung dieser Dienste sind sowohl flexibel als auch leistungsfähig.
