# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
Das Mobilitätsfeature in Skype for Business Server 2015 unterstützt die Funktionalität von Skype for Business auf mobilen Geräten.
  
Die Mobilitäts Funktionalität ist wie in lync Server 2013. Weitere Informationen zum Vorbereiten Ihrer Umgebung für Mobilität finden Sie unter [Planen der Mobilität für Skype for Business Server](../../plan-your-deployment/mobility.md).
