# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
Das Mobilitätsfeature in Skype für Business Server 2015 unterstützt Skype für Business-Funktionen auf mobilen Geräten.
  
Mobilitätsfunktionen ist dieselbe wie in Lync Server 2013. Weitere Informationen zur Vorbereitung Ihrer Umgebung Mobilität finden Sie unter [Planen für die Mobilität für Skype für Business Server](../../plan-your-deployment/mobility.md).
