<span data-ttu-id="e7c59-105">Das Planungstool für Skype for Business Server 2015 ist ein Assistenten gesteuertes, Interview ähnliches Tool, das Fragen zur von Ihnen entworfenen Skype for Business Server 2015-Topologie stellt.</span><span class="sxs-lookup"><span data-stu-id="e7c59-105">The Skype for Business Server 2015 Planning Tool is a wizard driven, interview-like tool that asks questions about the Skype for Business Server 2015 topology that you are designing.</span></span> <span data-ttu-id="e7c59-106">Das Planungs Tool verwendet die bereitgestellten Informationen in Verbindung mit den bevorzugten Methoden für Topologie-Design und-Kapazität, um eine empfohlene Topologie basierend auf den bereitgestellten Antworten darzustellen.</span><span class="sxs-lookup"><span data-stu-id="e7c59-106">The Planning Tool uses the information supplied, coupled with preferred practices for topology design and capacity, to present a recommended topology based on the answers supplied.</span></span> <span data-ttu-id="e7c59-107">Sie können das Planungstool über das [Planungstool für Skype for Business Server 2015](https://go.microsoft.com/fwlink/p/?LinkID=282725)herunterladen.</span><span class="sxs-lookup"><span data-stu-id="e7c59-107">You can download the Planning Tool from the [Skype for Business Server 2015 Planning Tool](https://go.microsoft.com/fwlink/p/?LinkID=282725).</span></span>
