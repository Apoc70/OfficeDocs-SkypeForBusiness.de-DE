# <a name="ip-version-planning-tool"></a>IP Version (Planning Tool)
 
Sie haben drei Optionen, wenn Sie IP-Adressen in Skype for Business Server 2015 konfigurieren. Sie können Skype for Business Server 2015 so konfigurieren, dass nur IP Version 4 (IPv4) unterstützt wird, nur IP Version 6 (IPv6) oder eine Kombination aus beiden (auch als Dual-Stack bezeichnet).
  
Weitere Informationen zum Auswählen Ihrer IP-Infrastruktur finden Sie unter [Planen von IPv6 in Skype for Business](../../plan-your-deployment/network-requirements/ipv6.md).
