# <a name="persistent-chat-planning-tool"></a>Persistent Chat (Planning Tool)
 
Persistent Chat Server können Benutzer zur Teilnahme an Unterhaltungen mit mehreren Teilnehmern, themenbasiertes, die über einen Zeitraum beibehalten werden.
  
Weitere Informationen zur Vorbereitung Ihrer Umgebung für Persistent Chat Server finden Sie unter [Planen für Persistent Chat Server in Skype für Business Server 2015](../../plan-your-deployment/persistent-chat-server/persistent-chat-server.md).
