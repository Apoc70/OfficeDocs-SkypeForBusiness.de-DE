# <a name="deploy-the-vis-server-role-in-skype-for-business-server"></a>Bereitstellen der Serverrolle gegenüber in Skype für Business Server
 
**Zusammenfassung:** Stellen Sie die Video Interop Server (gegenüber) Rolle in Skype für Business Server bereit.
  
Richten Sie den Dienst gegenüber auf dem Server, der soeben erstellten im Topologie-Generator, der Skype für Business Server-bereitstellungs-Assistenten zu starten, **installieren oder aktualisieren Skype für Server Geschäftssystem** drücken, und führen diese Schritte des Assistenten:
  
1.  Wählen Sie **Lokalen Konfigurationsspeicher installieren**.
    
2. Wählen Sie **einrichten oder Entfernen von Skype für Business Server-Komponenten**.
    
3. Wählen Sie **Zertifikate anfordern, installieren oder zuweisen**.
    
4. Wählen Sie **Dienste starten**.
    
The software for this service is now installed and running. Sie können das Tool Dienste Mmc, um festzustellen, ob der **Skype für Business Server Video Interop** -Dienst zusammen mit anderen Skype für Business Server-Dienste ausgeführt wird öffnen. Next, you must configure the VIS server or pool.
