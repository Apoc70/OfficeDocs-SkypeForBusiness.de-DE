# <a name="deploy-the-vis-server-role-in-skype-for-business-server"></a>Bereitstellen der VIS-Serverrolle in Skype for Business Server
 
**Zusammenfassung:** Stellen Sie die Rolle des Video-Interop-Servers (VIS) in Skype for Business Server bereit.
  
Wenn Sie den VIS-Dienst auf dem soeben in Topology Builder erstellten Server einrichten möchten, starten Sie den Skype for Business Server-Bereitstellungs-Assistenten, drücken Sie **installieren oder aktualisieren Sie das Skype for Business Server-System** , und führen Sie die folgenden Schritte im Assistenten aus:
  
1.  Wählen Sie **Lokalen Konfigurationsspeicher installieren**.
    
2. Wählen Sie **einrichten oder Entfernen von Skype for Business Server-Komponenten**aus.
    
3. Wählen Sie **Zertifikate anfordern, installieren oder zuweisen**.
    
4. Wählen Sie **Dienste starten**.
    
Die Software für diesen Dienst ist nun installiert und wird ausgeführt. Sie können das Dienste-MMC-Tool öffnen, um festzustellen, ob der **Skype for Business Server-Video-Interop-Server** Dienst zusammen mit anderen Skype for Business Server-Diensten ausgeführt wird. Next, you must configure the VIS server or pool.
