# <a name="deploy-the-vis-server-role-in-skype-for-business-server-2015"></a>Bereitstellen der VIS-Serverrolle in Skype for Business Server 2015
 
**Zusammenfassung:** Bereitstellen der Video Interop Server (gegenüber) Rolle in Skype für Business Server 2015.
  
Richten Sie den Dienst gegenüber auf dem Server, der soeben erstellten im Topologie-Generator, der Skype für Business Server-bereitstellungs-Assistenten zu starten, **installieren oder aktualisieren Skype für Server Geschäftssystem** drücken, und führen diese Schritte des Assistenten:
  
1.  Wählen Sie **Lokalen Konfigurationsspeicher installieren**.
    
2. Wählen Sie **einrichten oder Entfernen von Skype für Business Server-Komponenten**.
    
3. Wählen Sie **Zertifikate anfordern, installieren oder zuweisen**.
    
4. Wählen Sie **Dienste starten**.
    
Die Software für diesen Dienst ist nun installiert und ausgeführt wird. Sie können das Tool Dienste Mmc, um festzustellen, ob der **Skype für Business Server Video Interop** -Dienst zusammen mit anderen Skype für Business Server-Dienste ausgeführt wird öffnen. Im nächsten Schritt müssen Sie den gegenüber Server oder Pool konfigurieren.
