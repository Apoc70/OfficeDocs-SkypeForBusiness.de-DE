## <a name="configuring-skype-for-business-server-to-be-a-partner-application-for-exchange-server"></a><span data-ttu-id="a2e0c-113">Konfigurieren von Skype für Business Server werden als Partneranwendung für Exchange Server</span><span class="sxs-lookup"><span data-stu-id="a2e0c-113">Configuring Skype for Business Server to be a Partner Application for Exchange Server</span></span>

<span data-ttu-id="a2e0c-114">Die einfachste Möglichkeit zum Konfigurieren von Skype für Business Server eine partneranwendung mit Exchange Server 2016 oder Exchange Server 2013 ist das Skript konfigurieren EnterprisePartnerApplication.ps1, ein Windows PowerShell-Skript auszuführen, die mit Exchange Server verwendet werden soll.</span><span class="sxs-lookup"><span data-stu-id="a2e0c-114">The easiest way to configure Skype for Business Server to be a partner application with Exchange Server 2016 or Exchange Server 2013 is to run the Configure-EnterprisePartnerApplication.ps1 script, a Windows PowerShell script that ships with Exchange Server.</span></span> <span data-ttu-id="a2e0c-115">Um dieses Skript ausführen, müssen Sie die URL für die Skype für Business Server-Authentifizierung Metadatendokument angeben. <span data-ttu-id="a2e0c-118">Nach der Konfiguration der partneranwendung wird empfohlen, dass Sie beenden und neu (Internet Information Services, IIS) auf den Exchange-Postfach und Client Access-Servern starten.</span><span class="sxs-lookup"><span data-stu-id="a2e0c-118">After configuring the partner application it is recommended that you stop and restart Internet Information Services (IIS) on your Exchange mailbox and client access servers.</span></span> <span data-ttu-id="a2e0c-119">Mit einem Befehl wie den folgenden, der den Dienst auf dem Computer Atl-Exchange-001 neu gestartet wird, können Sie IIS neu starten:</span><span class="sxs-lookup"><span data-stu-id="a2e0c-119">You can restart IIS by using a command similar to this, which restarts the service on the computer atl-exchange-001:</span></span>

## <a name="configuring-exchange-server-to-be-a-partner-application-for-skype-for-business-server"></a><span data-ttu-id="a2e0c-121">Konfigurieren von Exchange Server als Partneranwendung für Skype für Business Server sein.</span><span class="sxs-lookup"><span data-stu-id="a2e0c-121">Configuring Exchange Server to be a Partner Application for Skype for Business Server</span></span>
