# <a name="install-sql-server-reporting-services-in-skype-for-business-server-2015"></a>Installieren von SQL Server Reporting Services in Skype für Business Server 2015
 
**Zusammenfassung:** Hier erfahren Sie, wo Sie Informationen zu SQL Server Reporting Services von Skype für Business Server 2015 verwendet wird.
  
Skype für Business Server 2015 können SQL Server Reporting Services (SSRS) zum Anzeigen und Überwachen von Berichten. Wenn Sie auf diese Funktion zurückgreifen möchten, müssen Reporting Services installiert und konfiguriert sein.
  
## <a name="install-sql-server-reporting-services"></a>Installieren von SQL Server Reporting Services

Wenn Sie beabsichtigen, verwenden Sie Skype für Business Server 2015 Monitoring Reports (Siehe den nächsten Artikel Weitere Informationen in dieser Dokumentation) muss Erstinstallation von SQL Server Reporting Services; Reporting Services können zum gleichen Zeitpunkt installieren von Microsoft SQL Server oder jederzeit nach der Installation von SQL Server installiert werden. Wenn Sie SQL Server nicht installiert haben, gehen Sie wie weiter oben in dieser Dokumentation. Beim Installieren von SQL Server, stellen Sie sicher, dass auf der Seite Featureauswahl Sie Reporting Services auswählen. Installiert, die SQL Server Reporting Services ein.
  
Erfahren Sie, wie zum Installieren von SQL Server Reporting Services finden Sie unter [SQL Server Reporting Services (SSRS)](https://technet.microsoft.com/en-us/library/ms159106.aspx).
  
Wenn Sie SQL Server ohne SQL Server Reporting Services installiert haben, können Sie dieses Feature gemäß der entsprechenden Vorgehensweise für SQL Server Reporting Services installieren.
