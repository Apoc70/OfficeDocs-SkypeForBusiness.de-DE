## <a name="making-the-best-use-of-the-server-performance-report"></a><span data-ttu-id="f1413-125">Optimale Verwendung des Berichts über Serverleistung</span><span class="sxs-lookup"><span data-stu-id="f1413-125">Making the best use of the Server Performance Report</span></span>

<span data-ttu-id="f1413-p104">Der Bericht über Serverleistung bietet verschiedene Möglichkeit, die Daten zu filtern. Sie können z. It's a good idea when viewing the server performance report to make use of these filters. For example, suppose you have a Mediation Server that has a poor call percentage of 3.24%. If you look solely at wireless calls, that same server might have a poor call percentage approaching 20%. That means that the server was having difficulty with wireless calls, a problem that is partially obscured because the server was not having problems with wired calls.</span></span>
