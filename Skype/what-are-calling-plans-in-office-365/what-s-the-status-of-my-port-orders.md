# Wie ist der Status meiner Portierungsanträge?

Sie können den Status Ihres Portierungsauftrags anzeigen, indem Sie zu **Skype for Business Admin Center** > **VoIP** > **Portierungsaufträge** wechseln. In der Spalte **Status** finden Sie die Status der einzelnen Portierungsaufträge. Wenn Sie Hilfe benötigen,[Kontaktieren des Office 365 Business-Supports - Administratorhilfe](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b).
  
In dieser Tabelle sehen Sie den Status Ihres Auftrags. Wenn Sie aktiv werden müssen, sehen Sie hier außerdem, welche Möglichkeiten Sie haben.
  
|**Status**|**Können Sie den Auftrag anzeigen?**|**Können Sie den Auftrag bearbeiten?**|**Können Sie den Auftrag stornieren?**|**Können Sie den Auftrag löschen?**|**Beschreibung**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**Wird verarbeitet** <br/> |Ja  <br/> |Nein  <br/> |Ja  <br/> |Nein  <br/> |Der Administrator hat den Auftrag erstellt, und der Auftrag ist bei Microsoft eingegangen.  <br/> |
|**Netzbetreiber wird kontaktiert** <br/> |Ja  <br/> |Nein  <br/> |Ja  <br/> |Nein  <br/> |Der Auftrag ist bei Microsoft eingegangen und wurde genehmigt. Jetzt holen wir die Genehmigung des alten Anbieters ein.  <br/> |
|**Übertragung genehmigt** <br/> |Ja  <br/> |Bindung an Festbestellung (Firm Order Commitment, FOC)  <br/> |Ja  <br/> |Nein  <br/> |Der Auftrag wurde vom alten Anbieter akzeptiert, und das FOC-Datum wurde festgelegt.  <br/> |
|**Übertragung ausstehend** <br/> |Ja  <br/> |Nein  <br/> |Nein  <br/> |Nein  <br/> |Die Übertragung findet in weniger als 24 Stunden statt. Daher kann der Auftrag nicht mehr bearbeitet oder storniert werden.  <br/> |
|**Fehler** <br/> |Nein  <br/> |Ja  <br/> |Ja  <br/> |Ja (Sie können den Portierungsauftrag jetzt bei einem Fehler nicht mehr löschen. Sie müssen den Portierungsauftrag erneut erstellen oder [Kontaktieren des Office 365 Business-Supports - Administratorhilfe](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b).  <br/> |Der alte Anbieter hat den Auftrag abgelehnt.  <br/> |
|**Abgeschlossen** <br/> |Ja  <br/> |Nein  <br/> |Nein  <br/> |Nein  <br/> |Die Nummern wurden erfolgreich übertragen.  <br/> |
|**Abgebrochen** <br/> |Nein  <br/> |Ja  <br/> |Nein  <br/> |Nein  <br/> |Der Administrator hat den Auftrag storniert.  <br/> |
   
 **Umfassende Schritt-für-Schritt-Anweisungen erhalten Sie unter [Übertragen von Telefonnummern zu Office 365](transfer-phone-numbers-to-office-365.md).**
