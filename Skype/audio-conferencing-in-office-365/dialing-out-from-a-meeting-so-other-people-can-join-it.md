# <a name="dialing-out-from-a-meeting-so-other-people-can-join-it"></a>Telefonverbindung aus einer Besprechung, damit andere Personen sie beitreten können

Als Organisator der Besprechung können Sie sich einwählen, bei Verwendung einer Skype für Business oder Microsoft-Teams app an um andere Personen, die die gleiche über das Telefon Besprechung teilnehmen zu lassen. Obwohl es nicht erforderlich ist, wenn Sie sich an eine Person telefonieren, empfiehlt es sich, dass Sie die vollständige Rufnummer, einschließlich den Länder-/Regionscode wählen. So erhalten Sie Anwahl arbeiten:
  
- Sie können Dial-Out-nur, wenn Sie an einer Besprechung mithilfe einer Skype für Business oder Microsoft-Teams app teilnehmen.
    
- Sie als Organisator der Besprechung wurden für Audiokonferenzen aktiviert.

 **Schritt 1:** Verwenden Sie in der Besprechung einladen die Option **Weitere Personen einladen** anwählen eine Telefonnummer ein.
  
![Dial-Out-mit Skype für Unternehmen.](../images/9896abec-7a6f-4148-ad09-76a1cf4b56e1.png)
  
 **Schritt 2:** Geben Sie die vollständige Telefonnummer, einschließlich den Länder-/Regionscode in das Feld ein.
  
![Dial-Out-Telefonnummer in Skype für Unternehmen](../images/084b4fb5-21eb-4f10-9a1a-c92b919084b9.png)
  
## <a name="supported-countries-and-regions"></a>Unterstützte Länder / Regionen

Dial-Out-ist nur für einige Länder/Regionen verfügbar. Vollständige Liste finden Sie unter [Verfügbarkeit von Ländern und Regionen für Audiokonferenzen und plant aufrufen](../country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md).
