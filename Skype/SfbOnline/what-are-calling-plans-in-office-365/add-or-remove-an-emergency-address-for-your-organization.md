# <a name="add-change-or-remove-an-emergency-address-for-your-organization"></a>Hinzufügen, Ändern oder Entfernen einer Notfalladresse für Ihre Organisation

Eine Notfalladresse muss mit einer Telefonnummer verbunden sein, aber wenn dies der Fall ist, kann dies zwischen Land/Regionen variieren. In den Vereinigten Staaten müssen Sie beispielsweise eine Notfalladresse zuordnen, wenn Sie die Telefonnummer einem Benutzer zuweisen. Im Vereinigten Königreich müssen Sie der Telefonnummer eine Notfalladresse zuordnen, wenn Sie die Telefonnummern von Office 365 erhalten oder wenn Sie Telefonnummern von Ihrem aktuellen Dienstanbieter übertragen.
  
No matter which country/region you are in, it's possible to add a location or locations to an emergency address or remove an emergency address. Depending on the number of physical locations in your organization, you can create them for buildings, floors, and offices. See [What are emergency locations, addresses, and call routing?](/microsoftteams/what-are-emergency-locations-addresses-and-call-routing) for some details.
  
Informationen dazu, wie Sie einen Anrufplan erhalten und wie viel er kostet, finden Sie unter [Skype for Business-und Microsoft Teams-Add-on-Lizenzierung](../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md).
  
## <a name="add-an-emergency-address"></a>Notfalladresse hinzufügen

1. Melden Sie sich bei Office 365 mit Ihrem Firmen- oder Schulkonto an.
    
2. Wechseln Sie zum **Microsoft Teams Admin Center** > **Legacy-Portal**.
    
3. Gehen Sie in der linken Navigationsleiste zu **VoIP** > **Notfallstandorte** und klicken auf die Schaltfläche **Neue Adresse hinzufügen**.
    
    > [!Important]
    > Damit Sie in der linken Navigationsleiste des Skype for Business admin Centers die Option **Stimme** sehen können, müssen Sie zunächst mindestens eine **Enterprise E5-Lizenz**, eine Lizenz für ein **Telefon System** -Add-on oder eine Lizenz für das Add-on **Audio Conferencing** kaufen.
    
4. Geben Sie im Aktionsbereich unter **Neue Adresse** die erforderlichen Informationen in die Felder ein.
    
5. Nachdem Sie alle Informationen für die Adresse eingegeben haben, klicken Sie auf über **prüfen**.
    
    > [!IMPORTANT]
    > Durch die Validierung einer Postanschrift oder Adresse wird sichergestellt, dass die Adresse legitim und richtig formatiert ist. Es ist möglich, dass eine teilweise korrekte Notfalladresse, wie z. B. ein falsch geschriebener Name einer Stadt, bei der Validierung anerkannt wird. Trotz des Schreibfehlers besteht die Adresse die Validierung, da die Kombination aus falsch geschriebenem Ortsnamen und anderen richtigen Adressangaben ausreicht, um den Anruf an die zuständige Rettungsleitstelle weiterzuleiten. 
    
    > Hinweis: in Belgien, Frankreich, Deutschland, Irland, den Niederlanden und Spanien ist es wichtig zu verstehen, dass zur erfolgreichen Aktivierung einer Telefonnummer in Office 365 die Adress Einrichtung am Notfall Standort, die zum Abrufen der Nummer verwendet wird, mit der Vorwahl der Telefonnummer.
  
    Wenn die Adresse nicht validiert werden kann, können Sie eine manuelle Überprüfungsanforderung senden, indem Sie auf **Eine Anforderung zur Überprüfung senden** klicken, wenn Sie versuchen, eine Adresse in den USA zu validieren, oder klicken Sie auf **Eine Service-Anforderung für Hilfe bei der Adressenvalidierung öffnen**, wenn Sie sich außerhalb der USA befinden.
    
6. Klicken Sie nach der Überprüfung der Adresse auf **Speichern**.
    
## <a name="change-an-emergency-address"></a>Ändern einer Notfalladresse

1. Melden Sie sich bei Office 365 mit Ihrem Firmen- oder Schulkonto an.
    
2. Wechseln Sie zum **Microsoft Teams Admin Center** > **Legacy-Portal**.
    
3. Wechseln Sie in der linken Navigationsleiste zu **VoIP** > -**Notfall Speicherorten**, wählen Sie die Adresse aus, die Sie ändern möchten, und klicken Sie im Bereich "Aktion" auf **Bearbeiten**.
    
    > [!IMPORTANT]
    > Damit Sie in der linken Navigationsleiste des Skype for Business admin Centers die Option **Stimme** sehen können, müssen Sie zunächst mindestens eine **Enterprise E5-Lizenz**, eine Lizenz für ein **Telefon System** -Add-on oder eine Lizenz für das Add-on **Audio Conferencing** kaufen.

4. Nehmen Sie die gewünschten Änderungen vor, und klicken Sie dann auf über **prüfen**.

5. Klicken Sie auf **Speichern**.

## <a name="remove-an-emergency-address"></a>Entfernen einer Notfalladresse

1. Melden Sie sich bei Office 365 mit Ihrem Firmen- oder Schulkonto an.
    
2. Wechseln Sie zum **Microsoft Teams Admin Center** > **Legacy-Portal**.
    
3. Wechseln Sie in der linken Navigationsleiste zu **VoIP** > -**Notfall Speicherorten**, wählen Sie die Adresse aus, die Sie löschen möchten, und klicken Sie im Bereich "Aktion" auf **Löschen**.
    
    > [!IMPORTANT]
    > Damit Sie in der linken Navigationsleiste des Skype for Business admin Centers die Option **Stimme** sehen können, müssen Sie zunächst mindestens eine **Enterprise E5-Lizenz**, eine Lizenz für ein **Telefon System** -Add-on oder eine Lizenz für das Add-on **Audio Conferencing** kaufen.

## <a name="troubleshooting"></a>Problembehandlung

**Die Nummer im Zustand "fehlgeschlagen".**

Nachdem Sie eine Nummer aus dem Office 365-Portal erworben haben, wurde der Status von **"Bereitstellung"** auf **"Fehler"** geändert.

Dieses Problem tritt häufig auf, wenn eine Nummer aus dem Portal hinzugefügt wird, wobei eine Notfalladresse verwendet wird, die auf einen Standort verweist, der nicht der Ortsvorwahl des Telefons entspricht.

Wenn Sie weitere Informationen zu den Nummern erhalten möchten, die nicht ordnungsgemäß aktiviert wurden, führen Sie die folgende PowerShell aus:
 
> [! Syntax] get-CsOnlineTelephoneNumber | Where-Object {$ _. ActivationState-cnotcontains "aktiviert"} | FL

Das Ergebnis sollte neben anderen Informationen wie Region, ID und ActivationState auch die PLZ enthalten.

**Beispiel**: für eine Madrid-Nummer wird die zurückgegebene PLZ "EMEA-es-all-M_MA" sein.

Wenn in der Tat eine falsche Notfalladresse verwendet wurde, stellen Sie sicher, dass Sie eine neue Notfalladresse erstellt haben, die der Vorwahl der Nummer entspricht, und weisen Sie Ihr die Nummer zu.

1. Melden Sie sich bei Office 365 mit Ihrem Firmen- oder Schulkonto an.
    
2. Wechseln Sie zum **Microsoft Teams Admin Center** > **Legacy-Portal**.
    
3. Wechseln Sie in der linken Navigationsleiste zu **VoIP** > ****-Rufnummern, und doppelklicken Sie dann auf die Nummer im Status **"Fehler"** , und wählen Sie dann im Menü Rechte Seite die **neue Notfalladresse**aus.


Bitte beachten Sie, dass sich der Status der Nummer nach dem Ändern der Notfalladresse in **"Zuordnungs Ausstehend"** ändert, und es kann bis zu 24 Stunden dauern, bis die Aktivierung erfolgreich durchgeführt wurde.
