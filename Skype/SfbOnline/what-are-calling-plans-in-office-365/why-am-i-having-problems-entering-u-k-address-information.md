# <a name="why-am-i-having-problems-entering-uk-address-information"></a>Warum habe ich Probleme bei der Eingabe in Großbritannien? Adressinformationen?

Hier sind einige Gründe, warum Sie möglicherweise Probleme haben, Ihre Adresse oder Adressen im Vereinigten Königreich (Großbritannien) korrekt zu überprüfen.
  
## <a name="whats-wrong-with-the-company-name"></a>Was ist mit dem Firmennamen falsch?

Wenn Sie Probleme bei der Eingabe des Firmennamens haben, beachten Sie Folgendes:
  
- Sie kann Buchstaben, Zahlen und die folgenden Sonderzeichen enthalten:,/-+. ' &amp; " _ 
    
    > [!CAUTION]
    > Im Vereinigten Königreich (Großbritannien) kann der Firmenname nicht das +-Zeichen haben. 
  
- Es muss mindestens 80 Zeichen lang sein.
    
- Sie kann nicht mit einem Sonderzeichen beginnen.
    
## <a name="whats-wrong-with-the-house-number"></a>Was ist falsch an der Hausnummer?

Wenn Sie Probleme bei der Eingabe der Hausnummer haben, beachten Sie Folgendes:
  
- Sie kann Buchstaben, Zahlen und die folgenden Sonderzeichen enthalten:-/[]
    
- Sie muss mit einer Zahl beginnen.
    
- Sie darf maximal 20 Zeichen lang sein.
    
- Sie kann einen groß-oder Kleinbuchstaben aufweisen.
    
## <a name="whats-wrong-with-the-house-name"></a>Was ist mit dem Hausnamen falsch?

Wenn Sie Probleme bei der Eingabe des Haus namens haben, beachten Sie Folgendes:
  
- Sie kann Buchstaben, Zahlen und die folgenden Sonderzeichen enthalten:,/-+. ' &amp; " _
    
- Es muss mindestens 50 Zeichen lang sein.
    
- Sie kann nicht mit einem Sonderzeichen beginnen.
    
## <a name="whats-wrong-with-the-street-name"></a>Was ist mit dem Straßennamen falsch?

Wenn Sie Probleme bei der Eingabe des Straßennamens haben, beachten Sie Folgendes:
  
- Sie kann Buchstaben, Zahlen und die folgenden Sonderzeichen enthalten:,/-+. ' &amp; " _ 
    
- Es muss mindestens 50 Zeichen lang sein.
    
- Sie kann nicht mit einem Sonderzeichen beginnen. 
    
## <a name="whats-wrong-with-the-city-or-town"></a>Was ist mit der Stadt oder der Stadt falsch?

Wenn Sie Probleme bei der Eingabe des Ortsnamens haben, beachten Sie Folgendes:
  
- Sie kann Buchstaben, Zahlen und die folgenden Sonderzeichen enthalten:,/-+. ' &amp; " _
    
- Es muss mindestens 50 Zeichen lang sein.
    
- Sie kann nicht mit einem Sonderzeichen beginnen. 
    
## <a name="whats-wrong-with-the-postal-code"></a>Was ist mit der Postleitzahl falsch?

Wenn Sie Probleme bei der Eingabe der Postleitzahl haben, beachten Sie Folgendes:
  
- Sie kann nur Buchstaben, Zahlen oder Leerzeichen enthalten.
    
- Es muss mindestens 8 Buchstaben enthalten, darunter Zahlen und Leerzeichen.
    
- Beachten Sie Folgendes: **Wenn Sie sich im Vereinigten Königreich (Großbritannien) befinden**, **müssen** Sie die Postleitzahl in eines der folgenden Formate setzen, wobei der Buchstabe a = Zeichen a-Z, das # Sign = eine Zahl aus 0-9 und der _ (Unterstrich) = Leerzeichen sind. Hier sind einige Beispiele:
    
|**Format**|**Beispiel**|
|:-----|:-----|
|AA # a_ # AA  <br/> |EC3B 2BD  <br/> |
|# a_ # AA  <br/> |E1W 1AP  <br/> |
|A # _ # AA  <br/> |F1-2AG  <br/> |
|# #_ # AA  <br/> |C44 8HT  <br/> |
|AA # _ # AA  <br/> |DR3 6HG  <br/> |
|AA # #_ # AA  <br/> |FN35  <br/> |
