## <a name="search-for-phone-numbers"></a><span data-ttu-id="ef5ee-109">Suchen nach Telefonnummern</span><span class="sxs-lookup"><span data-stu-id="ef5ee-109">Search for phone numbers</span></span>

3. <span data-ttu-id="ef5ee-114">Navigieren Sie in der linken Navigationsleiste zu **VoIP** > **Telefonnummern**, klicken Sie auf **Neue Nummer hinzufügen** >  und dann auf **Neue Nutzernummern**.</span><span class="sxs-lookup"><span data-stu-id="ef5ee-114">In the left navigation go to **Voice** > **Phone numbers**, and then click **Add new number** > **New user numbers**.</span></span>

    > <span data-ttu-id="ef5ee-115">Damit im linken Navigationsbereich des Skype for Business Admin Centers die Option **VoiP** angezeigt wird, müssen Sie zuerst mindestens eine **Enterprise E5-Lizenz**, eine Add-On-Lizenz für das **Telefonsystem** oder eine Add-On-Lizenz für **Audiokonferenzen** erwerben.</span><span class="sxs-lookup"><span data-stu-id="ef5ee-115">IMPORTANT: For you to see the **Voice** option in the left navigation in the Skype for Business admin center, you must first buy at least one **Enterprise E5 license**, one **Phone System** add-on license, or one **Audio Conferencing** add-on license.</span></span>  
    
    
4. <span data-ttu-id="ef5ee-116">Klicken Sie auf der Seite **Neue Benutzernummern hinzufügen** auf die Listen für **Land/Region** und **Bundesland**, um das gewünschte Bundesland auszuwählen, aus dem Sie Nummern auswählen möchten. 5.  <span data-ttu-id="ef5ee-p103">Geben Sie unter **Menge** die Anzahl der Telefonnummern ein, die Sie für Ihre Organisation benötigen, und klicken Sie auf **Hinzufügen**, um eine Reservierung zu erstellen. Wenn Sie länger als zehn Minuten brauchen, werden die Nummern wieder an den Telefonnummernpool zurückgegeben.</span><span class="sxs-lookup"><span data-stu-id="ef5ee-p103">Under **Quantity**, enter the number of phone numbers that you want for your organization and click **Add** to create a reservation.     > [!NOTE]
    > <span data-ttu-id="ef5ee-120">Neben **Benutzernummern insgesamt, die Sie erhalten können** können Sie die Anzahl der Telefonnummern sehen, die auf der Anzahl der Lizenzen basiert.</span><span class="sxs-lookup"><span data-stu-id="ef5ee-120">You can see the number of phone numbers, which is based on the number of licenses, listed next to **Total user numbers you can acquire**.</span></span> 
  
6. <span data-ttu-id="ef5ee-p104">Sie können auf **Nummern anzeigen** klicken, um eine vollständige Liste der Telefonnummern aufzurufen. Das ist nützlich, wenn Sie keine bestimmte Telefonnummer aus der Liste auswählen möchten.</span><span class="sxs-lookup"><span data-stu-id="ef5ee-p104">You can click **Show numbers** to see the full list of phone numbers. 7. <span data-ttu-id="ef5ee-123">Wählen Sie die gewünschten Telefonnummern aus, und klicken Sie dann auf **Abrufen von Rufnummern**.</span><span class="sxs-lookup"><span data-stu-id="ef5ee-123">Select the phone numbers you want ,and then click **Acquire numbers**.</span></span>
    
    > [!IMPORTANT]
    > <span data-ttu-id="ef5ee-124">Sie können mehr Telefonnummern erwerben als Sie Lizenzen haben.</span><span class="sxs-lookup"><span data-stu-id="ef5ee-124">You can acquire more phone numbers than you have licenses.</span></span> <span data-ttu-id="ef5ee-125">Um zu bestimmen, wie viele Telefonnummern Sie erwerben können, nehmen Sie Ihre Anzahl an Lizenzen, fügen Sie 10 Prozent der Anzahl an Lizenzen und dann 10 hinzu.</span><span class="sxs-lookup"><span data-stu-id="ef5ee-125">To determine how many phone numbers you can acquire, take your number of licenses, add 10 percent of the number of licenses, and then add 10.</span></span> <span data-ttu-id="ef5ee-126">Falls Sie beispielsweise 100 Lizenzen für **Anrufpläne für Inlandsanrufe**  und/oder **Anrufpläne für Auslandsanrufe**  haben, können Sie 120 Telefonnummern reservieren, vorausgesetzt, dass Sie noch keine Telefonnummern für diese 100 Benutzer erworben haben.</span><span class="sxs-lookup"><span data-stu-id="ef5ee-126">For example, if you have 100 Domestic Calling Plan and/or International Calling Plan licenses, you can reserve 120 phone numbers, assuming that you have not already acquired some phone numbers for those 100 users.> For more details, seeHow many phone numbers can you get?.</span></span> <span data-ttu-id="ef5ee-127">Weitere Details finden Sie unter[Wie viele Telefonnummern können Sie erhalten?](/microsoftteams/how-many-phone-numbers-can-you-get)</span><span class="sxs-lookup"><span data-stu-id="ef5ee-127">For details, see [How many phone numbers can you get?](/microsoftteams/how-many-phone-numbers-can-you-get)</span></span>
