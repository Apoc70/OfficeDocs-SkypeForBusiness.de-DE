## <a name="search-for-phone-numbers"></a><span data-ttu-id="cc8aa-109">Suchen nach Telefonnummern</span><span class="sxs-lookup"><span data-stu-id="cc8aa-109">Search for phone numbers</span></span>

 <span data-ttu-id="cc8aa-110">**So suchen Sie nach Telefonnummern für Ihre Benutzer**</span><span class="sxs-lookup"><span data-stu-id="cc8aa-110">**To search for phone numbers for your users**</span></span>

    <span data-ttu-id="cc8aa-114">**WICHTIG**: Damit im linken Navigationsbereich des Skype for Business Admin Centers die Option **Voice** angezeigt wird, müssen Sie zuerst mindestens eine **Enterprise E5-Lizenz**, eine Add-On-Lizenz für das **Telefonsystem** oder eine Add-On-Lizenz für **Audiokonferenzen** kaufen.</span><span class="sxs-lookup"><span data-stu-id="cc8aa-114">**IMPORTANT**: For you to see the **Voice** option in the left navigation in the Skype for Business admin center, you must first buy at least one **Enterprise E5 license**, one **Phone System** add-on license, or one **Audio Conferencing** add-on license.</span></span>
    
4. <span data-ttu-id="cc8aa-115">Klicken Sie auf der Seite **Neue Benutzernummern hinzufügen** auf die Listen für **Land/Region** und **Bundesland**, um das gewünschte Bundesland auszuwählen, aus dem Sie Nummern auswählen möchten. 5.  <span data-ttu-id="cc8aa-p103">Geben Sie unter **Menge** die Anzahl der Telefonnummern ein, die Sie für Ihre Organisation benötigen, und klicken Sie auf **Hinzufügen**, um eine Reservierung zu erstellen. Wenn Sie länger als zehn Minuten brauchen, werden die Nummern wieder an den Telefonnummernpool zurückgegeben.</span><span class="sxs-lookup"><span data-stu-id="cc8aa-p103">Under **Quantity**, enter the number of phone numbers that you want for your organization and click **Add** to create a reservation.     > [!NOTE]
    > <span data-ttu-id="cc8aa-119">Neben **Benutzernummern insgesamt, die Sie erhalten können** können Sie die Anzahl der Telefonnummern sehen, die auf der Anzahl der Lizenzen basiert.</span><span class="sxs-lookup"><span data-stu-id="cc8aa-119">You can see the number of phone numbers, which is based on the number of licenses, listed next to **Total user numbers you can acquire**.</span></span>

Das ist nützlich, wenn Sie keine bestimmte Telefonnummer aus der Liste auswählen möchten.</span><span class="sxs-lookup"><span data-stu-id="cc8aa-p104">You can click **Show numbers** to see the full list of phone numbers. 7. <span data-ttu-id="cc8aa-122">Wählen Sie die gewünschten Telefonnummern aus, und klicken Sie dann auf **Abrufen von Rufnummern**.</span><span class="sxs-lookup"><span data-stu-id="cc8aa-122">Select the phone numbers you want ,and then click **Acquire numbers**.</span></span>
