## <a name="assign-or-change-an-emergency-address"></a><span data-ttu-id="55775-110">Zuweisen oder Ändern einer Notfalladresse</span><span class="sxs-lookup"><span data-stu-id="55775-110">Assign or change an emergency address</span></span>

1. <span data-ttu-id="55775-111">Melden Sie sich bei Office 365 mit Ihrem Firmen- oder Schulkonto an.</span><span class="sxs-lookup"><span data-stu-id="55775-111">Sign in to Office 365 with your work or school account.</span></span>
    
2. <span data-ttu-id="55775-112">Wechseln Sie zum **Microsoft Teams Admin Center** > **Legacy-Portal**.</span><span class="sxs-lookup"><span data-stu-id="55775-112">Go to the **Microsoft Teams admin center** > **Legacy portal**.</span></span>
    
3. <span data-ttu-id="55775-113">Navigieren Sie in der linken Navigationsleiste zu **VoIP**, und klicken Sie dann auf **VoIP-Benutzer**.</span><span class="sxs-lookup"><span data-stu-id="55775-113">In the left navigation go to **Voice**, then click **Voice users**.</span></span>
    
    > [!IMPORTANT]
    > <span data-ttu-id="55775-114">Damit Sie in der linken Navigationsleiste des Skype for Business admin Centers die Option **Stimme** sehen können, müssen Sie zunächst mindestens eine **Enterprise E5-Lizenz**, eine Lizenz für ein **Telefon System** -Add-on oder eine Lizenz für das Add-on **Audio Conferencing** kaufen.</span><span class="sxs-lookup"><span data-stu-id="55775-114">For you to see the **Voice** option in the left navigation in the Skype for Business admin center, you must first buy at least one **Enterprise E5 license**, one **Phone System** add-on license, or one **Audio Conferencing** add-on license.</span></span>
    
4. <span data-ttu-id="55775-115">Suchen Sie auf der Seite **VoIP-Benutzer** nach dem Benutzer, dessen Notfalladresse Sie ändern möchten und wählen Sie ihn aus.</span><span class="sxs-lookup"><span data-stu-id="55775-115">On the **Voice users** page, locate and select the user you want to change the emergency address for.</span></span>
    
5. <span data-ttu-id="55775-116">Klicken Sie im Bereich „Aktion" unter **Notfallstandort** auf **Ändern**.</span><span class="sxs-lookup"><span data-stu-id="55775-116">In the Action pane, under **Emergency location**, click **Change**.</span></span>
    
    > [!IMPORTANT]
    > <span data-ttu-id="55775-117">Sie können nur eine bereits überprüfte Notfalladresse ändern.</span><span class="sxs-lookup"><span data-stu-id="55775-117">You can only change an emergency address that has already been validated.</span></span> <span data-ttu-id="55775-118">Wenn Sie eine Notfalladresse ändern möchten, die nicht überprüft wurde, löschen Sie Sie, und erstellen Sie eine weitere Notfalladresse.</span><span class="sxs-lookup"><span data-stu-id="55775-118">To change an emergency address that hasn't been validated, delete it and create another emergency address.</span></span> 
  
6. <span data-ttu-id="55775-119">Klicken Sie auf der Seite **Nummer zuweisen** auf **Standort ändern**.</span><span class="sxs-lookup"><span data-stu-id="55775-119">On the **Assign number** page, click **Change location**.</span></span>
    
7. <span data-ttu-id="55775-120">Geben Sie unter **Notfalladresse ändern in**den Namen des Orts ein, und klicken Sie auf **Suchen**.</span><span class="sxs-lookup"><span data-stu-id="55775-120">Under **Change emergency address to**, enter the name of the city and click **Search**.</span></span>
    
8. <span data-ttu-id="55775-121">Wählen Sie die Notfalladresse in der Dropdownliste Adresse aus, und klicken Sie auf **Speichern**.</span><span class="sxs-lookup"><span data-stu-id="55775-121">Select the emergency address from the address drop-down list, and then click **Save**.</span></span>
