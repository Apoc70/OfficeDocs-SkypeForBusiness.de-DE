## <a name="assign-or-change-an-emergency-address"></a><span data-ttu-id="29581-108">Zuweisen oder Ändern einer Notfalladresse</span><span class="sxs-lookup"><span data-stu-id="29581-108">Assign or change an emergency address</span></span>

1. <span data-ttu-id="29581-109">Melden Sie sich bei Office 365 mit Ihrem Firmen- oder Schulkonto an.</span><span class="sxs-lookup"><span data-stu-id="29581-109">Sign in to Office 365 with your work or school account.</span></span>
    
2. <span data-ttu-id="29581-110">Navigieren Sie zum **Office 365 Admin Center** > **Skype for Business**.</span><span class="sxs-lookup"><span data-stu-id="29581-110">Go to the **Office 365 admin center** > **Skype for Business**.</span></span>
    
3. <span data-ttu-id="29581-111">Rufen Sie in der linken Navigationsleiste **Voice** auf und klicken Sie dann auf **Sprachbenutzer**.</span><span class="sxs-lookup"><span data-stu-id="29581-111">In the left navigation go to **Voice** > then click **Voice users**.</span></span>
    
    > [!IMPORTANT]
    > <span data-ttu-id="29581-112">Damit im linken Navigationsbereich des Skype for Business Admin Centers die Option **Voice** angezeigt wird, müssen Sie zuerst mindestens eine **Enterprise E5-Lizenz**, eine Add-On-Lizenz für das **Telefonsystem** oder eine Add-On-Lizenz für **Audiokonferenzen** erwerben.</span><span class="sxs-lookup"><span data-stu-id="29581-112">**IMPORTANT**: For you to see the **Voice** option in the left navigation in the Skype for Business admin center, you must first buy at least one **Enterprise E5 license**, one **Phone System** add-on license, or one Audio Conferencing add-on license.</span></span>
    
4. <span data-ttu-id="29581-113">Suchen Sie auf der Seite **VoIP-Benutzer** nach dem Benutzer, dessen Notfalladresse Sie ändern möchten und wählen Sie ihn aus.</span><span class="sxs-lookup"><span data-stu-id="29581-113">On the **Voice users** page, locate and select the user you want to change the emergency address for.</span></span>

    > [!NOTE]
    > <span data-ttu-id="29581-115">Sie können nur eine bereits überprüfte Notfalladresse ändern.</span><span class="sxs-lookup"><span data-stu-id="29581-115">You can only change an emergency address that has already been validated.</span></span> <span data-ttu-id="29581-116">Wenn Sie eine nicht überprüfte Notfalladresse ändern möchten, löschen Sie die Adresse und erstellen Sie eine neue Notfalladresse.</span><span class="sxs-lookup"><span data-stu-id="29581-116">To change an emergency address that hasn't been validated, delete it and recreate another emergency address.</span></span> 
  
6. <span data-ttu-id="29581-117">Klicken Sie auf der Seite **Nummer zuweisen** auf **Standort ändern**.</span><span class="sxs-lookup"><span data-stu-id="29581-117">On the **Assign number** page, click **Change location**.</span></span>
    
7. <span data-ttu-id="29581-118">Geben Sie unter **Notfalladresse ändern in** den Namen der Stadt ein und klicken Sie auf **Suchen**.</span><span class="sxs-lookup"><span data-stu-id="29581-118">Under **Change emergency address to**, in the Find City box, enter the name of the city and click **Search**.</span></span>
