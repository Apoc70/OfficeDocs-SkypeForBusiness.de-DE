# <a name="moving-a-users-audio-conferencing-provider-to-microsoft"></a>Wechseln zu Microsoft als Audiokonferenzanbieter für einen Benutzer

Um Audiokonferenzen in Office 365 mit Skype für Unternehmen und die Microsoft-Teams verwenden, müssen Benutzer in Ihrer Organisation muss eine **Audiokonferenz** -Lizenz für sie und ihre Audiokonferenzanbieter zugewiesen an Microsoft festgelegt werden. Um weitere Informationen zur Lizenzierung und Kosten finden Sie unter [Testen oder erwerben Audiokonferenzen in Office 365](try-or-purchase-audio-conferencing-in-office-365.md) .
  
## <a name="to-move-a-users-audio-conferencing-provider-to-microsoft"></a>So verschieben Sie einen Benutzer für Audiokonferenzen an Microsoft

Wenn eine **Audiokonferenz** -Lizenz zu einem Benutzer, die einen audio Conferencing Provider nicht haben zugewiesen ist, vom Anbieter des Benutzers automatisch an Microsoft festgelegt, und Sie müssen keine weiteren Schritte ausführen. Wenn der Benutzer bereits einen Anbieter von Audiokonferenzen hatten, müssen Sie Anbieter des Benutzers an Microsoft ändern, nach einer **Audiokonferenz** -Lizenz zuweisen.
  
Um Microsoft festzulegen, wie der Anbieter von Audiokonferenzen für einen Benutzer mit einer **Audiokonferenz** -Lizenz zugewiesen, aber einen anderen Anbieter von Audiokonferenzen verwendet, folgendermaßen Sie vor:
  
1. Melden Sie sich bei Office 365 mit Ihrem Firmen- oder Schulkonto an.
    
2. Navigieren Sie zum **Office 365 Admin Center** > **Skype for Business**.
    
3. Navigieren Sie in der **Skype für Business Administrationscenter**zu **Audiokonferenzen**.
    
4. Wenn Sie ein Banner finden Sie unter aufgefordert werden, dass Benutzer, die eine **Audiokonferenz** haben Lizenz zugewiesen jedoch nicht, dass Microsoft legen Sie als Anbieter von Audiokonferenzen noch auf **Klicken Sie hier, um sie zu verschieben**. Wenn das Banner nicht angezeigt wird, in der **Skype für Business Administrationscenter** klicken Sie auf **Benutzer**, und wählen Sie dann den Filter für **Benutzer bereit, in der Audiokonferenz verschoben werden soll** .
    
5. Wählen Sie die Benutzer, den, die Sie verschieben möchten, und klicken Sie dann im Bereich Aktion auf **Bearbeiten**.
    
6. Wählen Sie in der Liste **Name des Anbieters** **Microsoft** .
    
    > [!NOTE]
    > Wenn der Anbieter von Audiokonferenzen eines Benutzers in Microsoft geändert wird, ändert sich der Audiokonferenzinformationen des Benutzers. Wenn Sie diese Informationen müssen, notieren Sie es irgendwo vor dem Ändern des Anbieters von einem Benutzer. 
  
7. Klicken Sie auf **Speichern**.
    
## <a name="what-else-should-i-know"></a>Was sollte ich noch wissen?

- Wenn Sie den Benutzer in der Liste auswählen, können Sie die Standardinformationen für die Audiokonferenz des Benutzers anzeigen, aber nicht möglich, der Audiokonferenzen PIN finden Sie unter. Sie können die Audiokonferenz PIN eines Benutzers zurücksetzen, indem Sie auf **Zurücksetzen**.
    
- Wenn Sie Audiokonferenzen Einstellungen für einen Benutzer ändern möchten, können Sie wählen Sie den Benutzer aus der Liste und dann klicken Sie auf **Bearbeiten** und auf nehmen Sie auf der Seite **Eigenschaften** die Änderungen vor.
