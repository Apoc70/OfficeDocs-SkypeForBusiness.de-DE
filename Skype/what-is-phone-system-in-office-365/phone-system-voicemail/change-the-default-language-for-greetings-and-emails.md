# <a name="change-the-default-language-for-greetings-and-emails"></a>Ändern Sie die Standardsprache für Ansagen und -e-Mails

Wenn Sie ein [Office 365 globaler Administrator angemeldet](https://support.office.com/en-us/article/da585eea-f576-4f55-a1e0-87090b6aaa9d)sind, können Sie Skype für Unternehmen seiner standardmäßigen Voicemail Begrüßung in einer anderen Sprache wiedergegeben einrichten. Die Standard-System-Ansage sieht aus wie "bitte eine Nachricht hinterlassen John Smith. Notieren Sie Ihre Nachricht ein, nach der Ton. Wenn Sie die Aufzeichnung abgeschlossen haben, legen Sie auf, oder drücken Sie die Rautetaste für weitere Optionen." 
  
 **Lesen Sie zunächst diese wichtigen Informationen:**
  
- **Die Sprachen, die Ihnen zur Verfügung stehen, hängt vom Speicherort Ihrer Organisation bestimmt werden**. Wenn Ihre Organisation in den USA befindet, können Sie beispielsweise die Standardsprache Englisch oder Spanisch festlegen. Wenn Ihre Organisation in Kanada befindet, können Sie zwischen Englisch als auch Französisch auswählen. Eine Liste der unterstützten Sprachen finden Sie unter [Sprachen für Voicemail-Ansage und Nachrichten von Skype für Unternehmen](languages-for-voicemail-greetings-and-messages.md).
    
- **Es ist nicht möglich, die Systemsprache für nur eine Person in der Organisation zu ändern. ** Sie können die Systemsprache nur für alle Personen in der Organisation ändern.
    
    > [!NOTE]
    > Benutzer können nach der Anmeldung ihre eigene Ansagesprache in ihren Einstellungen ändern. 
  
- **Möchten Sie Ihre Voicemail-Antwortnachricht aufzeichnen?** Finden Sie unter [Überprüfen von Skype für Business Voicemail und Optionen](https://support.office.com/en-us/article/2deea7f8-831f-4e85-a0d4-b34da55945a8).
    
## <a name="change-the-system-language-for-everyone-in-your-organization"></a>Ändern der Systemsprache für alle Personen in der Organisation

1. Melden Sie sich mit Ihrem [Office 365 globaler Administrator](https://support.office.com/en-us/article/da585eea-f576-4f55-a1e0-87090b6aaa9d) Konto am[https://portal.office.com/adminportal/home](https://portal.office.com/adminportal/home). 
    
2. Wählen Sie im Admin Center die Optionen **Einstellungen** > **Organisationsprofil** aus. 
    
     ![Choose Settings and then choose Organization profile.](../../images/9d9de520-bb84-409f-9417-96bd8ec86c48.png)
  
3. Wählen Sie **Bearbeiten** aus.
    
    ![Choose Edit.](../../images/e4a0b09d-2b68-4bc8-a0d3-230939843ee2.png)
  
4. Wählen Sie in der Liste **Bevorzugte Sprache** eine Sprache für alle in der Organisation aus.
    
5. Klicken Sie auf **Speichern**.
