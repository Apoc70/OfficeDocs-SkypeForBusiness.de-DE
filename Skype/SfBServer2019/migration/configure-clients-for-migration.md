# <a name="configure-clients-for-migration"></a>Konfigurieren der Clients für die Migration

Dieses Thema enthält die empfohlenen Schritte zur Clientbereitstellung, die Sie vor der Migration auf Skype for Business Server 2019 durchführen sollten. Diese Konfigurationsänderungen sollten auf lync Server 2013 oder Skype for Business Server 2015 vorgenommen werden (die Version, **von**der Sie migrieren).
  
### <a name="to-configure-clients-before-migration"></a>So konfigurieren Sie Clients vor der Migration

1. Stellen Sie die neuesten Server-, Client-und Geräte Updates (Hotfixes) für die vorhandene Installation bereit.
    
2. Verwenden Sie in der vorherigen Version von Skype for Business Server die Client Versions Filterung, damit nur Clients mit den aktuellsten Updates installiert werden.
