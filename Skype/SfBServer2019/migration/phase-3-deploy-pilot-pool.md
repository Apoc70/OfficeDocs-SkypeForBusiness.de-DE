# <a name="phase-3-deploy-skype-for-business-server-2019-pilot-pool"></a>Phase 3: Bereitstellen des pilotpools für Skype for Business Server 2019

In diesem Abschnitt werden die erforderlichen Schritte zum Bereitstellen eines pilotpools von Skype for Business Server 2019 beschrieben. Die Bereitstellung von Skype for Business Server 2019 erfordert die Verwendung des Topologie-Generators zum Definieren Ihrer Topologie und der Komponenten, die Sie bereitstellen möchten, Vorbereiten Ihrer Umgebung für die Bereitstellung der Skype for Business Server 2019-Komponenten, Veröffentlichen Ihrer Topologie Entwerfen Sie auf dem ersten Front-End-Server, und installieren und konfigurieren Sie dann die Skype for Business Server 2019-Software für die Komponenten für Ihre Bereitstellung. Wenn Sie fertig sind, wird die Bereitstellung Ihres Skype for Business Server 2019-pilotpools mit einem vorhandenen Legacy Pool koexistieren.
