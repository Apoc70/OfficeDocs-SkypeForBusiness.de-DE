# <a name="phase-3-deploy-skype-for-business-server-2019-pilot-pool"></a>Phase 3: Bereitstellen von Skype für Business Server 2019 pilot pool

In diesem Abschnitt werden die erforderlichen Schritte zur Bereitstellung eines pilotpools von Skype für Business Server 2019 behandelt. Die Bereitstellung von Skype für Business Server 2019 erforderlich Vorbereiten der Umgebung für die Bereitstellung von der Skype für Business Server 2019 Komponenten und Veröffentlichen Ihrer Topologie definieren Ihrer Topologie und Komponenten, die Sie bereitstellen möchten, mithilfe der Topologie-Generator Entwerfen Sie auf dem ersten Front-End-Server, und klicken Sie dann installieren und konfigurieren Skype für Business Server 2019 Software für die Komponenten für die Bereitstellung. Wenn abgeschlossen ist, wird Ihre Skype für die Bereitstellung eines pilotpools Business Server 2019 mit einer vorhandenen vorversionspool koexistieren.
