# <a name="phase-8-decommission-legacy-pools"></a>Phase 8: Außerbetriebsetzen der Legacypools

Im folgenden Thema finden Sie Anleitungen zum Aktualisieren von DNS-Einträgen, zum Verschieben des Inhalts Verwaltungsservers, zur Außerbetriebnahme von Pools sowie zum Deaktivieren und Entfernen von Servern und Pools aus einer Legacy Bereitstellung. Es sind nicht alle der in diesem Abschnitt aufgeführten Verfahren notwendig. Lesen Sie die Dokumentation, um das Verfahren zu ermitteln, das Sie für die Außerbetriebsetzung ausführen möchten. 
  
Einen datierten, aber erschöpfenden Artikel zum Entfernen von Servern und Serverrollen sowie eine Schritt-für-Schritt-Anleitung zur Außerbetriebnahme einer Bereitstellung finden Sie herunterladen [deinstallieren Microsoft lync Server und Entfernen von Serverrollen](https://go.microsoft.com/fwlink/p/?linkId=246227).
