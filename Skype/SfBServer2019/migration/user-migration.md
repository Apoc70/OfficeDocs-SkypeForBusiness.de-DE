# <a name="user-migration"></a>Benutzermigration

Eine allgemein akzeptierte bewährte Methode für Migrationen ist, erstellen mehrere Testbenutzer und deren Verwendung zur Durchführung von Systemtests. Nachdem Sie erfolgreich verschoben und diese Konten getestet haben, sollten Sie eine Gruppe von pilotproduktionsbenutzern identifizieren und dazugehörigen Konten verschieben und Validierungstests durchführen. Wenn Sie Standardkontingente Ergebnisse zu erhalten, können Sie die restlichen Benutzer in die neue Bereitstellung verschieben.
  
Weitere Informationen zum Aktivieren von Benutzern für Skype für Business Server 2019 finden Sie unter der Bereitstellung von Skype für Business Server 2019-Dokumentation.
