# <a name="back-up-systems-and-data"></a>Sichern von Systemen und Daten

Bevor Sie mit der Migration beginnen, müssen Sie eine vollständige Systemsicherung durchführen und Ihr vorhandenes System dokumentieren, einschließlich eines Inventars von Benutzerkonten, die in jedem Pool verwaltet werden, sodass Sie ein Rollback ausführen können, falls dies erforderlich ist. Mehrere Tools und Programme stehen für die Sicherung und Wiederherstellung von Daten, Einstellungen und Systemen zur Verfügung.

Ausführliche Informationen und Verfahren finden Sie unter Sicherungs-und Wiederherstellungsverfahren für Ihre Version.
