# <a name="back-up-systems-and-data"></a>Sichern von Systemen und Daten

Vor Beginn die Migration Ausführen einer vollständigen Sicherung, und Dokumentieren Sie das bestehende System, einschließlich einer Bestandsaufnahme der Benutzerkonten, die in jedem Pool verwaltet werden, damit Sie ein Rollback können zurück, wenn es erforderlich ist. Mehrere Tools und Programme stehen für das Sichern und Wiederherstellen von Daten, Einstellungen und Systemen.

Ausführliche Informationen und Verfahren finden Sie unter Backup and restore Verfahren für Ihre Version.
