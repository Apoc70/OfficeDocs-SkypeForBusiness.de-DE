# <a name="run-best-practices-analyzer"></a>Ausführen von Best Practices Analyzer

Das Best Practices Analyzer Tool erfasst Konfigurationsinformationen aus einer Bereitstellung und bestimmt, ob die Konfiguration den Microsoft best Practices festgelegt ist. Sie können das Tool installieren, auf einem Clientcomputer, der Microsoft .NET Framework 3.5 Servicepack 1 ausgeführt wird, oder direkt auf dem Server, der die legacy-Installation ausgeführt wird. Es wird empfohlen, dass Sie installieren und führen Sie dieses Tool auf einem Clientcomputer. Die Verwaltungstools sollten auch lokal auf dem Clientcomputer installiert sein, damit der Best Practices Analyzer eine umfassende Auswahl an Daten gesammelt werden können. 
  
Sie können Best Practices Analyzer herunterladen, aus dem Microsoft Download Center unter [https://go.microsoft.com/fwlink/p/?linkid=246173](https://go.microsoft.com/fwlink/p/?linkid=246173).
