Es wird empfohlen, dass Sie installieren und führen Sie dieses Tool auf einem Clientcomputer. Die Verwaltungstools sollten auch lokal auf dem Clientcomputer installiert sein, damit der Best Practices Analyzer eine umfassende Auswahl an Daten gesammelt werden können.
