# Einrichten des Computers für Skype für Business Online Management mithilfe von Windows PowerShell

> [!IMPORTANT]
> Dieser Artikel wurde maschinell übersetzt. Bitte beachten Sie den Haftungsausschluss.  
  
Remote-Verwaltung von Skype for Business Online mithilfe von Microsoft PowerShell wird nur unter 64-Bit-Computern unter einem der folgenden Betriebssysteme unterstützt:
  
- Windows 10
    
- Windows 8.1
    
- Windows 8
    
- Windows Server 2012 R2
    
- Windows Server 2012
    
- Windows Server 2008
    
- Windows 7
    
Zusätzlich zu den unterstützten Betriebssystem muss der Computer auch Folgendes ausgeführt werden:
  
- PowerShell
    
- Skype for Business Online-Konnektormodul
    
Finden Sie Informationen zum Herunterladen und installieren diese Elemente in den folgenden Themen:
  
- [Herunterladen und Installieren des Skype for Business Online-Connectormoduls](download-and-install-the-skype-for-business-online-connector-module.md)
    
- [Herunterladen und Installieren von Windows PowerShell 3.0](download-and-install-windows-powershell-3-0.md)
