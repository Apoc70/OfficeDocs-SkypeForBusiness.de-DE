# Anpassen von Besprechungseinladungen

> [!IMPORTANT]
> Dieser Artikel wurde maschinell übersetzt. Bitte beachten Sie den Haftungsausschluss.  
  
Als Administrator können Sie sicherstellen, dass alle Skype for Business besprechungseinladungen einige hilfreiche Informationen, die Ihre Benutzer müssen möglicherweise, wie z. B. Links zu Ihrer Supportwebsite und Haftungsausschluss, Ihr Firmenlogo und anderen Text, die Sie einschließen möchten enthalten.
  
## Tipps zum Erstellen eines Logos für Besprechungseinladungen
<a name="__top"> </a>

1. Erstellen Sie ein Bild, das höchstens 188 Pixel breit und 30 Pixel hoch ist (dies ist ziemlich klein).
    
2. Speichern Sie das Bild im JPG-Format.
    
3. Speichern Sie das Bild an einem zentralen Ort, die jeder in Ihrem Unternehmen zugreifen können, wie etwa einen freigegebenen Ordner im Netzwerk freigegebenen OneDrive for Business-Ordner Dropbox, usw..
    
    Jetzt können Sie es in Ihre Besprechungseinladungen aufnehmen. Weitere Informationen hierzu finden Sie in den folgenden Schritten.
    
## Anpassen von Besprechungseinladungen
<a name="__top"> </a>

1. Melden Sie sich bei Office 365 mit Ihrem Firmen- oder Schulkonto an.
    
2. Wählen Sie die Office 365 Admin Center, **Admin centers** > **Skype for Business**.
    
3. Im Admin Center von Skype for Business wählen Sie **Onlinebesprechungen**. 
    
4. Füllen Sie die folgenden Felder aus:
    
    ![Here's what it might look like when you complete the form to customize your meeting invitations.](../images/b0a7c3c6-0d86-41c6-b116-331143bbe398.png)
  
|**Option**|**Beschreibung**|
|:-----|:-----|
|**Logo-URL** <br/> |Geben Sie die URL ein, auf denen Ihr Logo gespeichert ist. Denken Sie daran, dass es im Format JPG sein muss!  <br/> |
|**Hilfe-URL** <br/> |Wenn Ihre Organisation eine Supportwebsite soll hat zu wechseln, wenn sie Probleme haben, geben Sie die URL hier.  <br/> |
|**URL für rechtliche Hinweise** <br/> |Wenn Ihre Organisation eine juristische Website soll hat zu wechseln, wenn sie Probleme haben, geben Sie die URL hier.  <br/> |
|**Fußzeilentext** <br/> |Geben Sie eine nur aus Text bestehende Fußnote für Ihre Nutzer ein.  <br/> |
   
5. Klicken Sie auf **Speichern**.
    
6. Warten Sie etwa eine Stunde, bis die Änderungen auf alle Office 365-Dienste verteilt wurden. Wechseln Sie dann zu Outlook und planen Sie eine Skype-Besprechung, um zu sehen, wie Ihre Änderungen aussehen. 
    
    ![Here's what a custom invitation looks like, without the graphic](../images/ebb5c03c-c23d-4da7-97f1-9b13e26a6cf8.png)
