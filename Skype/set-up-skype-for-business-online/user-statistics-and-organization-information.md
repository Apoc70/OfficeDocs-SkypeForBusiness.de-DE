# <a name="user-statistics-and-organization-information"></a>Benutzerinformationen Verwendungsstatistiken und Organisation

Die Skype für Business Dashboard bietet Ihnen einen schnellen Snapshot der wie der Dienst für Ihr Unternehmen eingerichtet ist. Wenn Sie Skype für Unternehmen noch nicht konfiguriert haben, finden Sie unter [Einrichten von Skype für Business Online](set-up-skype-for-business-online.md) eine schrittweise Anleitung.
  
## <a name="how-to-change-skype-for-business-online-user-statistics"></a>Ändern von Skype für Business Online Benutzer Statistiken

|||
|:-----|:-----|
|**Eigenschaft** <br/> |**Zum Ändern** <br/> |
|Skype für Business Online-Benutzern  <br/> |Zum Hinzufügen oder aus der Skype für Geschäfts-app mit Personen blockieren, finden Sie unter [zuweisen oder Entfernen von Lizenzen für Office 365 für Unternehmen](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc) oder [Entfernen von Lizenzen von Benutzern in Office 365 für Unternehmen](https://support.office.com/article/9b497c85-d0a4-4735-80fa-d3565bc05bd1).  <br/> Eine vollständige Liste der Aktionen, wenn ein Benutzer auf Ihr Unternehmen verlässt finden Sie unter [einem früheren Mitarbeiter von Office 365 zu entfernen](https://support.office.com/article/44d96212-4d90-4027-9aa9-a95eddb367d1).  <br/> |
|Benutzer, die für einwahlkonferenzen aktiviert  <br/> |Damit Personen für einwahlkonferenzen, finden Sie unter [Einrichten von Audiokonferenzen für Skype für Unternehmen und die Microsoft-Teams](../audio-conferencing-in-office-365/set-up-audio-conferencing.md).  <br/> |
|Benutzer in der cloud  <br/> |Dies ist die Anzahl von Skype für Business Online-Benutzern, die über Office 365 Benutzerkonten verfügen.  <br/> Hinzufügen von Personen zu Ihrem Office 365-Abonnement (d. h., Office 365 Konten für sie erstellt), finden Sie unter [Benutzer einzeln oder in einer Sammeloperation zu Office 365 - Admin Hilfe hinzufügen](https://support.office.com/article/1970f7d6-03b5-442f-b385-5880b9c256ec).  <br/> |
|Benutzer synchronisiert und online verwaltet  <br/> |Dies ist die Anzahl der Skype für Business Online Benutzer, die von einem lokalen Active Directory synchronisiert werden.  <br/> Informationen zum Einrichten der lokalen Active Directory finden Sie unter [Einrichten von verzeichnissynchronisierung für Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
   
> [!NOTE]
> Wenn Sie, dass Unavailable aufgeführt sehen, bedeutet dies keine Informationen oder sind Fehler aufgetreten. 
  
## <a name="how-to-change-information-about-your-organization"></a>Gewusst wie: Ändern von Informationen über Ihre Organisation

|||
|:-----|:-----|
|**Eigenschaft** <br/> |**Zum Ändern** <br/> |
|Name der Organisation  <br/> |Dies ist der Name, den Sie angegeben haben, wenn Sie sich für Office 365 haben angemeldet. Zum Ändern dieser Einstellung finden Sie unter [Ändern Ihres Unternehmens Adresse, technischen Kontakt ein, und vieles mehr](https://support.office.com/article/a36e5a52-4df2-479e-bb97-9e67b8483e10).  <br/> |
|Organisations-ID  <br/> |Sie können Ihre Organisation ID nicht ändern.  <br/> |
|Organisation erstellt  <br/> |Sie können diesen Wert nicht ändern.  <br/> |
|Klicken Sie auf Active Directory-Synchronisierung  <br/> |Dies ist, ob Ihre Organisation zum einen Skype für Business Online-Organisation synchronisiert wird, die lokal wurde.  <br/> Informationen zum Einrichten der lokalen Active Directory finden Sie unter [Einrichten von verzeichnissynchronisierung für Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
|Organisation Domänen  <br/> |Wenn Sie Office 365 zunächst einrichten, sind diese Domänen, die registriert wurden.  <br/> Um eine Domäne hinzuzufügen, finden Sie unter [mehrere Domänen zu Office 365 hinzufügen](https://support.office.com/article/2d2fa996-b760-411d-a5cc-190d63f13207).  <br/> Zum Entfernen einer Domäne finden Sie unter [Entfernen einer Domäne aus Office 365](https://support.office.com/article/f09696b2-8c29-4588-a08b-b333da19810c).  <br/> |
