# Diese Skype for Business Online-Funktion ist nicht aktiviert.

> [!IMPORTANT]
> Dieser Artikel wurde maschinell übersetzt. Bitte beachten Sie den [Haftungsausschluss](e95a7d09-0c45-4b95-956e-f099b7b0e659.md#MT_Footer).  
  
Es tut uns Leid, aber es sieht so aus, als würde der Benutzer nicht über eine Lizenz für diese Funktion verfügen.
  
|**Mögliche Ursache**|**Vorgeschlagene Aktion**|
|:-----|:-----|
|Sie verfügen über Lizenzen zur Verfügung, aber Sie haben die Funktion für diesen Benutzer aktiviert.  <br/> |Wechseln Sie zu der **Office 365-Verwaltungskonsole** > **aktive Benutzer** diesen Benutzer zu lizenzieren. <br/> |
|Der Benutzer ist an einem Speicherort, in dem dieses Feature nicht verfügbar ist.  <br/> |Falls möglich, wechseln Sie zum **Office 365 Administrationscenter** > **aktive Benutzer**, und weisen Sie diesem Benutzer an einem Speicherort, in dem die Funktion steht.  <br/> |
|Die Funktion wurde für Ihre Organisation nicht aktiviert.  <br/> |Wechseln Sie zu **Office 365 Admin Center** > **Skype for Business** und aktivieren Sie die Funktion für die gesamte Organisation. <br/> |
